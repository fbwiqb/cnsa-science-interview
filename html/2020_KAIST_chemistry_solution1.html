해설 1

1-(1)

그레이엄의 법칙에 의하면 기체의 확산 속도(\(v\))는 기체 분자량(\(M\))의 제곱근에 반비례한다.

\[\frac{v_2}{v_1} = \sqrt{\frac{M_1}{M_2}}\]

반응이 일어나기 위해 기체 A와 기체 B가 이동한 거리의 비율은 3:2이다. 이를 그레이엄의 법칙에 적용하면

\[\frac{v_A}{v_B} = \frac{3}{2} = \sqrt{\frac{M_B}{M_A}}, \quad \frac{M_B}{M_A} = \left(\frac{3}{2}\right)^2 = 2.25\]

즉, 기체 B의 분자량이 기체 A보다 2.25배 크다. (또는 \(M_A : M_B = 4 : 9\))

1-(2)

각 기체의 부분 압력을 이용하여 평형상수를 표현하면 다음과 같다.

\[K = \frac{P_C}{P_A \times P_B} \quad (P_A, P_B, P_C\text{는 각 기체의 부분 압력})\]

밸브를 연 후, 반응이 일어나기 전 각 기체의 부분압력을 계산하면

\[P_A = \frac{1\text{기압} \times 2L}{2L + 1L + 1L} = 0.5\text{기압}, \quad P_B = \frac{1\text{기압} \times 1L}{2L + 1L + 1L} = 0.25\text{기압}\]

전체 부피가 일정한 경우 부분압력은 몰수에 비례한다. 그리고 반응 계수 비가 1:1:1이므로, 반응 후 \(P_C\)의 부분압력을 \(x\)라고 하면 아래와 같은 식을 세울 수 있다.

\[K = \frac{P_C}{P_A \times P_B} = \frac{x}{(0.5 - x)(0.25 - x)} = 4\]

이를 2차 방정식으로 변환하면 \(4x^2 - 4x + 0.5 = 0\)이고 \(x\)는 0.85 또는 0.15가 된다. 이때 \(P_C\)는 0.25 이하여야 하므로 \(x = 0.15\)가 된다.

평형 상태에서 각 기체의 부분압력은

\[P_A = 0.35\text{기압}, \quad P_B = 0.1\text{기압}, \quad P_C = 0.15\text{기압}\]

이다. 평형 상태에서 용기 내에 존재하는 기체는 A, B, C, 공기로 4가지 기체가 존재한다. 공기는 다른 기체와 반응하지 않으며 부분 압력은

\[P_{\text{공기}} = \frac{1\text{기압} \times 1L}{4L} = 0.25\text{기압}\]

이다. 따라서 용기 내부의 기체의 압력은

\[P_A + P_B + P_C + P_{\text{공기}} = 0.35 + 0.1 + 0.15 + 0.25 = 0.85\text{기압이다.}\]
