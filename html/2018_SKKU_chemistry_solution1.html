[화학I-i] 기체 반응과 분자량

0\(\degree\)C, 1기압에서 A(g) 4.0g과 AB2(g) 7.0g이 모두 반응하여 AB2 생성

반응 전 전체 부피 = 8.4 L

0\(\degree\)C 1기압에서 1몰 = 22.4 L이므로:

전체 몰수 = 8.4/22.4 = 0.375 mol

반응식: A + AB2 \(\rightarrow\) 생성물 (계수 조정 필요)

원자 보존을 고려하면: A + AB2 \(\rightarrow\) A2B2 또는 다른 형태

A의 분자량을 Ma, AB2의 분자량을 Mb라 하면:

4.0/Ma + 7.0/Mb = 0.375

또한 Mb = Ma + 2*원자량(B)

두 반응물이 모두 반응하므로 화학양론비에 맞게 소모됨.

반응식을 xA + yAB2 \(\rightarrow\) 생성물로 놓으면,

몰비 = (4.0/Ma) : (7.0/Mb) = x : y

가장 간단한 경우: A + AB2 \(\rightarrow\) A2B2 (1:1 반응)

4.0/Ma = 7.0/Mb, 또한 총 몰수 = 2*(4.0/Ma) = 0.375

4.0/Ma = 0.1875, Ma = 21.3... (정수가 안 됨)

다른 경우: 2A + AB2 \(\rightarrow\) A3B2 (2:1 반응)

4.0/Ma : 7.0/Mb = 2:1

7.0/Mb = (1/2)(4.0/Ma) = 2.0/Ma

Mb = 7.0*Ma/2.0 = 3.5*Ma

또한 Mb = Ma + 2*B원자량

총 몰수: 4.0/Ma + 7.0/(3.5*Ma) = 4.0/Ma + 2.0/Ma = 6.0/Ma = 0.375

Ma = 16 (산소 O? 또는 CH4의 C+H4는 아니므로... 원소 A의 원자량 = 16)

Mb = 3.5*16 = 56, B의 원자량 = (56-16)/2 = 20 (Ne는 비활성기체이므로 불가...)

생성물 A3B2의 분자량 = 3*16 + 2*20 = 88

[화학I-ii] 수소 원자 에너지 준위

바닥 상태에서 에너지 y의 빛으로 전자 전이 후, 에너지 q로 이온화

\(E_n = -C/n^2\) (C는 상수)

에너지 y를 흡수하여 n=1에서 n=k로 전이:

\(y = E_k - E_1 = -C/k^2 + C = C(1 - 1/k^2)\)

이온화 에너지 (n=k에서 n=\(\infty\)):

\(q = 0 - E_k = C/k^2\)

q의 최솟값: k가 최대일 때 (가장 높은 에너지 준위)

\(y = C(1-1/k^2) \rightarrow C/k^2 = C - y \rightarrow q_{\min} = C - y\)

바닥 상태 이온화 에너지가 C이므로: \(q_{\min} = C - y\)

즉, 전체 이온화 에너지 C에서 이미 흡수한 y를 빼면 된다.

[화학I-iv] 극성 분자 판별

대전체에 끌리는 것 = 극성 분자

\(\text{OF}_2\): O가 중심, 굽은형 \(\rightarrow\) 극성

\(\text{CS}_2\): C가 중심, 직선형, 대칭 \(\rightarrow\) 비극성

\(\text{NF}_3\): N이 중심, 삼각뿔형(비공유전자쌍) \(\rightarrow\) 극성

HCN: 직선형이지만 비대칭(H-C\(\equiv\)N) \(\rightarrow\) 극성

\(\text{C}_3\text{F}_2\)O (포스겐 유사?): 확인 필요. \(\text{C}_2\text{F}_2\) 형태면 극성

대전체에 끌리는 분자: \(\text{OF}_2\), \(\text{NF}_3\), HCN

[화학I-v] 루이스 전자점식과 결합각

폼산(HCOOH): C가 중심원자

C는 O와 이중결합, OH와 단일결합, H와 단일결합

\(\rightarrow\) 3개의 전자 영역 \(\rightarrow\) 삼각평면 구조 \(\rightarrow\) 결합각 약 120\(\degree\)

인산(\(\text{H}_3\text{PO}_4\)): P가 중심원자

P는 1개의 P=O 이중결합, 3개의 P-OH 단일결합

\(\rightarrow\) 4개의 전자 영역 \(\rightarrow\) 사면체 구조 \(\rightarrow\) 결합각 약 109.5\(\degree\)

비교: C의 결합각(약 120\(\degree\)) > P의 결합각(약 109.5\(\degree\))
