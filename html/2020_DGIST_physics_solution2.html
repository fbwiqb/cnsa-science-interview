해설 2

1.

아래의 그림에서 알 수 있듯이, 자기장 영역에서 전하량 q를 갖는 입자의 운동은 반지름 \(r\left(= \dfrac{mv}{qB}\right)\) 의 원운동이다.

즉, \(\tan\dfrac{\theta}{2} = \dfrac{a}{r} = \dfrac{aqB}{mv}\) 를 얻는다.

따라서, \(\tan\theta = \dfrac{\sin\theta}{\cos\theta} = \dfrac{2\sin\dfrac{\theta}{2}\cos\dfrac{\theta}{2}}{\cos^2\dfrac{\theta}{2} - \sin^2\dfrac{\theta}{2}} = \dfrac{2\tan\dfrac{\theta}{2}}{1 - \tan^2\dfrac{\theta}{2}} = \dfrac{2aqBmv}{m^2v^2 - a^2q^2B^2}\) 을 얻는다.

2.

가. 아래의 그림에서 어느 하나의 질량 (가령, A) 에 작용하는 중력의 합 \(F\) 는

\[F = \frac{Gm^2}{(2R)^2} + 2 \times \cos\frac{\pi}{4} \times \frac{Gm^2}{(\sqrt{2}\,R)^2} = \frac{Gm^2}{R^2}\left[\frac{1 + 2\sqrt{2}}{4}\right]\]

이 되고, 원궤도의 중심 방향으로 작용한다.

나. 이때 해당 질량이 속력 \(v\) 로 원운동하기 위해서는 위에서 구한 중력의 합 \(F\) 가 구심력으로 작용하여야 한다. 즉, \(\dfrac{mv^2}{R} = \dfrac{Gm^2}{R^2}\left[\dfrac{1 + 2\sqrt{2}}{4}\right]\) 로부터, \(v = \dfrac{1}{2}\sqrt{\dfrac{Gm}{R}(1 + 2\sqrt{2})}\) 를 얻는다. 나머지 세 개의 질량에도 똑같은 논의가 적용되어서 네 개의 질량은 같은 속력 \(v\) 로 원운동한다.
