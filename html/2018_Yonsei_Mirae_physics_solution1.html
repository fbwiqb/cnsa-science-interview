문제 2-1. 도르래에 매달린 두 물체

(가) \(m_2\)가 높이 \(H\)까지 올라간 후 최고 높이와 시간

\(m_1 = 2m\), \(m_2 = m\). 초기에 \(m_1\)이 높이 \(h\)에, \(m_2\)가 지면에 있다.

\(m_1\)이 더 무거우므로 \(m_1\)이 내려가고 \(m_2\)가 올라간다.

가속도:

\[a = \frac{(m_1 - m_2)g}{m_1 + m_2} = \frac{(2m - m)g}{2m + m} = \frac{g}{3}\]

\(m_1\)이 지면에 닿을 때(\(h\)만큼 이동) \(m_2\)의 속도:

\[v^2 = 2ah = \frac{2gh}{3}\]
\[v = \sqrt{\frac{2gh}{3}}\]

이 순간 \(m_2\)는 높이 \(h\)에 있다. \(m_1\)이 지면에 닿은 후 줄이 느슨해지고, \(m_2\)는 초기 속도 \(v\)로 위로 던져진 것과 같다.

추가 상승 높이:

\[\Delta h = \frac{v^2}{2g} = \frac{2gh/3}{2g} = \frac{h}{3}\]

최고 높이:

\[H = h + \frac{h}{3} = \frac{4h}{3}\]

걸리는 시간:

1단계(\(m_1\)이 떨어질 때까지): \(h = \frac{1}{2}at_1^2\)에서 \(t_1 = \sqrt{\frac{2h}{a}} = \sqrt{\frac{6h}{g}}\)

2단계(\(m_2\)가 자유 상승): \(v = gt_2\)에서 \(t_2 = \frac{v}{g} = \frac{1}{g}\sqrt{\frac{2gh}{3}} = \sqrt{\frac{2h}{3g}}\)

총 시간:

\[t_H = t_1 + t_2 = \sqrt{\frac{6h}{g}} + \sqrt{\frac{2h}{3g}} = \sqrt{\frac{2h}{3g}}\left(\sqrt{9} + 1\right) = \sqrt{\frac{2h}{3g}}(3 + 1) = 4\sqrt{\frac{2h}{3g}}\]

(나) \(m_2\)가 \(H\)에서 \(h\)까지 내려온 후 \(m_1\)의 속도

\(m_2\)는 높이 \(H = \frac{4h}{3}\)에서 자유 낙하하여 높이 \(h\)까지 내려온다. 낙하 거리: \(\frac{h}{3}\)

\[v_2^2 = 2g \times \frac{h}{3} = \frac{2gh}{3}\]
\[v_2 = \sqrt{\frac{2gh}{3}}\]

이 순간 줄이 다시 팽팽해지며 \(m_1\)(지면에서 정지)과 연결된다. 줄이 팽팽해지는 순간 비탄성적인 충격이 발생한다(줄은 늘어나지 않으므로). 이 순간 운동량 보존을 적용한다.

\(m_2\)는 아래로 속도 \(v_2\), \(m_1\)은 줄을 통해 위로 올라가야 한다. 줄로 연결되어 있으므로 두 물체의 속도 크기가 같아야 한다.

줄이 팽팽해지는 순간, \(m_2\)의 속도가 \(v_2\)(아래), \(m_1\)의 속도는 0이다.

순간적인 줄의 장력에 의한 충격량(impulsive tension)으로 두 물체의 속도가 같은 크기 \(V'\)가 된다.

\(m_2\)에 대해: \(m(-V') - m(-v_2) = J\) (위 방향 장력 충격량)

\(m_1\)에 대해: \(2m(V') - 0 = J\) (장력은 위로, \(m_1\)을 위로 끌어올림)

두 식에서 \(J = 2mV'\)이고 \(m(v_2 - V') = 2mV'\)

\[v_2 = 3V'\]
\[V' = \frac{v_2}{3} = \frac{1}{3}\sqrt{\frac{2gh}{3}}\]

\(m_1\)의 속도:

\[V = V' = \frac{1}{3}\sqrt{\frac{2gh}{3}}\]

(다) \(m_1\)이 올라갈 수 있는 최고 높이 \(H'\)

\(V'\)의 속도로 \(m_1\)이 올라가고 \(m_2\)가 내려가면서 다시 Atwood 기계 운동을 한다.

초기 속도 \(V'\)에서 가속도 \(a = \frac{(m_1 - m_2)g}{m_1 + m_2} = \frac{g}{3}\) (\(m_1\)이 더 무거우므로 \(m_1\)이 다시 내려가는 방향)

\(m_1\)이 올라가고 있으므로 감속된다.

\[0 = V'^2 - 2 \times \frac{g}{3} \times H'\]
\[H' = \frac{3V'^2}{2g} = \frac{3}{2g} \times \frac{2gh}{3 \times 9} = \frac{3 \times 2gh}{2g \times 27} = \frac{h}{9}\]

따라서 \(m_1\)이 올라갈 수 있는 최고 높이는 \(H' = \frac{h}{9}\)이다.

문제 2-2. 두 줄에 연결된 원운동

(가) 두 줄이 팽팽해지는 최소 속도

두 줄의 길이가 각각 \(l\)이고, \(z\)축 위 두 점 사이 간격이 \(2d\)이다. 물체가 수평면에서 원운동을 한다.

수평 원운동의 평면은 두 연결점의 중간 높이에 있다. 아래쪽 줄이 팽팽해지려면 아래쪽 줄의 장력이 0 이상이어야 한다.

두 연결점의 중간에서 원운동한다고 하면, 각 줄이 \(z\)축과 이루는 각도를 \(\alpha\)라 하자.

기하학: \(\cos\alpha = \frac{d}{l}\)이므로 원운동 반지름 \(r = l\sin\alpha = l\sqrt{1 - d^2/l^2} = \sqrt{l^2 - d^2}\)

위쪽 줄의 장력 \(T_1\), 아래쪽 줄의 장력 \(T_2\)로 놓자.

수직 방향: \(T_1\cos\alpha - T_2\cos\alpha - mg = 0\) → \((T_1 - T_2)\cos\alpha = mg\)

수평 방향(구심력): \((T_1 + T_2)\sin\alpha = \frac{mv^2}{r}\)

아래쪽 줄이 팽팽한 최소 조건: \(T_2 = 0\)

\(T_2 = 0\)일 때: \(T_1\cos\alpha = mg\) → \(T_1 = \frac{mg}{\cos\alpha} = \frac{mgl}{d}\)

\(T_1\sin\alpha = \frac{mv_c^2}{r}\)

\[\frac{mgl}{d} \times \sin\alpha = \frac{mv_c^2}{r}\]
\[v_c^2 = \frac{gl\sin\alpha \times r}{d} = \frac{gl\sin\alpha \times l\sin\alpha}{d} = \frac{gl^2\sin^2\alpha}{d}\]

\(\sin^2\alpha = 1 - \frac{d^2}{l^2} = \frac{l^2 - d^2}{l^2}\)이므로

\[v_c^2 = \frac{g(l^2 - d^2)}{d}\]
\[v_c = \sqrt{\frac{g(l^2 - d^2)}{d}}\]

(나) 아래쪽 줄을 끊은 후의 각도

아래쪽 줄을 끊고 \(v = \sqrt{\frac{3gl}{2}}\)로 수평 원운동을 유지한다.

위쪽 줄만 남아 있으므로, 줄의 길이 \(l\)이고 \(z\)축과 각도 \(\phi\)를 이룬다.

원운동 반지름: \(r = l\sin\phi\)

수직 방향: \(T\cos\phi = mg\)

수평 방향: \(T\sin\phi = \frac{mv^2}{r} = \frac{mv^2}{l\sin\phi}\)

두 식을 나누면:

\[\tan\phi = \frac{v^2}{gl\sin\phi}\]
\[\sin\phi \tan\phi = \frac{v^2}{gl}\]
\[\frac{\sin^2\phi}{\cos\phi} = \frac{v^2}{gl} = \frac{3}{2}\]

\(\sin^2\phi = 1 - \cos^2\phi\)로 놓으면

\[\frac{1 - \cos^2\phi}{\cos\phi} = \frac{3}{2}\]

\(\cos\phi = c\)로 놓으면:

\[\frac{1 - c^2}{c} = \frac{3}{2}\]
\[2(1 - c^2) = 3c\]
\[2c^2 + 3c - 2 = 0\]
\[c = \frac{-3 + \sqrt{9 + 16}}{4} = \frac{-3 + 5}{4} = \frac{1}{2}\]

따라서 \(\cos\phi = \frac{1}{2}\), \(\phi = 60°\)이다.

문제 2-3. 구면 위의 미끄럼 운동

질량 \(M\)인 물체(반지름 \(R\)의 오목한 구면)가 마찰 없는 바닥에 있고, 질량 \(m\)이 꼭대기에서 구면을 따라 미끄러져 내려온다. \(M\)은 자유롭게 움직인다.

(가) \(\theta = \pi/2\)일 때 \(m\)과 \(M\)의 속도

운동량 보존(수평 방향, 초기 운동량 0):

\[mv_x + MV = 0\]

여기서 \(v_x\)는 \(m\)의 수평 속도, \(V\)는 \(M\)의 수평 속도이다.

\(\theta = \pi/2\)에서 \(m\)은 구면의 바닥에 있으므로, 구면의 접선이 수평이고 \(m\)의 속도는 수평이다(\(v = v_x\)).

에너지 보존: \(m\)이 높이 \(R\)만큼 내려갔으므로

\[mgR = \frac{1}{2}mv^2 + \frac{1}{2}MV^2\]

운동량 보존: \(mv_x + MV = 0\) → \(V = -\frac{m v_x}{M}\)

\(\theta = \pi/2\)에서 \(m\)의 속도가 수평이면 \(v = v_x\)이다.

\[mgR = \frac{1}{2}mv^2 + \frac{1}{2}M\frac{m^2 v^2}{M^2} = \frac{1}{2}mv^2\left(1 + \frac{m}{M}\right) = \frac{1}{2}mv^2 \cdot \frac{M + m}{M}\]
\[v^2 = \frac{2MgR}{M + m}\]
\[v = \sqrt{\frac{2MgR}{M + m}}\]
\[V = \frac{m}{M}v = \frac{m}{M}\sqrt{\frac{2MgR}{M + m}} = \sqrt{\frac{2m^2 gR}{M(M + m)}}\]

(방향은 \(m\)과 반대)

(나) 임의의 \(\theta\)에서 \(m\)의 속도의 제곱

\(M\)에 대한 \(m\)의 상대 속도를 \(\vec{v}_{\text{rel}}\)이라 하면, \(m\)은 \(M\)에 대해 구면을 따라 미끄러진다.

\(M\)의 속도를 \(V_M\)(수평 방향)이라 하면, 운동량 보존:

\[mv_x + MV_M = 0\]

여기서 \(v_x\)는 \(m\)의 수평 속도이다.

상대 속도의 크기를 \(v_{\text{rel}}\)이라 하면, \(m\)은 \(M\)에 대해 오목한 구면의 원호를 따라 운동한다. \(\theta = 0\)(꼭대기)에서 접선은 연직 방향이고, \(\theta = \pi/2\)(바닥)에서 접선은 수평 방향이므로, 각도 \(\theta\)에서 접선의 성분은

\[v_{\text{rel},x} = v_{\text{rel}}\sin\theta, \quad v_{\text{rel},y} = v_{\text{rel}}\cos\theta\]

\(M\)은 수평으로만 움직이므로 \(m\)의 수직 속도 성분은 상대 속도의 수직 성분과 같다:

\[v_y = v_{\text{rel},y} = v_{\text{rel}}\cos\theta\]

\(m\)의 수평 속도:

\[v_{\text{rel},x} = v_x - V_M = v_x + \frac{m v_x}{M} = v_x \cdot \frac{M + m}{M}\]
\[v_x = \frac{M}{M+m}v_{\text{rel}}\sin\theta\]

\(M\)의 속도:

\[V_M = -\frac{m}{M+m}v_{\text{rel}}\sin\theta\]

\(m\)의 속도의 제곱:

\[v^2 = v_x^2 + v_y^2 = \frac{M^2}{(M+m)^2}v_{\text{rel}}^2\sin^2\theta + v_{\text{rel}}^2\cos^2\theta\]

에너지 보존:

\[mgR(1 - \cos\theta) = \frac{1}{2}mv^2 + \frac{1}{2}MV_M^2\]
\[\frac{1}{2}MV_M^2 = \frac{1}{2}M \cdot \frac{m^2}{(M+m)^2}v_{\text{rel}}^2\sin^2\theta\]

총 운동에너지:

\[\frac{1}{2}m\left[\frac{M^2}{(M+m)^2}\sin^2\theta + \cos^2\theta\right]v_{\text{rel}}^2 + \frac{1}{2}\frac{Mm^2}{(M+m)^2}\sin^2\theta \cdot v_{\text{rel}}^2\]
\[= \frac{1}{2}v_{\text{rel}}^2\left[\frac{mM^2\sin^2\theta}{(M+m)^2} + m\cos^2\theta + \frac{Mm^2\sin^2\theta}{(M+m)^2}\right]\]
\[= \frac{1}{2}v_{\text{rel}}^2\left[\frac{mM(M+m)\sin^2\theta}{(M+m)^2} + m\cos^2\theta\right]\]
\[= \frac{1}{2}v_{\text{rel}}^2 m\left[\frac{M\sin^2\theta}{M+m} + \cos^2\theta\right]\]
\[= \frac{1}{2}v_{\text{rel}}^2 m\left[\frac{M\sin^2\theta + (M+m)\cos^2\theta}{M+m}\right]\]
\[= \frac{1}{2}v_{\text{rel}}^2 m\left[\frac{M + m\cos^2\theta}{M+m}\right]\]

에너지 보존:

\[mgR(1-\cos\theta) = \frac{1}{2}v_{\text{rel}}^2 m \cdot \frac{M + m\cos^2\theta}{M + m}\]
\[v_{\text{rel}}^2 = \frac{2gR(1-\cos\theta)(M+m)}{M + m\cos^2\theta}\]

이제 \(m\)의 속도의 제곱:

\[v^2 = v_x^2 + v_y^2 = \frac{M^2\sin^2\theta}{(M+m)^2}v_{\text{rel}}^2 + \cos^2\theta \cdot v_{\text{rel}}^2\]
\[= v_{\text{rel}}^2\left[\frac{M^2\sin^2\theta + (M+m)^2\cos^2\theta}{(M+m)^2}\right]\]
\[= \frac{2gR(1-\cos\theta)(M+m)}{M + m\cos^2\theta} \cdot \frac{M^2\sin^2\theta + (M+m)^2\cos^2\theta}{(M+m)^2}\]
\[= \frac{2gR(1-\cos\theta)\left[M^2\sin^2\theta + (M+m)^2\cos^2\theta\right]}{(M+m)(M + m\cos^2\theta)}\]

검증: \(\theta = \pi/2\)를 대입하면 \(\cos(\pi/2) = 0\), \(\sin(\pi/2) = 1\)이므로

\[v^2 = \frac{2gR \cdot 1 \cdot M^2}{(M+m) \cdot M} = \frac{2MgR}{M+m}\]

이는 (가)의 결과와 일치한다.
