편광과 이중슬릿 간섭

이 문제는 영의 이중슬릿 실험에 편광판을 추가하여, 빛의 간섭과 편광의 관계를 분석하는 문제이다.

기본 설정: 파장 \(\lambda\)인 빛이 \(45°\) 편광판을 통과하여 \(S_0\)를 지나고, 이중슬릿 \(S_1\), \(S_2\)를 통과하여 스크린에 간섭무늬를 형성한다. 슬릿 간격 \(d\), 스크린까지 거리 \(L\), 첫 번째 극소 위치 \(x_1 = \frac{L\lambda}{2d}\).

(2-1) \(S_1\), \(S_2\) 앞에 모두 \(y\)방향 편광판을 놓은 경우

\(45°\) 편광판을 통과한 빛의 전기장은 \(x\)성분과 \(y\)성분을 동일한 크기로 갖는다. 이를 다음과 같이 쓸 수 있다:

\[\vec{E} = E_0 \cos(\omega t)(\hat{x} + \hat{y}) / \sqrt{2}\]

\(S_1\), \(S_2\) 앞에 \(y\)방향 편광판을 놓으면, 두 슬릿을 통과하는 빛은 모두 \(y\)방향 편광이 된다. 각 슬릿을 통과하는 빛의 세기는 원래의 \(\frac{1}{2}\)이다 (말뤼스 법칙: \(I = I_0 \cos^2 45° = \frac{I_0}{2}\)).

두 슬릿을 통과한 빛은 동일한 편광 방향(\(y\))을 가지므로 완전히 간섭할 수 있다. 따라서 원래와 동일한 간섭무늬 패턴이 나타나되, 전체 세기가 절반으로 줄어든다.

극대 위치: \(x = \frac{m \lambda L}{d}\) (\(m = 0, \pm 1, \pm 2, \ldots\))

극소 위치: \(x = \frac{(m + \frac{1}{2}) \lambda L}{d}\)

\[I(x) = \frac{I_0}{2} \cos^2\left(\frac{\pi d x}{\lambda L}\right)\]

그래프는 원래 [그림 2-3]과 동일한 형태이지만 최대 세기가 \(\frac{I_0}{2}\)이다.

(2-2) \(S_1\) 앞에 \(y\)방향, \(S_2\) 앞에 \(x\)방향 편광판을 놓은 경우

\(S_1\)을 통과한 빛은 \(y\)방향 편광, \(S_2\)를 통과한 빛은 \(x\)방향 편광이다. 두 빛의 편광 방향이 서로 수직이다.

서로 수직으로 편광된 두 파동은 간섭하지 않는다. 이는 간섭이란 두 파동의 중첩에서 비롯되는데, 전기장 성분이 서로 다른 방향이면 세기의 교차항이 0이 되기 때문이다:

\[I = |\vec{E_1} + \vec{E_2}|^2 = |E_1|^2 + |E_2|^2 + 2\vec{E_1} \cdot \vec{E_2}\]

\(\vec{E_1} \perp \vec{E_2}\)이면 \(\vec{E_1} \cdot \vec{E_2} = 0\)이므로 간섭항이 사라진다.

따라서 스크린에는 간섭무늬가 나타나지 않고, 위치에 무관하게 균일한 세기의 빛이 관측된다.

\[I(x) = \frac{I_0}{8} + \frac{I_0}{8} = \frac{I_0}{4} = \text{const.}\]

각 슬릿에서 나오는 빛의 세기가 \(\frac{I_0}{8}\)이고(원래 단일 슬릿 세기 \(\frac{I_0}{4}\)에서 편광판을 통과하며 \(\frac{1}{2}\)), 두 빛이 간섭 없이 단순히 더해지므로 전체 세기는 \(\frac{I_0}{4}\)로 균일하다.

(2-3) 문제 2-2 상태에서 스크린에 \(45°\) 편광판을 추가한 경우

\(S_1\)에서 오는 \(y\)방향 편광 빛이 \(45°\) 편광판을 통과하면, \(45°\) 방향 성분만 남는다:

\[E_1' = \frac{E_1}{\sqrt{2}}\]

\(S_2\)에서 오는 \(x\)방향 편광 빛이 \(45°\) 편광판을 통과하면, 역시 \(45°\) 방향 성분만 남는다:

\[E_2' = \frac{E_2}{\sqrt{2}}\]

이제 두 빛은 모두 같은 편광 방향(\(45°\))을 가지므로 간섭이 가능해진다.

\(S_1\)을 통과한 빛의 원래 전기장이 \(E_1 = E_a \cos(\omega t)\)이고, \(S_2\)를 통과한 빛의 전기장이 \(E_2 = E_a \cos(\omega t + \delta)\)라 하자 (여기서 \(\delta = \frac{2\pi d x}{\lambda L}\)는 경로차에 의한 위상차).

그런데 \(y\)방향 편광이 \(45°\) 편광판을 통과할 때와 \(x\)방향 편광이 \(45°\) 편광판을 통과할 때, \(45°\) 방향으로의 사영 부호가 다르다. \(45°\) 편광 방향을 \(\hat{n} = \frac{1}{\sqrt{2}}(\hat{x} + \hat{y})\)라 하면:

- \(S_1\)(\(y\)편광): \(\hat{y} \cdot \hat{n} = \frac{1}{\sqrt{2}}\)

- \(S_2\)(\(x\)편광): \(\hat{x} \cdot \hat{n} = \frac{1}{\sqrt{2}}\)

두 사영의 크기가 같고 부호도 같으므로, 합성 전기장은:

\[E' = \frac{E_a}{\sqrt{2}}[\cos(\omega t) + \cos(\omega t + \delta)]\]

세기는:

\[I(x) \propto \cos^2\left(\frac{\delta}{2}\right) = \cos^2\left(\frac{\pi d x}{\lambda L}\right)\]

따라서 간섭무늬가 다시 나타나며, 원래의 간섭무늬와 동일한 패턴을 보인다. 다만 전체 세기는 원래보다 줄어든다. 각 슬릿에서 세기가 \(\frac{I_0}{8}\)이고, \(45°\) 편광판을 통과하며 다시 \(\frac{1}{2}\)이 되므로, 각 성분의 세기는 \(\frac{I_0}{16}\)이다.

최대 세기(보강 간섭): \(I_{\max} = 4 \times \frac{I_0}{16} = \frac{I_0}{4}\) (진폭이 더해지므로 세기는 4배)

\[I_{\max} = \frac{I_0}{4}, \quad I_{\min} = 0\]

그래프는 [그림 2-3]과 동일한 형태의 간섭무늬이되, 최대 세기가 \(\frac{I_0}{4}\)이다.

(2-4) 문제 2-2 장치에서 광자를 한 개씩 보내는 경우 - 경로 식별 가능 여부

알아낼 수 있다. 문제 2-2의 장치에서 \(S_1\) 앞에는 \(y\)방향 편광판, \(S_2\) 앞에는 \(x\)방향 편광판이 있다. 따라서:

- 광자가 \(S_1\)을 통과했다면 \(y\)방향 편광 상태

- 광자가 \(S_2\)를 통과했다면 \(x\)방향 편광 상태

스크린에 도착한 광자의 편광 상태를 측정하면(\(y\)편광인지 \(x\)편광인지), 해당 광자가 어느 슬릿을 통과했는지 100% 확정할 수 있다. 이것이 바로 "경로 정보(which-path information)"가 존재하는 상황이다.

경로 정보가 완전히 존재하면 파동의 간섭은 사라진다. 이는 문제 2-2에서 간섭무늬가 나타나지 않는 것과 일치하며, 양자역학의 상보성 원리(complementarity principle)로 설명된다: 입자의 경로 정보와 파동의 간섭무늬는 동시에 완전하게 관측될 수 없다.

(2-5) 문제 2-3 장치에서 광자를 한 개씩 보내는 경우 - 경로 식별 가능 여부

알아낼 수 없다. 문제 2-3의 장치에서는 스크린 앞에 \(45°\) 편광판이 추가되어 있다. 광자가 이 편광판을 통과하면, 원래의 편광 정보(\(y\) 또는 \(x\))가 소멸되고 모두 \(45°\) 편광 상태가 된다.

- \(S_1\)을 통과한 \(y\)편광 광자 \(\rightarrow\) \(45°\) 편광판 통과 \(\rightarrow\) \(45°\) 편광

- \(S_2\)를 통과한 \(x\)편광 광자 \(\rightarrow\) \(45°\) 편광판 통과 \(\rightarrow\) \(45°\) 편광

스크린에 도착한 광자는 모두 \(45°\) 편광이므로, 편광 측정을 해도 어느 슬릿을 통과했는지 구별할 수 없다. 경로 정보가 사라졌으므로, 양자역학의 상보성 원리에 의해 간섭무늬가 다시 나타난다.

이는 문제 2-3에서 \(45°\) 편광판을 추가했을 때 간섭무늬가 복원된 결과와 정확히 일치한다. \(45°\) 편광판은 "양자 지우개(quantum eraser)" 역할을 하여, 경로 정보를 지우고 간섭 현상을 복원시킨다.
