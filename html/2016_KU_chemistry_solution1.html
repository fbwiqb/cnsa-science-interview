산화수와 탄소 화합물의 구조

[문제 1] 아세트산의 두 탄소 원자의 산화수 비교

아세트산(\(\text{CH}_3\text{COOH}\))의 구조식:

\[\text{H}_3\text{C}-\text{C}(=\text{O})(-\text{OH})\]

전기음성도: \(\text{H} = 2.1\), \(\text{C} = 2.5\), \(\text{O} = 3.5\)

1번 탄소 (산소와 결합한 쪽, \(-\text{COOH}\)의 C):

1번 탄소의 결합:

- \(\text{C=O}\) 이중결합: \(\text{O}\)의 전기음성도(\(3.5\)) > \(\text{C}\)의 전기음성도(\(2.5\))이므로, 공유 전자쌍 4개(이중결합)가 모두 \(\text{O}\)로 이동 → 산화수 기여: \(+2\)

- \(\text{C-OH}\) 단일결합: \(\text{O}\)의 전기음성도 > \(\text{C}\)의 전기음성도이므로, 공유 전자쌍 2개가 \(\text{O}\)로 이동 → 산화수 기여: \(+1\)

- \(\text{C-C}\) 단일결합: 같은 원소이므로 전자쌍을 균등 분배 → 산화수 기여: \(0\)

1번 탄소의 산화수: \(+2 + 1 + 0 = \boxed{+3}\)

2번 탄소 (수소와 결합한 쪽, \(\text{CH}_3\)-의 C):

2번 탄소의 결합:

- \(\text{C-H}\) 결합 3개: \(\text{C}\)의 전기음성도(\(2.5\)) > \(\text{H}\)의 전기음성도(\(2.1\))이므로, 각 결합에서 공유 전자쌍이 \(\text{C}\)로 이동 → 산화수 기여: \(3 \times (-1) = -3\)

- \(\text{C-C}\) 단일결합: 같은 원소이므로 균등 분배 → 산화수 기여: \(0\)

2번 탄소의 산화수: \(-3 + 0 = \boxed{-3}\)

검증: 아세트산 \(\text{C}_2\text{H}_4\text{O}_2\)에서 산화수의 합 = 0

\((+3) + (-3) + 4(+1) + 2(-2) = 0\) ✓

따라서 1번 탄소(산화수 \(+3\))는 2번 탄소(산화수 \(-3\))보다 산화수가 훨씬 크다. 이는 1번 탄소가 전기음성도가 큰 산소 원자들과 결합하여 전자를 빼앗겨 높은 산화수를 가지고, 2번 탄소는 전기음성도가 작은 수소 원자들과 결합하여 전자를 얻어 낮은 산화수를 가지기 때문이다.

[문제 2] 탄소의 산화수가 \(+2\), \(0\), \(-2\)인 가장 간단한 C, H, O 화합물

탄소 산화수 \(+2\)인 화합물:

포름알데히드(\(\text{HCHO}\), \(\text{CH}_2\text{O}\)):

구조식: \(\text{H}-\text{C}(=\text{O})-\text{H}\)

- \(\text{C=O}\): 산화수 기여 \(+2\)

- \(\text{C-H}\) 2개: 산화수 기여 \(-2\)

탄소 산화수 = \(+2 + (-2) = 0\) (이것은 산화수 \(0\)이다)

다시 계산한다. 탄소 산화수 \(+2\)인 화합물:

포름산(\(\text{HCOOH}\), \(\text{CH}_2\text{O}_2\)):

구조식: \(\text{H}-\text{C}(=\text{O})(-\text{OH})\)

- \(\text{C=O}\): 산화수 기여 \(+2\)

- \(\text{C-OH}\): 산화수 기여 \(+1\)

- \(\text{C-H}\): 산화수 기여 \(-1\)

탄소 산화수 = \(+2 + 1 + (-1) = +2\) ✓

분자량: \(12 + 2(1) + 2(16) = 46\)

탄소 산화수 \(0\)인 화합물:

포름알데히드(\(\text{HCHO}\), \(\text{CH}_2\text{O}\)):

구조식: \(\text{H}_2\text{C=O}\)

- \(\text{C=O}\): 산화수 기여 \(+2\)

- \(\text{C-H}\) 2개: 산화수 기여 \(-2\)

탄소 산화수 = \(+2 + (-2) = 0\) ✓

분자량: \(12 + 2(1) + 16 = 30\)

탄소 산화수 \(-2\)인 화합물:

메탄올(\(\text{CH}_3\text{OH}\), \(\text{CH}_4\text{O}\)):

구조식: \(\text{H}_3\text{C-OH}\)

- \(\text{C-OH}\): 산화수 기여 \(+1\)

- \(\text{C-H}\) 3개: 산화수 기여 \(-3\)

탄소 산화수 = \(+1 + (-3) = -2\) ✓

분자량: \(12 + 4(1) + 16 = 32\)

탄소의 산화수가 \(+2\)보다 더 큰 값을 가질 수 있는지:

C, H, O 세 원소로 이루어진 중성 탄소화합물에서 탄소는 최대 4개의 결합을 형성한다. 탄소의 산화수가 최대가 되려면 전기음성도가 큰 원소(O)와 가능한 많이 결합해야 한다.

탄소가 4개의 결합 모두 산소와 결합하면: \(\text{C}\) 산화수 = \(+4\) (예: \(\text{CO}_2\))

그러나 \(\text{CO}_2\)에는 수소가 없으므로 C, H, O 세 원소가 모두 포함되어야 하는 조건을 만족하지 않는다.

탄산(\(\text{H}_2\text{CO}_3\))의 경우:

구조식: \((\text{HO})-\text{C}(=\text{O})(-\text{OH})\)

- \(\text{C=O}\): 산화수 기여 \(+2\)

- \(\text{C-OH}\) 2개: 산화수 기여 \(+2\)

탄소 산화수 = \(+2 + 2 = +4\) ✓

따라서 C, H, O로 이루어진 중성 탄소화합물에서 탄소의 산화수는 \(+2\)보다 더 큰 값인 \(+4\)까지 가질 수 있다. 이는 탄소가 산소와의 결합 수를 늘림으로써 가능하며, 탄산(\(\text{H}_2\text{CO}_3\))이 그 예이다.
