해설 3

가)

공이 수평방향으로는 등속운동 수직방향으로는 등가속도 운동을 한다. 따라서 시간 \(t\)가 흐른 후,

공은 수평방향으로 거리가 \(x = vt\)이고, 수직방향으로는 아래로 움직이므로 \(y = -\dfrac{1}{2}gt^2\) 이다.

경사면에 부딪치는 조건은 \(x = -y\) 이다. 이것을 이용하면

\(vt_P = \dfrac{1}{2}gt_P^2\)에서 \(t_P = \dfrac{2v}{g}\) 일 때 경사면에 부딪치게 된다.

따라서 점 P의 x좌표는 \(x = vt_P = v\dfrac{2v}{g} = \dfrac{2v^2}{g}\) 이다.

나)

점 P에 부딪히기 직전 속도를 구하면,

속도의 x 성분 \(v_{Px} = v\)로 변화가 없고, 속도의 y 성분 \(v_{Py} = -gt_P = -g\dfrac{2v}{g} = -2v\)이다.

점 P에 부딪칠 때, 완전탄성충돌이므로 충돌전후 속도벡터의 길이는 변화가 없고, 방향만 바뀐다. 속도벡터는 점 P에 대해서 입사각과 반사각이 같아야 한다. 즉 경사면에 세운 수선에 대해서 부딪치기 전 수선과 속도벡터가 이루는 각이 부딪치고 난 후 수선과 속도벡터가 이루는 각과 같아야 한다.

그림에서 부딪치고 난 직후 공의 속도는 x 성분은 \(v_{xf} = 2v\)이고 y성분은 \(v_{yf} = -v\)이다.

다)

P점에서 출발할 때, 속도의 x성분은 \(v_{xf} = 2v\)이고, 속도의 y성분은 \(v_{yf} = -v\)이다.

P점에서 x방향으로 이동변위는 \(x_2 = 2vt\)이고, y방향으로 이동변위는 \(y_2 = -vt - \dfrac{1}{2}gt^2\)이다.

경사면에 부딪치기 위해서는

\(x_2 = -y_2\)이므로, \(2vt_Q = vt_Q + \dfrac{1}{2}gt_Q^2\)이 되어야 한다.

따라서

\[t_Q = \frac{2v}{g}\]

이다.

이 시간동안 P점에서 x 방향 이동변위는 \(x_2 = 2vt_Q = \dfrac{4v^2}{g}\)이고,

y 방향 이동변위는 \(y_2 = -vt_Q - \dfrac{1}{2}gt_Q^2 = -v\dfrac{2v}{g} - \dfrac{1}{2}g\dfrac{4v^2}{g^2} = -\dfrac{4v^2}{g}\) 이다.

Q점의 좌표는 P점의 좌표에서 이동한 변위를 고려해야한다.

따라서 Q점\(= \left(\dfrac{2v^2}{g} + \dfrac{4v^2}{g},\, -\dfrac{2v^2}{g} - \dfrac{4v^2}{g}\right) = \left(\dfrac{6v^2}{g},\, -\dfrac{6v^2}{g}\right)\)이다.

Q점의 x좌표는 \(\dfrac{6v^2}{g}\) 이다.
