2-1. 전자 11개를 채울 때 마지막 전자의 에너지 준위

1차원 무한 퍼텐셜 우물의 에너지 준위: \(E_n = n^2 E_0\) (\(n = 1, 2, 3, \ldots\))

파울리 배타 원리에 의해, 각 에너지 준위에 스핀 양자수 \(+\frac{1}{2}\)과 \(-\frac{1}{2}\) 두 개의 전자가 들어갈 수 있다.

전자 배치:

- \(n = 1\) (\(E_1 = E_0 = 0.1\) eV): 2개

- \(n = 2\) (\(E_2 = 4E_0 = 0.4\) eV): 2개

- \(n = 3\) (\(E_3 = 9E_0 = 0.9\) eV): 2개

- \(n = 4\) (\(E_4 = 16E_0 = 1.6\) eV): 2개

- \(n = 5\) (\(E_5 = 25E_0 = 2.5\) eV): 2개

- \(n = 6\) (\(E_6 = 36E_0 = 3.6\) eV): 1개 (11번째 전자)

마지막 전자가 들어가는 에너지 준위는 \(E_6 = 36E_0 = 3.6\) eV이다.

2-2. 자발 방출로 나올 수 있는 빛의 에너지

빛의 에너지 범위: 0.5~1.5 eV (균일 분포)

가능한 전이를 분석한다. 기저 상태 배치에서 전자가 빛을 흡수하여 들뜬 상태로 올라간 후 원래 준위로 돌아오며 빛을 방출한다.

흡수 가능한 전이 (에너지가 0.5~1.5 eV 범위):

각 준위 사이의 에너지 차이:

- \(E_2 - E_1 = 3E_0 = 0.3\) eV (범위 밖)

- \(E_3 - E_2 = 5E_0 = 0.5\) eV (경계, 포함)

- \(E_4 - E_3 = 7E_0 = 0.7\) eV

- \(E_5 - E_4 = 9E_0 = 0.9\) eV

- \(E_6 - E_5 = 11E_0 = 1.1\) eV

- \(E_7 - E_6 = 13E_0 = 1.3\) eV

- \(E_3 - E_1 = 8E_0 = 0.8\) eV

- \(E_4 - E_2 = 12E_0 = 1.2\) eV

- \(E_5 - E_3 = 16E_0 = 1.6\) eV (범위 밖)

- \(E_4 - E_1 = 15E_0 = 1.5\) eV (경계, 포함)

- \(E_5 - E_2 = 21E_0 = 2.1\) eV (범위 밖)

단, 전자가 점유된 준위에서만 흡수가 가능하고, 빈 준위로만 전이할 수 있다.

기저 상태에서 \(n = 1\) ~ \(n = 5\)는 완전히 채워져 있고, \(n = 6\)에는 1개, \(n \geq 7\)은 비어있다.

가능한 흡수 전이(0.5~1.5 eV 범위):

- \(n=5 \to n=6\): \(E_6 - E_5 = 11E_0 = 1.1\) eV (가능, \(n=6\)에 빈자리 1개)

- \(n=4 \to n=6\): 이미 \(n=6\)에 빈자리 1개 (\(E_6 - E_4 = 20E_0 = 2.0\) eV, 범위 밖)

- \(n=6 \to n=7\): \(E_7 - E_6 = 13E_0 = 1.3\) eV (가능, \(n=6\)에 전자 1개, \(n=7\) 비어있음)

- \(n=5 \to n=7\): \(E_7 - E_5 = (49-25)E_0 = 24E_0 = 2.4\) eV (범위 밖)

- \(n=3 \to n=6\): \((36-9)E_0 = 27E_0 = 2.7\) eV (범위 밖)

- \(n=4 \to n=5\): 이미 \(n=5\) 채워짐 (불가)

- \(n=1 \to n=6\): \((36-1)E_0 = 3.5\) eV (범위 밖)

결국 가능한 흡수 전이:

1. \(n=5 \to n=6\): 1.1 eV 흡수 → 1.1 eV 방출

2. \(n=6 \to n=7\): 1.3 eV 흡수 → 1.3 eV 방출

따라서 상자에서 나올 수 있는 빛의 에너지는 1.1 eV와 1.3 eV이다.

2-3. 광전자의 운동 에너지

일함수 \(W = 1.2\) eV

방출된 빛의 에너지가 1.1 eV와 1.3 eV이다.

광전효과 조건: 빛의 에너지 > 일함수

- 1.1 eV: \(1.1 < 1.2\)이므로 광전자 방출 불가

- 1.3 eV: \(1.3 > 1.2\)이므로 광전자 방출 가능

광전자의 최대 운동 에너지:

\[K = h\nu - W = 1.3 - 1.2 = 0.1 \text{ eV}\]

2-4. 이중 슬릿 실험

(a) 하나의 전자가 이중 슬릿을 지나 스크린에서 발견될 확률 분포:

전자의 확률 분포는 이중 슬릿에 의한 간섭 패턴을 따른다. 중앙에서 최대 확률이 나타나고, 양쪽으로 밝은 무늬(극대)와 어두운 무늬(극소)가 교대로 나타나는 간섭 패턴의 형태이다. 보강 간섭 조건은 \(d\sin\theta = n\lambda\) (\(n = 0, \pm 1, \pm 2, \ldots\))이다.

(b) 전자 한 개가 스크린에 도달한 결과:

전자 한 개는 스크린의 특정한 한 점에서만 발견된다. 입자적 성질에 의해 점으로 관측되며, 어느 위치에서 발견될지는 확률적이다. 한 개의 전자로는 간섭 패턴을 관측할 수 없다.

(c) 무수히 많이 반복한 결과:

실험을 매우 많이 반복하면, 각 전자가 하나씩 도달한 점들이 축적되어 간섭 패턴이 나타난다. 중앙이 가장 밝고 양쪽으로 밝고 어두운 줄무늬가 교대하는 이중 슬릿 간섭 무늬가 형성된다. 이는 전자의 파동성을 보여주는 결과이다.
