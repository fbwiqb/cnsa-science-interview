해설 5

평행판 축전기와 유전체 — 에너지 비교

주어진 조건:

- 공기로 채워진 경우의 전기용량: \(C_{\text{공기}}\)

- 나무토막으로 채워진 경우의 전기용량: \(C_{\text{나무}}\)

- 나무의 유전율 \(\kappa > 1\)이므로: \(C_{\text{나무}} = \kappa C_{\text{공기}} > C_{\text{공기}}\)

(1) 전하 \(Q\)로 충전할 때 더 많은 에너지가 필요한 경우

축전기에 저장되는 에너지는:

\[U = \frac{Q^2}{2C}\]

전하 \(Q\)가 동일하므로, 전기용량 \(C\)가 작을수록 에너지가 크다.

\(C_{\text{공기}} < C_{\text{나무}}\)이므로:

\[U_{\text{공기}} = \frac{Q^2}{2C_{\text{공기}}} > \frac{Q^2}{2C_{\text{나무}}} = U_{\text{나무}}\]
\[\boxed{\text{공기로 채워진 경우에 더 많은 에너지가 필요하다}}\]

(2) 나무토막을 빼내는 데 필요한 일

전하 \(Q\)로 충전된 축전기에서 나무토막을 빼내면, 전하 \(Q\)는 보존되면서(외부 회로가 연결되지 않았으므로) 전기용량이 \(C_{\text{나무}}\)에서 \(C_{\text{공기}}\)로 감소한다.

나무토막을 빼내기 전의 에너지:

\[U_{\text{전}} = \frac{Q^2}{2C_{\text{나무}}}\]

나무토막을 빼낸 후의 에너지:

\[U_{\text{후}} = \frac{Q^2}{2C_{\text{공기}}}\]

에너지 보존에 의해, 나무토막을 빼내는 데 필요한 일은:

\[W = U_{\text{후}} - U_{\text{전}} = \frac{Q^2}{2C_{\text{공기}}} - \frac{Q^2}{2C_{\text{나무}}}\]
\[\boxed{W = \frac{Q^2}{2}\left(\frac{1}{C_{\text{공기}}} - \frac{1}{C_{\text{나무}}}\right)}\]

\(C_{\text{공기}} < C_{\text{나무}}\)이므로 \(W > 0\)이다. 즉, 나무토막을 빼내려면 외부에서 일을 해주어야 한다.

(3) 전위차 \(V\)로 충전할 때 더 많은 에너지가 필요한 경우

전위차 \(V\)가 동일할 때 축전기에 저장되는 에너지는:

\[U = \frac{1}{2}CV^2\]

전기용량 \(C\)가 클수록 에너지가 크다.

\(C_{\text{나무}} > C_{\text{공기}}\)이므로:

\[U_{\text{나무}} = \frac{1}{2}C_{\text{나무}}V^2 > \frac{1}{2}C_{\text{공기}}V^2 = U_{\text{공기}}\]
\[\boxed{\text{나무토막으로 채워진 경우에 더 많은 에너지가 필요하다}}\]

(1)번과 (3)번의 결과가 반대인 이유는 조건이 다르기 때문이다. (1)은 전하량을 고정했고, (3)은 전위차를 고정했다.
