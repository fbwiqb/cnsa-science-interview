광전효과와 전기 퍼텐셜 에너지

(물리 1)

i) 광전자의 속도비

금속판 \(A\)의 문턱 진동수가 \(f_A\), 금속판 \(B\)의 문턱 진동수가 \(f_B\)이고, 진동수 \(f\)인 빛을 쪼였을 때 광전효과 방정식은 다음과 같다.

금속판 \(A\): \(hf - hf_A = \frac{1}{2}mv_A^2\)

금속판 \(B\): \(hf - hf_B = \frac{1}{2}mv_B^2\)

속도비의 제곱은 다음과 같다.

\[\frac{v_A^2}{v_B^2} = \frac{hf - hf_A}{hf - hf_B} = \frac{f - f_A}{f - f_B}\]

따라서 속도비는 다음과 같다.

\[\boxed{\frac{v_A}{v_B} = \sqrt{\frac{f - f_A}{f - f_B}}}\]

ii) \(-q\) 전하가 탈출하는 데 필요한 최소 진동수

\(N\)개의 \(+Q\) 전하가 반지름 \(R\)인 원 위에 일정한 간격으로 놓여 있고, \(-q\) 전하가 원의 중심에 있다. 원의 중심에서 각 \(+Q\) 전하까지의 거리는 모두 \(R\)이다.

\(-q\) 전하가 원의 중심에서 갖는 총 전기 퍼텐셜 에너지는 다음과 같다.

\[E_P = -N \cdot k_e\frac{qQ}{R} = -\frac{Nk_eqQ}{R}\]

\(-q\) 전하가 \(+Q\) 전하들로부터 무한히 멀리 탈출하면 전기 퍼텐셜 에너지는 0이 된다. 따라서 탈출에 필요한 최소 에너지는 다음과 같다.

\[E_{\min} = 0 - E_P = \frac{Nk_eqQ}{R}\]

광자의 에너지가 이 최소 에너지 이상이어야 하므로 다음과 같다.

\[hf \geq \frac{Nk_eqQ}{R}\]

최소 진동수는 다음과 같다.

\[\boxed{f_{\min} = \frac{Nk_eqQ}{hR}}\]

(물리 2) \(E_A = E_C\), \(E_B = E_D\)를 만족하는 \(-q\) 전하의 위치의 개수

정사각형의 네 꼭짓점에 양전하 \(A\), \(B\), \(C\), \(D\)가 놓여 있고, 전하량의 비는 \(Q_A : Q_B : Q_C : Q_D = 1 : 2 : 3 : 4\)이다.

\(-q\) 전하가 위치 \((x, y)\)에 놓였을 때, 각 꼭짓점의 전하에 의한 전기 퍼텐셜 에너지는 다음과 같다.

\[E_A = -k_e\frac{qQ_A}{r_A}, \quad E_B = -k_e\frac{qQ_B}{r_B}, \quad E_C = -k_e\frac{qQ_C}{r_C}, \quad E_D = -k_e\frac{qQ_D}{r_D}\]

여기서 \(r_A, r_B, r_C, r_D\)는 \(-q\) 전하로부터 각 꼭짓점까지의 거리이다.

\(E_A = E_C\) 조건:

\[\frac{Q_A}{r_A} = \frac{Q_C}{r_C} \quad \Rightarrow \quad \frac{1}{r_A} = \frac{3}{r_C} \quad \Rightarrow \quad r_C = 3r_A\]

\(E_B = E_D\) 조건:

\[\frac{Q_B}{r_B} = \frac{Q_D}{r_D} \quad \Rightarrow \quad \frac{2}{r_B} = \frac{4}{r_D} \quad \Rightarrow \quad r_D = 2r_B\]

정사각형의 한 변의 길이를 \(a\)라 하고, 꼭짓점을 다음과 같이 설정한다.

- \(A = (0, a)\) (위쪽 왼쪽)

- \(B = (0, 0)\) (아래쪽 왼쪽)

- \(C = (a, 0)\) (아래쪽 오른쪽)

- \(D = (a, a)\) (위쪽 오른쪽)

\(r_C = 3r_A\)인 점의 자취는 아폴로니우스의 원이다. 점 \(A\)와 \(C\)에 대해 \(\frac{r_C}{r_A} = 3\)인 점들의 집합은 원을 이룬다.

마찬가지로 \(r_D = 2r_B\)인 점의 자취도 아폴로니우스의 원이다.

두 아폴로니우스 원의 교점의 개수가 답이 된다.

\(r_C = 3r_A\) 조건에서 아폴로니우스의 원: \(A\)와 \(C\)를 잇는 선분을 \(3:1\)로 내분하는 점과 외분하는 점을 지름의 양 끝으로 하는 원이다.

\(r_D = 2r_B\) 조건에서 아폴로니우스의 원: \(B\)와 \(D\)를 잇는 선분을 \(2:1\)로 내분하는 점과 외분하는 점을 지름의 양 끝으로 하는 원이다.

두 원은 일반적으로 0, 1, 또는 2개의 교점을 갖는다. 정사각형을 포함하는 평면에서 두 아폴로니우스 원은 서로 교차하므로, 두 조건을 동시에 만족하는 점의 개수는 다음과 같다.

\[\boxed{2 \text{개}}\]
