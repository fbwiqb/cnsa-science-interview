(화학 1)

i) \( ext{O}^{2-}\), \( ext{F}^-\), \( ext{Ne}\), \( ext{Na}^+\), \( ext{Mg}^{2+}\)은 모두 전자 수가 10개인 등전자 종이다.

등전자 종에서는 양성자 수(핵전하)가 클수록 전자를 더 강하게 끌어당기므로 반지름이 작아진다.

각 종의 양성자 수: \( ext{O}^{2-}\)(8개), \( ext{F}^-\)(9개), \( ext{Ne}\)(10개), \( ext{Na}^+\)(11개), \( ext{Mg}^{2+}\)(12개)

따라서 반지름이 작은 것부터 차례로:

\[oxed{ ext{Mg}^{2+} < ext{Na}^+ < ext{Ne} < ext{F}^- < ext{O}^{2-}}\]

양성자 수가 많을수록 유효 핵전하가 커서 전자를 더 강하게 잡아당기므로 이온 반지름이 작아진다.

ii) 세 화합물의 격자 에너지 비교

격자 에너지: \(U = -k\dfrac{Q_1 Q_2}{d_1}\)

\( ext{NaCl}\): \(|Q_1 Q_2| = 1\), 이온 간 거리 \(d_1\)

\( ext{MgCl}_2\): \(|Q_1 Q_2| = 2\) (\( ext{Mg}^{2+}\)의 전하가 \(+2\)), \(d_2 < d_1\) (\( ext{Mg}^{2+}\)의 반지름 < \( ext{Na}^+\)의 반지름)

\( ext{MgO}\): \(|Q_1 Q_2| = 4\) (\( ext{Mg}^{2+}\)는 \(+2\), \( ext{O}^{2-}\)는 \(-2\)), \(d_3 < d_2\) (\( ext{O}^{2-}\)의 반지름 < \( ext{Cl}^-\)의 반지름)

격자 에너지의 크기: \(|U( ext{MgO})| > |U( ext{MgCl}_2)| > |U( ext{NaCl})|\)

에너지-거리 좌표평면에서: \(d_3 < d_2 < d_1\)이고 \(E_3 < E_2 < E_1 < 0\)

\((d_3, E_3)\): \( ext{MgO}\) - 가장 왼쪽, 가장 아래 (에너지가 가장 낮음)

\((d_1, E_1)\): \( ext{NaCl}\) - 가장 오른쪽, 가장 위 (에너지가 가장 덜 낮음)

(화학 2)

에너지: \(E = -\dfrac{lpha}{d} + \dfrac{eta}{d^2}\)

평형 거리에서 \(rac{dE}{dd} = 0\): \(\dfrac{lpha}{d^2} - \dfrac{2eta}{d^3} = 0 \implies d_i = \dfrac{2eta}{lpha}\)

평형 에너지: \(E_i = -\dfrac{lpha^2}{4eta}\)

조건: \(1 \leq d_i \leq 2\), \(E_i \geq -1\)

\(lpha^2 \leq 4eta\)이고 \(eta \leq lpha\) (from \(d_i \geq 1\))

\(eta = rac{lpha^2}{4}\)로 놓으면 \(E_i = -1\), \(d_i = rac{lpha}{2}\)

\(d_i \leq 2\)에서 \(lpha \leq 4\), \(eta \leq lpha\)에서 \(rac{lpha^2}{4} \leq lpha\)이므로 \(lpha \leq 4\)

\(lpha = 4\)일 때 \(eta = 4\): \(lpha + eta = oxed{8}\)
