이중슬릿의 음파 간섭과 물질파

슬릿 사이의 거리 \(d\), 스크린까지의 거리 \(L\) (\(L \gg d\)). 각 슬릿의 폭은 무시한다.

(2-1) 한쪽 끝이 닫힌 관의 기본 진동수와 이중슬릿 극대점 간격

한쪽 끝이 닫힌(폐관) 길이 \(l\)의 관에서는 닫힌 끝이 마디(node), 열린 끝이 배(antinode)가 된다. 기본 진동(1차 공명)에서 관의 길이는 파장의 \(\frac{1}{4}\)에 해당한다:

\[l = \frac{\lambda}{4} \quad \Rightarrow \quad \lambda = 4l\]

기본 진동수:

\[\boxed{f = \frac{v}{\lambda} = \frac{v}{4l}}\]

이 음파의 파장 \(\lambda = 4l\)로 이중슬릿 간섭 실험을 수행하면, 보강 간섭 조건:

\[d\sin\theta = m\lambda \quad (m = 0, \pm 1, \pm 2, \ldots)\]

\(L \gg d\)이므로 \(\sin\theta \approx \tan\theta = \frac{x}{L}\) 근사를 적용:

\[x_m = \frac{m\lambda L}{d} = \frac{4mlL}{d}\]

인접한 극대점 사이의 거리:

\[\boxed{\Delta x = \frac{\lambda L}{d} = \frac{4lL}{d}}\]

(2-2) 음원이 \(v_s\)의 속력으로 스크린을 향해 움직일 때의 극대점 간격 변화

음원이 속력 \(v_s\)로 슬릿을 향해 접근하면, 도플러 효과에 의해 슬릿에서 관측되는 진동수가 변한다:

\[f' = \frac{v}{v - v_s} f = \frac{v}{v - v_s} \cdot \frac{v}{4l} = \frac{v^2}{4l(v - v_s)}\]

슬릿에 도달하는 음파의 파장:

\[\lambda' = \frac{v}{f'} = \frac{4l(v - v_s)}{v}\]

음원이 \(d\)에 비해 충분히 먼 곳에서 접근하므로, 두 슬릿에 도달하는 음파는 거의 평면파로 간주할 수 있고, 두 슬릿에서의 파장은 모두 \(\lambda'\)이다.

슬릿과 스크린 사이에서 음파의 속력은 여전히 \(v\)이다. 슬릿을 통과한 음파의 파장은 \(\lambda'\)이다(도플러 효과는 음원-관측자 사이에서만 발생하고, 슬릿이 음파를 재방출하는 점파원 역할을 하므로, 슬릿에서 스크린까지는 파장 \(\lambda'\)이 유지된다).

스크린 위 극대점 사이의 거리:

\[\boxed{\Delta x' = \frac{\lambda' L}{d} = \frac{4l(v - v_s)L}{vd}}\]

\(v_s > 0\)이므로 \(\lambda' < \lambda\)이고, 따라서 \(\Delta x' < \Delta x\)이다. 극대점 사이의 거리가 줄어든다.

원래 간격과의 비교:

\[\frac{\Delta x'}{\Delta x} = \frac{\lambda'}{\lambda} = \frac{v - v_s}{v} = 1 - \frac{v_s}{v}\]

(2-3) 전압 \(V\)로 가속된 전자의 이중슬릿 간섭 극대점 간격

정지 상태의 전자가 전압 \(V\)로 가속되면, 에너지 보존에 의해:

\[eV = \frac{1}{2}mv^2\]
\[v = \sqrt{\frac{2eV}{m}}\]

드브로이 물질파 파장:

\[\lambda_{\text{dB}} = \frac{h}{mv} = \frac{h}{m\sqrt{\frac{2eV}{m}}} = \frac{h}{\sqrt{2meV}}\]

이중슬릿에서의 간섭 극대점 간격:

\[\boxed{\Delta x = \frac{\lambda_{\text{dB}} L}{d} = \frac{hL}{d\sqrt{2meV}}}\]

전자를 하나씩 보내는 실험을 무수히 반복하면, 개별 전자는 스크린에 점으로 도착하지만, 많은 전자가 축적되면 보강간섭 위치에 전자가 많이, 상쇄간섭 위치에 전자가 적게 분포하는 간섭 패턴이 나타난다.

(2-4) 전자선 결정 회절을 통한 파동성 증명

동일한 에너지를 가진 전자선을 결정에 쏘면, 결정의 규칙적인 원자 배열이 회절 격자 역할을 한다.

결정 내 원자 사이의 간격을 \(d_{\text{crystal}}\)이라 하면, 브래그 회절 조건(Bragg's law)에 의해:

\[2d_{\text{crystal}} \sin\theta = n\lambda_{\text{dB}} \quad (n = 1, 2, 3, \ldots)\]

여기서 \(\theta\)는 결정면과 입사 전자선이 이루는 각도이고, \(\lambda_{\text{dB}}\)는 전자의 드브로이 파장이다.

결정에서 튀어 나온(회절된) 전자의 각도 분포를 측정하면, 특정 각도에서 전자의 수가 극대를 보이는 회절 패턴이 나타난다. 이 패턴은 파동의 보강간섭에 의한 것이다.

핵심 논증:

1. 전자의 운동 에너지로부터 드브로이 파장 \(\lambda_{\text{dB}} = \frac{h}{\sqrt{2mE}}\)를 계산한다.

2. 회절 패턴의 극대 각도 \(\theta\)를 측정한다.

3. 브래그 조건 \(2d_{\text{crystal}}\sin\theta = n\lambda\)에서, X선 회절 등으로 이미 알려진 \(d_{\text{crystal}}\) 값을 사용하여 \(\lambda\)를 계산한다.

4. 이렇게 실험적으로 구한 \(\lambda\)가 드브로이 파장 \(\lambda_{\text{dB}}\)와 일치하면, 전자가 파동으로서 행동한다는 것이 증명된다.

이것이 바로 1927년 데이비슨과 저머(Davisson-Germer)가 수행한 실험의 원리이며, 니켈 결정에 전자를 쏘아 회절 패턴을 관측함으로써 전자의 파동성을 최초로 실험적으로 입증하였다. 회절 극대의 각도에서 계산한 파장이 드브로이가 예측한 \(\lambda = h/p\)와 정확히 일치하였다.

만약 전자가 순수한 입자라면 결정에서 산란된 전자는 모든 방향으로 균일하게 분포해야 하지만, 실제로는 특정 각도에서만 강한 신호가 나타나는 것이 파동성의 결정적 증거이다.
