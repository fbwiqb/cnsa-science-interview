이산화 탄소 환원과 메탄올 합성 전기화학

2-1. 반쪽 반응식 완성 및 전체 반응식

산화 전극 반응식:

물이 산화되어 산소 기체와 수소 이온을 생성한다. 전하 균형을 맞추기 위해 전자를 추가한다:

\[2\text{H}_2\text{O}(\ell) \rightarrow \text{O}_2(g) + 4\text{H}^+(aq) + 4e^-\]

환원 전극 반응식:

\(\text{CO}_2\)가 환원되어 \(\text{CH}_3\text{OH}\)가 생성된다. 산화수 변화: C는 +4에서 -2로 6전자 감소.

원자 균형과 전하 균형을 맞추면:

\[\text{CO}_2(g) + 5\text{H}_2\text{O}(\ell) + 6e^- \rightarrow \text{CH}_3\text{OH}(aq) + 6\text{OH}^-(aq)\]

검증 - 좌변: C 1, O 2+5=7, H 10, 전하 -6 / 우변: C 1, O 1+6=7, H 4+6=10, 전하 -6. 균형이 맞는다.

전체 반응식:

산화 반응을 \(\times\frac{3}{2}\) 하여 전자 6개를 맞춘다:

산화: \(3\text{H}_2\text{O}(\ell) \rightarrow \frac{3}{2}\text{O}_2(g) + 6\text{H}^+(aq) + 6e^-\)

환원: \(\text{CO}_2(g) + 5\text{H}_2\text{O}(\ell) + 6e^- \rightarrow \text{CH}_3\text{OH}(aq) + 6\text{OH}^-(aq)\)

합산: \(\text{CO}_2(g) + 8\text{H}_2\text{O}(\ell) \rightarrow \text{CH}_3\text{OH}(aq) + \frac{3}{2}\text{O}_2(g) + 6\text{H}^+(aq) + 6\text{OH}^-(aq)\)

\(6\text{H}^+ + 6\text{OH}^- = 6\text{H}_2\text{O}\)이므로:

\[\text{CO}_2(g) + 2\text{H}_2\text{O}(\ell) \rightarrow \text{CH}_3\text{OH}(aq) + \frac{3}{2}\text{O}_2(g)\]

정수 계수로:

\[2\text{CO}_2(g) + 4\text{H}_2\text{O}(\ell) \rightarrow 2\text{CH}_3\text{OH}(aq) + 3\text{O}_2(g)\]

2-2. 생성되는 메탄올의 양과 수위 변화

산화 전극 쪽(\(\text{N}_2\) 수조)에서는 \(\text{O}_2\)가 발생하고, 환원 전극 쪽(\(\text{CO}_2\) 수조)에서는 \(\text{CO}_2\)가 소모된다.

전체 반응: \(\text{CO}_2(g) + 2\text{H}_2\text{O}(\ell) \rightarrow \text{CH}_3\text{OH}(aq) + \frac{3}{2}\text{O}_2(g)\)

\(\text{CO}_2\) \(n\) mol이 소모되면 \(\text{O}_2\) \(\frac{3}{2}n\) mol이 생성된다.

산화 전극 수조: \(\text{N}_2\) 기체 위에 \(\text{O}_2\) \(\frac{3}{2}n\) mol 추가 → 압력 증가

환원 전극 수조: \(\text{CO}_2\) \(n\) mol 감소 → 압력 감소

두 수조 사이의 압력 차이 = 0.01 atm

\[\Delta P = P_{\text{산화}} - P_{\text{환원}}\]

산화 수조 압력 증가: \(\frac{3n/2 \times 0.08 \times 300}{24} = \frac{36n}{24} = 1.5n\) atm

환원 수조 압력 감소: \(\frac{n \times 0.08 \times 300}{24} = n\) atm

압력 차이: \(1.5n + n = 2.5n = 0.01\) atm

\[n = 0.004 \text{ mol}\]

메탄올 생성량: 반응식에서 \(\text{CO}_2\) 1 mol당 \(\text{CH}_3\text{OH}\) 1 mol이므로:

\[n_{\text{CH}_3\text{OH}} = 0.004 \text{ mol}\]

수위 변화:

산화 수조에서 기체(\(\text{O}_2\))가 생성되어 기체 부피가 증가하므로 수면이 하강(기체가 수면을 누름)한다. 환원 수조에서 기체(\(\text{CO}_2\))가 소모되어 기체 부피가 감소하므로 수면이 상승한다.

또한 반투막을 통한 물 이동을 고려하면: 산화 수조의 압력이 더 높으므로 물이 환원 수조 쪽으로 이동하는 경향이 있다.

종합적으로 환원 전극 쪽 수조의 수위가 높아진다.

2-3. 아연 전극 사용 시

[그림 2-2]에서 산화 경향: Zn > \(\text{H}_2\) > \(\text{CH}_3\text{OH}\) > \(\text{H}_2\text{O}\) > Au

(1) 아연 전극에서 산화되는 물질:

아연의 산화 경향이 \(\text{H}_2\text{O}\)보다 크므로, 물 대신 아연(Zn)이 산화된다:

\[\text{Zn}(s) \rightarrow \text{Zn}^{2+}(aq) + 2e^-\]

(2) 수위 변화:

기존(물 산화): 산화 수조에서 \(\text{O}_2\) 기체가 발생 → 기체 압력 증가

아연 산화: Zn이 \(\text{Zn}^{2+}\) 이온으로 용해 → 기체 발생 없음

산화 수조에서 기체가 발생하지 않으므로, 산화 수조의 기체 압력이 증가하지 않는다.

환원 수조에서는 동일하게 \(\text{CO}_2\)가 소모된다.

따라서 문제 2-2와 비교하면:

- 산화 수조: 기체 발생 없음 → 수위 변화 작음 (기존에는 \(\text{O}_2\) 생성으로 수위 하강)

- 환원 수조: 동일하게 \(\text{CO}_2\) 소모 → 수위 상승

산화 수조의 수위는 문제 2-2보다 높고(기체가 발생하지 않으므로), 환원 수조의 수위 변화는 동일하다.

(3) 아연 vs 금 사용 시 수용액 색 차이:

아연 전극: Zn이 산화되어 \(\text{Zn}^{2+}\) 이온이 용해됨. \(\text{Zn}^{2+}\)는 무색이고 기체 발생이 없으므로 산화 수조의 용액은 무색이다. pH 변화도 없으므로 페놀프탈레인은 색이 변하지 않는다.

금 전극: Au의 산화 경향이 \(\text{H}_2\text{O}\)보다 작으므로, Au는 산화되지 않고 물이 산화된다 (문제 2-1과 동일):

\[2\text{H}_2\text{O}(\ell) \rightarrow \text{O}_2(g) + 4\text{H}^+(aq) + 4e^-\]

\(\text{H}^+\)이 생성되어 산성이 되므로 페놀프탈레인은 변하지 않지만, 환원 전극 쪽에서는 \(\text{OH}^-\)가 생성되어 염기성이 되므로 페놀프탈레인이 분홍색(적색)으로 변한다.

아연 사용 시: 산화 수조에서 \(\text{H}^+\) 생성 없이 \(\text{Zn}^{2+}\)만 생성, 환원 수조에서는 동일하게 \(\text{OH}^-\) 생성 → 환원 수조의 페놀프탈레인이 분홍색으로 변한다. 산화 수조는 무색 유지.

금 사용 시: 물이 산화되어 \(\text{H}^+\) 생성 → 산화 수조는 산성(무색), 환원 수조는 염기성(분홍색). 아연과 동일한 색 패턴이지만, 아연 사용 시에는 산화 수조가 중성에 가깝고, 금 사용 시에는 산화 수조가 산성이라는 차이가 있다.

2-4. 반응 조건에 따른 메탄올 생성 속도

기본 조건: \(\text{CO}_2\) 1 mol을 넣고 추가 공급 없음. [그림 2-3(1)]에서 \(\text{CO}_2\) 농도가 \(c_0\)에서 시작하여 시간 \(t\)에서 \(c_0/2\), 시간 \(2t\)에서 \(c_0/4\)로 감소한다. 반감기가 일정하므로 1차 반응이다.

A 조건: \(\text{CO}_2\) 1 atm 유지 → 농도 일정 → 생성 속도 일정 (직선적 메탄올 농도 증가)

B 조건: A 조건에서 \(\text{CO}_2\) 0.2 atm으로 전환

1차 반응에서 속도 = \(k[\text{CO}_2]\)이므로, \(\text{CO}_2\) 압력이 1 atm에서 0.2 atm으로 줄면 속도가 1/5로 감소한다. 메탄올 농도는 계속 증가하지만 기울기가 1/5로 감소한다.

C 조건: B 조건에 \(\text{N}_2\)를 추가하여 전체 압력 1 atm 유지

\(\text{N}_2\)는 반응에 참여하지 않는 불활성 기체이다. \(\text{CO}_2\)의 분압은 여전히 0.2 atm이다. 반응 속도는 \(\text{CO}_2\)의 분압(또는 농도)에만 의존하므로 B 조건과 동일하다.

따라서 메탄올 농도 개형:

- A 조건: 가파른 직선 증가 (기울기 \(= k \cdot c_0\))

- B 조건으로 전환: 기울기가 1/5로 감소한 직선 증가

- C 조건으로 전환: 기울기 변화 없음 (B와 동일하게 유지)

2-5. 열화학 반응

(1) '가' 반응식과 반응 엔탈피:

'가' 반응: \(\text{CO}_2\)와 \(\text{H}_2\text{O}\)로부터 \(\text{CH}_3\text{OH}\)와 \(\text{O}_2\) 생성

\[\text{CO}_2(g) + 2\text{H}_2\text{O}(g) \rightarrow \text{CH}_3\text{OH}(g) + \frac{3}{2}\text{O}_2(g)\]

'나' 반응: \(\text{CO}_2(g) + 3\text{H}_2(g) \rightarrow \text{CH}_3\text{OH}(g) + \text{H}_2\text{O}(g)\), \(\Delta H = -49\) kJ

\(\text{H}_2\text{O}(g)\)의 생성 반응: \(\text{H}_2(g) + \frac{1}{2}\text{O}_2(g) \rightarrow \text{H}_2\text{O}(g)\), \(\Delta H_f = -242\) kJ/mol

'가' 반응을 '나' 반응과 물의 생성 반응으로 조합한다:

'나' 반응: \(\text{CO}_2 + 3\text{H}_2 \rightarrow \text{CH}_3\text{OH} + \text{H}_2\text{O}\), \(\Delta H = -49\)

물 생성 역반응 \(\times 3\): \(3\text{H}_2\text{O} \rightarrow 3\text{H}_2 + \frac{3}{2}\text{O}_2\), \(\Delta H = 3 \times 242 = +726\)

합산: \(\text{CO}_2 + 3\text{H}_2\text{O} \rightarrow \text{CH}_3\text{OH} + \text{H}_2\text{O} + \frac{3}{2}\text{O}_2 + 3\text{H}_2 - 3\text{H}_2\)

정리: \(\text{CO}_2 + 2\text{H}_2\text{O} \rightarrow \text{CH}_3\text{OH} + \frac{3}{2}\text{O}_2\)

\[\Delta H_{가} = -49 + 726 = +677 \text{ kJ}\]

따라서 '가' 반응의 반응 엔탈피는 +677 kJ (강한 흡열 반응)이다.

(2) '가' 반응과 문제 2-1 전체 반응의 엔탈피 비교:

'가' 반응: \(\text{CO}_2(g) + 2\text{H}_2\text{O}(g) \rightarrow \text{CH}_3\text{OH}(g) + \frac{3}{2}\text{O}_2(g)\)

문제 2-1 전체 반응: \(\text{CO}_2(g) + 2\text{H}_2\text{O}(\ell) \rightarrow \text{CH}_3\text{OH}(aq) + \frac{3}{2}\text{O}_2(g)\)

두 반응의 차이:

- 반응물: \(\text{H}_2\text{O}(g)\) vs \(\text{H}_2\text{O}(\ell)\) → 물의 기화열만큼 차이

- 생성물: \(\text{CH}_3\text{OH}(g)\) vs \(\text{CH}_3\text{OH}(aq)\) → 메탄올의 응축열 및 용해열만큼 차이

따라서 두 반응의 엔탈피가 다른 이유는 반응물과 생성물의 상태(기체/액체/수용액)가 다르기 때문이다. 물의 기화열(\(\text{H}_2\text{O}(\ell) \rightarrow \text{H}_2\text{O}(g)\))과 메탄올의 용해 과정(\(\text{CH}_3\text{OH}(g) \rightarrow \text{CH}_3\text{OH}(aq)\))에 수반되는 엔탈피 변화가 포함되므로 값이 달라진다.

(3) 온도 변화에 따른 평형과 반응 속도:

'가' 반응은 흡열 반응(\(\Delta H = +677\) kJ)이다.

온도가 \(T_1\)에서 \(T_2\)로 올라가면 (\(T_2 > T_1\)):

르 샤틀리에 원리에 의해, 온도 상승은 흡열 방향(정반응)을 촉진한다.

따라서:

- \(P_1'\) (메탄올 압력): \(P_1\)보다 증가 (정반응 촉진으로 메탄올 생성 증가)

- \(P_2'\) (이산화탄소 압력): \(P_2\)보다 감소 (정반응 촉진으로 \(\text{CO}_2\) 소모 증가)

평형에 도달하는 시간: 온도가 높아지면 분자의 운동 에너지가 증가하여 정반응과 역반응 모두 속도가 빨라진다. 따라서 평형에 도달하는 데 걸리는 시간은 짧아진다.
