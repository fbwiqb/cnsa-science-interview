[화학 결합의 변화와 반응열]

(1-1) 프로페인 연소 반응의 결합 변화

반응 (가): \(\text{C}_3\text{H}_8 + 5\text{O}_2 \rightarrow 3\text{CO}_2 + 4\text{H}_2\text{O}\)

\(\text{C}_3\text{H}_8\) (프로페인)의 구조를 분석하면:

- C-H 결합: 8개

- C-C 결합: 2개

\(3\text{CO}_2\)에서 C=O 결합: \(3 \times 2 = 6\)개

\(4\text{H}_2\text{O}\)에서 O-H 결합: \(4 \times 2 = 8\)개

따라서 빈칸을 채우면:

\[8\text{C-H} + 2\text{C-C} + 5\text{O=O} \rightarrow 6[\text{C=O}] + 8[\text{O-H}]\]

빈칸 답: (8)C-H + (2)C-C + 5 O=O \(\rightarrow\) (6)[C=O] + (8)[O-H]

[수소 연료 전지의 전극 반응]

(1-3) 연료 전지의 전극 반응과 표준 전극 전위

제시된 반응 \(2\text{H}_2 \rightarrow 4\text{H}^+ + 4e^-\)는 산화 반응이다. 수소가 전자를 잃으므로 이 반응은 산화 전극(anode, 음극)에서 일어난다.

다른 전극(환원 전극, cathode, 양극)에서 일어나는 반응:

\[\text{O}_2 + 4\text{H}^+ + 4e^- \rightarrow 2\text{H}_2\text{O}\]

표준 전극 전위 계산:

전체 반응: \(2\text{H}_2 + \text{O}_2 \rightarrow 2\text{H}_2\text{O}\), \(\Delta G = -474\,\text{kJ}\)

\[\Delta G = -nFE^\circ\]
\[E^\circ = -\frac{\Delta G}{nF} = -\frac{-474 \times 10^3}{4 \times 96500} = \frac{474000}{386000} = 1.23\,\text{V}\]

전체 전지 전위: \(E^\circ_{\text{cell}} = E^\circ_{\text{cathode}} - E^\circ_{\text{anode}}\)

표준 수소 전극(SHE)에서: \(2\text{H}^+ + 2e^- \rightarrow \text{H}_2\), \(E^\circ = 0\,\text{V}\)

따라서:

- 산화 전극(anode): \(E^\circ_{\text{anode}} = 0\,\text{V}\) (수소의 산화)

- 환원 전극(cathode): \(E^\circ_{\text{cathode}} = E^\circ_{\text{cell}} + E^\circ_{\text{anode}} = 1.23 + 0 = \mathbf{1.23\,\text{V}}\)

[광합성과 산화-환원]

(1-4) 광합성에서 탄소의 변화

광합성: \(6\text{CO}_2 + 12\text{H}_2\text{O} \rightarrow [\text{C}(\text{H}_2\text{O})]_6 + 6\text{H}_2\text{O} + 6\text{O}_2\)

탄소의 산화수 변화:

\(\text{CO}_2\)에서 C의 산화수: O의 전기음성도(3.5)가 C(2.5)보다 크므로, C-O 결합의 전자는 O에 귀속된다. C의 산화수 = +4

포도당 \(\text{C}_6\text{H}_{12}\text{O}_6\)에서 C의 평균 산화수: \(6x + 12(+1) + 6(-2) = 0\)이므로 \(x = \mathbf{0}\)

따라서 탄소는 산화수 +4에서 0으로 환원된다. 이는 전기음성도가 큰 산소와의 결합이 줄어들고, 전기음성도가 작은 수소와의 결합이 늘어나는 것에 해당한다.

두 단계의 반응:

광합성은 명반응(빛 반응)과 암반응(캘빈 회로)의 두 단계로 나뉜다.

반응 1 (명반응 - 물의 광분해):

\[12\text{H}_2\text{O} \rightarrow 6\text{O}_2 + 24\text{H}^+ + 24e^-\]

(또는 간략히: \(2\text{H}_2\text{O} \rightarrow \text{O}_2 + 4\text{H}^+ + 4e^-\))

이 반응에서 산소 기체가 생성된다. 물의 산소가 산화되어(\(-2 \rightarrow 0\)) \(\text{O}_2\)가 된다.

반응 2 (암반응 - \(\text{CO}_2\) 고정):

\[6\text{CO}_2 + 24\text{H}^+ + 24e^- \rightarrow \text{C}_6\text{H}_{12}\text{O}_6 + 6\text{H}_2\text{O}\]

이 반응에서 \(\text{CO}_2\)의 탄소가 환원되어 포도당이 생성된다.

[결합 에너지와 물의 광분해]

(1-5) 결합 에너지 관점에서 물의 광분해 가능성

물의 광분해 반응: \(2\text{H}_2\text{O} \rightarrow 2\text{H}_2 + \text{O}_2\)

결합 에너지로 반응 엔탈피를 계산한다.

끊어지는 결합: O-H 4개 (물 2분자)

형성되는 결합: H-H 2개 + O=O 1개

표 1에서 O-H 결합에너지는 직접 주어지지 않았지만, 반응 (나)에서 \(\Delta G = -474\,\text{kJ}\)이므로 역반응의 \(\Delta G = +474\,\text{kJ}\)이다.

결합 에너지 관점으로 추정하면:

\[\Delta H = 4E(\text{O-H}) - [2E(\text{H-H}) + E(\text{O=O})]\]
\[= 4E(\text{O-H}) - [2(440) + 500] = 4E(\text{O-H}) - 1380\]

O-H 결합에너지 \(\approx 460\,\text{kJ/mol}\)으로 추정하면:

\[\Delta H \approx 4(460) - 1380 = 1840 - 1380 = +460\,\text{kJ/mol}\]

물 1몰의 분해에 필요한 에너지: 약 \(+230\,\text{kJ/mol}\)

파장 680 nm인 빛 1몰의 에너지 = 180 kJ/mol이므로, 광자 하나의 에너지만으로는 물 분해에 필요한 에너지를 충족시키기 어렵다.

따라서 실제 광합성에서는 여러 개의 광자(최소 2개 이상)가 필요하며, 이는 광계 II와 광계 I을 거치는 Z-scheme에 의해 달성된다. 680 nm 광자 2개의 에너지(\(2 \times 180 = 360\,\text{kJ/mol}\))는 물 분해에 필요한 에너지를 초과하므로, 빛 에너지를 이용한 물의 광분해는 가능하다.
