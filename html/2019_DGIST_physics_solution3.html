해설 3

문항 해설

1. 충격파의 원리를 묻는 문제. 날아가는 전투기에서 발생하는 음파가 구면파를 이루고 그 음원이 음파속도보다 빨리 진행할 때 구면파들을 잇는 파면이 일정한 각도를 이루는 직선이 됨을 이해

2. 충격파가 일정한 각도를 지니고 진행하고, 이 경우 등속직선운동을 하는 또 다른 물체 (여객기)와의 상대운동을 이해하는 지 묻는 문제

3. 음원이 움직일 경우의 도플러 효과를 묻는 문제

4. 파장과 주파수 사이의 관계를 묻는 문제.

예시 답안

(1) \(\sin\theta = \dfrac{v_{\text{음파}}}{v_{\text{전투기}}} = \dfrac{340\,\text{m/s}}{680\,\text{m/s}} = 0.5\) 그러므로 각도는 \(30°\)

(2) 두 물체가 수평방향 상대속도 510 m/s로 움직이고 있고 \(t=0\)에서 510m 만큼 수직으로 떨어져 있으므로 \(\tan 30° = \dfrac{510\,\text{m}}{x}\) 에서 구한 \(x = 510\sqrt{3}\,\text{m}\) 만큼을 상대속도 510m/s로 움직이는데 드는 시간을 계산하면

\[t = \frac{510\sqrt{3}}{510} = \sqrt{3} \approx 1.7\,\text{s}\]

(3) \(f_0 = 1000\,Hz\)이고 음원이 170 m/s로 움직이고 있으므로

\[f = \frac{f_0}{1 + \dfrac{170\,\text{m/s}}{340\,\text{m/s}}} = \frac{2}{3}f_0 = 667\,Hz\]

(4) 음원이 정지해 있을 때의 파장은 \(\lambda_0 = \dfrac{340\,\text{m/s}}{1000\,Hz} = 0.34\,m\). 음원이 움직일 때 진행방향의 뒷편에서의 파장은 \(\lambda = \lambda_0 +\) [한주기동안 음원이 움직인 거리] \(= 0.34 + 0.17 = 0.51\,\text{m} = 51\,\text{cm}\). 또는 위의 결과를 이용하여

\[\lambda = \frac{v_{\text{음파}}}{f} = \frac{v_{\text{음파}}}{\dfrac{2}{3}f_0} = \frac{3}{2}\lambda_0 = 51\,cm\]

로 구해도 됨.
