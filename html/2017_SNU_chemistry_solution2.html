[에텐의 고리화 반응 - 총괄성질과 분자량]

(2-1) 고리형 탄화수소의 n 결정 방법

\(n\)개의 에텐(\(\text{C}_2\text{H}_4\)) 분자가 반응하여 고리형 탄화수소(사이클로알케인)를 형성한다. 생성물의 분자식은 \(\text{C}_{2n}\text{H}_{4n}\)이다.

생성물이 비휘발성이므로, 용액의 총괄적 성질(colligative properties)을 이용하여 분자량을 구하고 이로부터 \(n\)을 결정할 수 있다.

방법 1 - 끓는점 오름(비점 상승):

비휘발성 용질인 생성물을 알려진 질량의 용매에 녹인 후, 끓는점 오름(\(\Delta T_b = K_b \cdot m\))을 측정하면 몰랄 농도 \(m\)을 구할 수 있다. 용질의 질량을 알고 있으므로 분자량 \(M\)을 계산할 수 있고, \(M = 28n\)이므로 \(n\)을 결정한다.

방법 2 - 어는점 내림(빙점 강하):

마찬가지로 어는점 내림(\(\Delta T_f = K_f \cdot m\))을 측정하여 분자량을 구한다.

방법 3 - 삼투압:

반투과막을 이용하여 삼투압(\(\Pi = MRT\), \(M\)은 몰 농도)을 측정하면 분자량을 결정할 수 있다.

이 세 방법 모두 용질의 분자량을 결정할 수 있으므로, \(M = 28n\)에서 \(n\) 값을 알아낼 수 있다.

[고리화 반응의 엔탈피 변화]

(2-2) 반응의 엔탈피 변화량

반응: \(n\text{C}_2\text{H}_4 \rightarrow \text{C}_{2n}\text{H}_{4n}\) (고리형 사이클로알케인)

\(n\)개의 에텐이 고리를 형성할 때 결합의 변화:

- 각 에텐의 C=C 이중결합이 C-C 단일결합으로 전환된다 (\(n\)개)

- 인접 에텐 단위 사이에 새로운 C-C 단일결합이 형성된다 (\(n\)개)

- C-H 결합은 생성물에서도 유지되므로 변화 없음

따라서 결합 변화를 정리하면:

- 끊어지는 결합: C=C \(n\)개

- 형성되는 결합: C-C \(2n\)개 (고리 내의 모든 C-C 단일결합)

\[\Delta H = \sum(\text{끊어지는 결합 에너지}) - \sum(\text{형성되는 결합 에너지})\]
\[\Delta H = n \times E(\text{C=C}) - 2n \times E(\text{C-C})\]
\[\Delta H = n(728) - 2n(348) = 728n - 696n\]
\[\boxed{\Delta H = +32n \text{ kJ/mol}}\]

이는 약한 흡열 반응이다. 이중결합 하나(728 kJ)를 끊어 단일결합 두 개(348 \(\times\) 2 = 696 kJ)를 형성하므로 결합에너지 면에서 약간 불리하다.

[엔트로피와 자유 에너지]

(2-3) 엔트로피 변화량과 고리화 반응의 n 의존성

\(T = 47\,{}^\circ\text{C} = 320\,\text{K}\)에서 평형(\(\Delta G^\circ = 0\), 즉 \(K = 1\))을 이룬다고 가정하면:

\[\Delta G^\circ = \Delta H^\circ - T\Delta S^\circ = 0\]
\[\Delta S^\circ = \frac{\Delta H^\circ}{T} = \frac{32n \text{ kJ/mol}}{320 \text{ K}} = 0.1n \text{ kJ/(mol} \cdot \text{K)} = +100n \text{ J/(mol} \cdot \text{K)}\]

그런데 이 반응에서 \(n\)개의 기체 분자가 1개의 기체 분자로 변하므로 기체 몰수가 \((n-1)\)만큼 감소하여 실제로는 \(\Delta S < 0\)이어야 한다.

이 모순은 문제 2-2에서 결합에너지만으로 계산한 \(\Delta H\)가 근사값이기 때문에 발생한다. 실제 \(\Delta H\)는 고리 변형에너지(ring strain) 등을 포함하면 음수일 수 있다. 문제의 의도에 따라 \(\Delta G^\circ = 0\) 조건에서 \(\Delta S\)를 형식적으로 구하면 위와 같다.

보다 물리적으로 올바른 분석:

\(n\)개의 기체 분자 \(\rightarrow\) 1개의 기체 분자이므로:

\[\Delta S < 0\]

\(n\)이 커질수록:

- \(\Delta H = +32n\) kJ/mol (더 큰 흡열)

- \(|\Delta S|\)도 증가 (기체 분자 수 감소가 \((n-1)\)으로 더 커짐)

따라서 \(\Delta G = \Delta H - T\Delta S\)에서, \(\Delta H > 0\)이고 \(-T\Delta S > 0\)이므로:

\[\Delta G = 32n + T|\Delta S|\]

\(n\)이 커질수록 \(\Delta G\)가 더 양수가 되므로, \(n\)이 클수록 반응이 비자발적이 된다. 따라서 \(n\)이 작을수록 고리화 반응이 유리하다.

[상 예측과 반응 조건]

(2-4) 생성물의 상과 n의 관계, 반응 조건

\(27\,{}^\circ\text{C} = 300\,\text{K}\), \(1\,\text{atm}\)에서 에텐은 기체이다.

생성물(\(\text{C}_{2n}\text{H}_{4n}\))의 상은 분자량과 분자 간 힘에 의존한다.

- \(n\)이 작으면 (\(n = 2, 3\)): 분자량이 작아 분자 간 인력(런던 분산력)이 약하여 기체

- \(n\)이 중간이면 (\(n\) 중간): 액체

- \(n\)이 크면: 분자량이 커 분자 간 인력이 강해 고체

기화에너지가 \(34.8 \times n\) kJ/mol이므로, \(n\)이 커질수록 기화에너지가 커져 더 높은 온도에서 기화된다. 기화에너지가 분자를 기체 상태로 만드는 데 필요한 에너지이므로, 특정 \(n\) 이상에서는 상온에서 액체 또는 고체로 존재한다.

반응물(에텐, 기체)과 생성물이 하나의 상으로 존재하려면, 둘 다 기체 상태여야 한다. 이를 위해서는:

- 높은 온도: 생성물의 기화를 촉진

- 낮은 압력: 기체 상태 유지에 유리 (끓는점 감소)

따라서 그래프에서 영역 3 (저압, 고온) 조건이 적합하다.

[기체 분출 속도와 생성물 구조]

(2-5) 분출 속도비로부터 생성물 구조 예측

그레이엄의 분출 법칙에 의하면:

\[\frac{v_1}{v_2} = \sqrt{\frac{M_2}{M_1}}\]

분출 속도비가 \(1 : \frac{1}{\sqrt{2}} : \frac{1}{\sqrt{3}}\)이므로:

가장 빠른 기체의 분자량을 \(M_1\)이라 하면:

\[\frac{v_1}{v_2} = \sqrt{\frac{M_2}{M_1}} = \frac{1}{1/\sqrt{2}} = \sqrt{2} \quad \Rightarrow \quad M_2 = 2M_1\]
\[\frac{v_1}{v_3} = \sqrt{\frac{M_3}{M_1}} = \frac{1}{1/\sqrt{3}} = \sqrt{3} \quad \Rightarrow \quad M_3 = 3M_1\]

따라서 세 생성물의 분자량 비: \(M_1 : M_2 : M_3 = 1 : 2 : 3\)

에텐의 분자량은 28이므로, 생성물의 분자량 = \(28n\)이다.

\(n_1 : n_2 : n_3 = 1 : 2 : 3\)이면 가장 가벼운 생성물은 \(n = 2\)이므로:

- \(n = 2\): 사이클로뷰테인 (\(\text{C}_4\text{H}_8\), \(M = 56\))

- \(n = 4\): 사이클로옥테인 (\(\text{C}_8\text{H}_{16}\), \(M = 112\))

- \(n = 6\): 사이클로도데케인 (\(\text{C}_{12}\text{H}_{24}\), \(M = 168\))

그러나 \(n_1 : n_2 : n_3\)는 반드시 \(2:4:6\)일 필요는 없다. 일반적으로 \(M = 28n\)이므로:

- \(M_1 = 28n_1\), \(M_2 = 28n_2 = 2 \times 28n_1\), \(M_3 = 28n_3 = 3 \times 28n_1\)

- \(n_2 = 2n_1\), \(n_3 = 3n_1\)

\(n_1 = 2\) (최소 정수, \(n \geq 2\))이면:

- 사이클로뷰테인 (\(\text{C}_4\text{H}_8\), \(n = 2\))

- 사이클로옥테인 (\(\text{C}_8\text{H}_{16}\), \(n = 4\))

- 사이클로도데케인 (\(\text{C}_{12}\text{H}_{24}\), \(n = 6\))
