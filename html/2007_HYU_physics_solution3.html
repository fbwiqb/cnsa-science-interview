공기저항을 받는 낙하 운동 — 종단속도와 차원 분석

주어진 조건:

- 질량 \(m\)인 물체가 정지 상태에서 낙하

- 공기저항력: \(-bv^2\) (속력의 제곱에 비례)

- 중력가속도: \(g\), \(b\)는 상수

(1) 종단속도

종단속도 \(v_t\)에서는 가속도가 0이므로, 중력과 공기저항력이 평형을 이룬다:

\[mg = bv_t^2\]
\[v_t = \sqrt{\frac{mg}{b}}\]
\[\boxed{v_t = \sqrt{\frac{mg}{b}}}\]

(2) 상수 \(b\)의 차원

\(mg = bv^2\)에서 \(b\)의 차원을 구하면:

\[[b] = \frac{[mg]}{[v^2]} = \frac{MLT^{-2}}{(LT^{-1})^2} = \frac{MLT^{-2}}{L^2T^{-2}} = \frac{M}{L}\]
\[\boxed{[b] = ML^{-1}}\]

(3) 종단속도에 도달할 때까지 낙하한 거리 \(S\)의 \(g, m, b\) 의존성

차원 분석을 이용하여 \(S\)가 \(g, m, b\)에 어떻게 의존하는지 구한다.

\(S = g^a m^b b^c\)로 놓으면, 각 물리량의 차원은:

- \([S] = L\)

- \([g] = LT^{-2}\)

- \([m] = M\)

- \([b] = ML^{-1}\)

차원 방정식:

\[L = (LT^{-2})^a \cdot M^b \cdot (ML^{-1})^c\]
\[L^1 M^0 T^0 = L^{a-c} \cdot M^{b+c} \cdot T^{-2a}\]

각 차원에 대해:

- \(T\): \(-2a = 0 \implies a = 0\)

- \(L\): \(a - c = 1 \implies c = -1\)

- \(M\): \(b + c = 0 \implies b = 1\)

따라서:

\[S \propto g^0 \cdot m^1 \cdot b^{-1} = \frac{m}{b}\]
\[\boxed{S \propto \frac{m}{b}}\]

\(S\)는 \(g\)에 무관하고, \(m\)에 비례하며, \(b\)에 반비례한다. 이는 물리적으로도 타당하다. 질량이 클수록 종단속도에 도달하기까지 더 많이 가속되어야 하므로 낙하 거리가 길어지고, 공기저항 상수 \(b\)가 클수록 빨리 종단속도에 도달하므로 낙하 거리가 짧아진다.
