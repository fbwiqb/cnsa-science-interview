해설 1

(문제 1-1)

화학 반응식 구하기

\[4\text{NH}_3\text{(g)} + 5\text{O}_2\text{(g)} \rightarrow 4\text{NO(g)} + 6\text{H}_2\text{O(g)}\]

반응 후의 각 성분의 몰분율 계산

\(n = PV/RT\)를 이용하여 각 성분의 몰수를 구함

- \(\text{NH}_3\)의 경우 \(n = (1.0 \text{ 기압})(3.0 \text{ L})/RT = 3.0a\) (편의상 \(1/RT\)를 \(a\)로 놓음)

- \(\text{O}_2\)의 경우 \(n = (2.0 \text{ 기압})(2.0 \text{ L})/RT = 4.0a\)

\(4\text{NH}_3\)\(5\text{O}_2\)\(4\text{NO}\)\(6\text{H}_2\text{O}\)
반응 전\(3.0a\)\(4.0a\)\(0\)\(0\)
변화\(-3.0a\)\(-3.75a\)\(+3.0a\)\(+4.5a\)
반응 후\(0\)\(0.25a\)\(3.0a\)\(4.5a\)

반응 후의 기체 혼합물의 총 몰수 = \(0.25a + 3.0a + 4.5a = 7.75a\)

NO의 몰분율 = \(3.0a/7.75a = 3.0/7.75\)

칸막이 제거 직후의 각 성분의 부분 압력 계산(반응 전)

- 용기 부피 = \(3.0 + 2.0 = 5.0\) L

- \(\text{NH}_3\)의 부분 압력 = \((1.0 \text{ 기압})(3.0 \text{ L})/5.0 \text{ L} = 0.60\) 기압

- \(\text{O}_2\)의 부분 압력 = \((2.0 \text{ 기압})(2.0 \text{ L})/5.0 \text{ L} = 0.80\) 기압

반응물 몰비 확인: 이론적 반응물 몰비 \(n_{O_2}/n_{NH_3} = 5/4 = 1.25\), 실제 몰비 \(= 0.80/0.60 = 1.33\)

\(\therefore\) \(\text{NH}_3\)는 완전히 소모되어 NO로 전환됨(\(\text{O}_2\)는 일부가 미반응으로 남음)

\(\text{NH}_3\)가 완전히 NO로 전환되므로, \(\text{NH}_3\)의 몰수가 NO의 몰수가 됨

즉, \(\text{NH}_3\)의 부분 압력(0.60 기압)이 그대로 NO의 부분 압력(0.60 기압)이 됨

(문제 1-2)

열화학 반응식 구하기

주어진 열화학 반응식을 이용하여 프로페인의 열화학 반응식 구하기

\[3\text{C(s)} + 3\text{O}_2\text{(g)} \rightarrow 3\text{CO}_2\text{(g)} \qquad \Delta H_1° = 3(-394) \text{ kJ}\]
\[4\text{H}_2\text{(g)} + 2\text{O}_2\text{(g)} \rightarrow 4\text{H}_2\text{O(g)} \qquad \Delta H_2° = 2(-484) \text{ kJ}\]
\[\text{C}_3\text{H}_8\text{(g)} \rightarrow 3\text{C(s)} + 4\text{H}_2\text{(g)} \qquad \Delta H_3° = -104 \text{ kJ}\]
\[\text{C}_3\text{H}_8\text{(g)} + 5\text{O}_2\text{(g)} \rightarrow 3\text{CO}_2\text{(g)} + 4\text{H}_2\text{O(g)} \qquad \Delta H° = -2254 \text{ kJ}\]

\(\Delta G°\) 값 판단하기

반응식을 보면 기체 반응물 6몰 \(\rightarrow\) 기체 생성물 7몰로 몰수가 증가함

따라서 엔트로피가 증가함(\(\Delta S° > 0\))

\(\Delta G° = \Delta H° - T\Delta S°\) 에서

\(\Delta H° < 0\), \(\Delta S° > 0\) 이므로 \(\Delta G° < 0\)

\(\therefore\) 반응은 자발적임
