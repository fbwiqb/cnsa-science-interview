문제 1-1. 염색체와 유전자 위치 분석

(1)

제시문 (나)에서 각 시험관에 2개의 세포가 들어있고, 유전자 A, B, C, D의 DNA 상대량이 측정되었다. 세포들은 분열이 완료된 직후의 세포이며, 염색체 수 이상이나 결실은 없다.

분열 완료 직후의 세포 유형:

- 체세포 분열 완료 직후: 2n 세포, 각 상염색체 유전자는 2개씩

- 감수 1분열 완료 직후: n 세포(상동 염색체 분리), 각 유전자의 DNA 상대량은 체세포의 절반이나 자매 염색분체가 아직 결합되어 있으므로 상대량 2 (단, 1개당 기준값 1로 계산하면 유전자 2개)

- 감수 2분열 완료 직후: n 세포(자매 염색분체 분리), 각 유전자 1개씩

그래프에서 읽히는 DNA 상대량(이미지 기준):

시험관ABCD
1번1232
2번2221
3번2121
4번2211
5번2122

각 시험관에 2개의 세포가 있으므로, 측정된 DNA 상대량은 두 세포의 합이다.

분열 완료 직후 세포의 유전자 개수:

- 체세포(2n): 상염색체 유전자 = 2개, X 연관 유전자 = 남성은 1개, 여성은 2개

- 감수 1분열 후 세포: 각 유전자 = 1 또는 2 (자매 염색분체 결합 상태)

- 감수 2분열 후 세포: 각 유전자 = 1개 (상염색체), 0 또는 1개 (성염색체)

유전자 D의 상대량이 시험관에 따라 0~2로 변동하는 것은, D가 성염색체(X 염색체) 위에 위치할 가능성을 시사한다. 남성(XY)의 경우 X 연관 유전자는 1개뿐이므로, 체세포에서 D = 1이 된다.

유전자 C의 상대량이 1~3으로 변동하는 것은 C도 특이적인 위치에 있을 수 있다.

염색체 배치 추론:

- 유전자 A: 상염색체 위치. 체세포에서 항상 2개.

- 유전자 B: 상염색체 위치. 체세포에서 항상 2개.

- 유전자 C: A와 같은 상염색체 또는 다른 상염색체 위치.

- 유전자 D: X 염색체 위치 가능성. 남성 체세포에서 1개.

A, B, C, D가 서로 다른 위치에 있고 각 시험관의 데이터를 종합하면, 일부 유전자는 같은 염색체의 다른 팔(arm)에, 일부는 성염색체에 위치한다고 추론할 수 있다.

문제 1-2. 시험관 1~5번의 세포별 유전자 개수

(2)

각 시험관의 DNA 상대량을 두 세포로 분배하면:

시험관 1번 (A=1, B=2, C=3, D=2):

- 세포 ①: 체세포(2n) → A=1(?), 또는 감수 2분열 후 세포

- 합이 맞도록 분배: 세포 ① (A=0, B=1, C=2, D=1) + 세포 ② (A=1, B=1, C=1, D=1) 등 다양한 조합

정확한 분배를 위해서는 각 세포 유형의 유전자 개수 제약 조건을 만족하는 조합을 찾아야 한다. 가능한 세포 유형별 유전자 개수:

- 체세포(2n): 상염색체 유전자 = 2, X연관 유전자 = 1(남성) 또는 2(여성)

- 감수 1분열 후: 상염색체 유전자 = 2(자매 염색분체), X연관 유전자 = 0 또는 2

- 감수 2분열 후: 상염색체 유전자 = 1, X연관 유전자 = 0 또는 1

이 사람이 남성이고 D가 X 염색체에 위치한다고 가정하면:

시험관세포 1 유형세포 1 (A,B,C,D)세포 2 유형세포 2 (A,B,C,D)
1번감수 2분열 후(0,1,1,1)(1,1,2,1)감수 1분열 후
2번체세포(2n)(1,1,1,1)(1,1,1,0)감수 2분열 후
3번감수 2분열 후(1,0,1,0)(1,1,1,1)감수 2분열 후
4번감수 2분열 후(1,1,0,1)(1,1,1,0)감수 2분열 후
5번체세포(2n)(1,0,1,1)(1,1,1,1)감수 1분열 후

(위 표는 가능한 분배의 한 예시이며, 실제 그래프의 정확한 수치에 따라 달라질 수 있다.)

핵심 원리는 각 시험관의 총 DNA 상대량을 생물학적으로 가능한 세포 유형 2개로 분배하여, 모든 유전자 개수가 해당 세포 유형에서 가능한 값인지 확인하는 것이다.
