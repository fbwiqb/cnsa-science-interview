문제 1-1. 코돈과 아미노산 연결

(1)

제시문 (나)에서 반복 서열로 합성한 mRNA의 번역 결과를 분석한다. 인공 합성 mRNA는 첫 뉴클레오타이드부터 3개씩 번역된다.

CCCG 반복 서열:

...CCCGCCCGCCCGCCCG...

코돈 분리: CCC / GCC / CGC / CCG / CCC / GCC / ...

- 가능한 코돈: CCC, GCC, CGC, CCG

- 제시문 (가)의 코돈 표에서: CCC는 CCU/CCC/CCA/CCG 그룹 → 아미노산 X (프롤린)

- CCG도 같은 그룹 → 같은 아미노산 X (프롤린)

- GCC는 GCU/GCC/GCA/GCG 그룹 → 아미노산 Y (알라닌)

- CGC는 AGA/AGG/CGU/CGC/CGA/CGG 그룹 → 아미노산 Z (아르기닌)

따라서 CCCG 반복: 아미노산 비율 = 프롤린 : 알라닌 : 아르기닌 = 2 : 1 : 1 (50% : 25% : 25%)

AAAG 반복 서열:

...AAAGAAAGAAAGAAAG...

코돈 분리: AAA / GAA / AGA / AAG / AAA / ...

- AAA: AAA/AAG 그룹 → 아미노산 (라이신)

- AAG: 같은 그룹 → 라이신

- GAA: GAA/GAG 그룹 → 아미노산 (글루탐산)

- AGA: AGA/AGG/CGU/CGC/CGA/CGG 그룹 → 아미노산 (아르기닌)

따라서 AAAG 반복: 라이신 : 글루탐산 : 아르기닌 = 2 : 1 : 1 (50% : 25% : 25%)

AAGG 반복 서열:

...AAGGAAGGAAGGAAGG...

코돈 분리: AAG / GAA / GGA / AGG / AAG / ...

- AAG: 라이신

- GAA: 글루탐산

- GGA: GGU/GGC/GGA/GGG 그룹 → 아미노산 (글리신)

- AGG: AGA/AGG/CGU/CGC/CGA/CGG 그룹 → 아르기닌

따라서 AAGG 반복: 라이신 : 글루탐산 : 글리신 : 아르기닌 = 1 : 1 : 1 : 1 (25% : 25% : 25% : 25%)

유전 암호 표 작성:

U-C-A-G 순서로 64개 코돈 표를 구성하고, 확인된 코돈을 표시한다:

확인된 코돈:

- CCC → 프롤린, CCG → 프롤린

- GCC → 알라닌

- CGC → 아르기닌

- AAA → 라이신, AAG → 라이신

- GAA → 글루탐산

- AGA → 아르기닌, AGG → 아르기닌

- GGA → 글리신

각 아미노산의 전체 코돈 수 대비 확인된 코돈 비율:

- 프롤린 (4개 중 2개 확인): 50%

- 알라닌 (4개 중 1개 확인): 25%

- 아르기닌 (6개 중 3개 확인): 50%

- 라이신 (2개 중 2개 확인): 100%

- 글루탐산 (2개 중 1개 확인): 50%

- 글리신 (4개 중 1개 확인): 25%

전체: 6개 아미노산에 대해 총 22개의 코돈 중 10개 확인 = 약 45.5%

문제 1-2. 장기 질환 해결을 위한 생명공학 기술

(2)

특정 장기가 기능하지 못하는 상태를 해결하기 위해 이론적으로 가능한 생명공학 기술은 다음과 같다:

1. 장기 이식(Organ Transplantation):

다른 사람(기증자)으로부터 건강한 장기를 이식받는 방법이다. 면역 거부 반응을 억제하기 위해 면역 억제제를 사용해야 하며, 기증자와의 조직 적합성이 중요하다.

2. 줄기세포 치료(Stem Cell Therapy):

환자 자신의 체세포에서 유도만능줄기세포(iPSC)를 만들거나 배아줄기세포를 이용하여, 필요한 장기의 세포로 분화시켜 손상된 장기를 재생하는 방법이다. 면역 거부 반응이 적다는 장점이 있다.

3. 이종 장기 이식(Xenotransplantation):

유전자 변형 기술로 면역 거부 반응을 일으키는 유전자를 제거한 돼지 등의 동물에서 장기를 얻어 이식하는 방법이다.

4. 3D 바이오프린팅(3D Bioprinting):

환자의 세포와 생체 재료를 이용하여 3D 프린팅 기술로 인공 장기를 제작하는 방법이다.

5. 유전자 치료(Gene Therapy):

장기 기능 상실의 원인이 특정 유전자의 이상인 경우, 바이러스 벡터 등을 이용하여 정상 유전자를 해당 장기의 세포에 전달하여 기능을 회복시키는 방법이다.

6. 체세포 핵이식 클로닝:

환자의 체세포 핵을 탈핵한 난자에 이식하여 배반포를 만들고, 이로부터 줄기세포를 얻어 필요한 장기 세포로 분화시키는 방법이다(치료적 클로닝).
