신장의 배설과 페니실린 약동학

(문제 1-1) 페니실린의 배설속도와 여과속도의 차이 및 그래프 분석

그래프에서 배설속도와 여과속도의 차이는 분비속도를 의미한다. 신장에서의 배설은 여과와 분비 두 과정에 의해 이루어지며, 재흡수가 없다고 가정하면 배설속도 = 여과속도 + 분비속도이다. 따라서 배설속도에서 여과속도를 빼면 세뇨관에서의 분비에 의한 배설 기여분을 알 수 있다.

여과속도는 혈중농도에 비례하여 일정하게 증가하는 직선으로 나타난다. 이는 사구체에서의 여과가 단순한 혈압차에 의한 물리적 과정이므로, 혈중농도가 높아지면 여과되는 양도 비례하여 증가하기 때문이다.

배설속도가 A 지점에서 기울기가 꺾이는 이유는, 세뇨관에서의 분비 과정이 ATP를 소비하는 능동수송체 단백질에 의해 이루어지기 때문이다. 능동수송체의 수는 한정되어 있으므로, 페니실린의 혈중농도가 낮을 때는 분비속도가 혈중농도에 비례하여 증가하지만, A 지점에서 수송체가 포화 상태에 도달하면 더 이상 분비속도가 증가하지 않는다. 따라서 A 지점 이후에는 배설속도가 여과속도와 평행하게(기울기가 같게) 증가한다.

프로베네시드를 페니실린과 함께 투여하면, 프로베네시드가 세뇨관의 분비과정에서 페니실린과 경쟁한다. 이로 인해 페니실린의 분비가 감소하게 되므로, 배설속도 그래프에서 분비에 의한 기여분이 줄어든다. 결과적으로 배설속도 그래프는 여과속도 그래프에 더 가까워지며, A 지점이 더 낮은 혈중농도에서 나타나거나 배설속도 곡선이 전체적으로 아래로 이동한다. 이는 페니실린이 체내에 더 오래 머물러 혈중농도가 오랫동안 유지되는 효과를 가져온다.

배설속도와 여과속도의 차이는 분비속도이며, A 지점의 꺾임은 능동수송체의 포화 때문이고, 프로베네시드 투여 시 경쟁적 억제로 분비가 감소하여 배설속도 그래프가 여과속도에 가까워진다.

(문제 1-2) 세뇨관에서의 원뇨 이동 시간 계산

주어진 조건을 정리하면:

- 세뇨관은 직경이 일정하다.

- 원뇨농도 \(\rho(x)\)는 \(x\)에 대한 1차 함수이다.

- 입구(\(x = 0\))에서 \(\rho(0) = 1\), 출구(\(x = L\))에서 \(\rho(L) = 2\)

- 모든 위치에서 원뇨농도 \(\times\) 원뇨속도 = 1

\(\rho(x) = 1 + \frac{x}{L}\) (1차 함수로 입구에서 1, 출구에서 2)

원뇨속도를 \(v(x)\)라 하면, \(\rho(x) \cdot v(x) = 1\)이므로:

\[v(x) = \frac{1}{\rho(x)} = \frac{1}{1 + \frac{x}{L}} = \frac{L}{L + x}\]

\(v(x) = \frac{dx}{dt}\)이므로:

\[dt = \frac{dx}{v(x)} = \frac{L + x}{L} dx\]

입구에서 출구까지 걸리는 시간 \(T\):

\[T = \int_0^L \frac{L + x}{L} dx = \frac{1}{L} \int_0^L (L + x) \, dx = \frac{1}{L} \left[ Lx + \frac{x^2}{2} \right]_0^L\]
\[T = \frac{1}{L} \left( L^2 + \frac{L^2}{2} \right) = \frac{1}{L} \cdot \frac{3L^2}{2} = \frac{3L}{2}\]

따라서:

\[L = \frac{2T}{3}\]

세뇨관의 길이 \(L = \dfrac{2T}{3}\)이다.
