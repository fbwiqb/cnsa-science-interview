1-1. 비타민 C 합성 결함 효소 확인 실험

(1)

실험 설계:

사람 체세포를 여러 그룹으로 나누어 각각 다른 전구 물질을 배지에 첨가한 후 비타민 C 생성 여부를 확인한다.

실험 그룹배지에 첨가하는 물질관찰 항목
대조군포도당만비타민 C 생성 여부
실험군 1전구 물질 A비타민 C 생성 여부
실험군 2전구 물질 B비타민 C 생성 여부
실험군 3전구 물질 C비타민 C 생성 여부

각 효소 결함 시 예상 결과:

- 효소 1 결함 시: 포도당 → 전구 물질 A 전환 불가. 대조군(포도당): 비타민 C 생성 X. 실험군 1(전구 물질 A): 비타민 C 생성 O. 실험군 2(전구 물질 B): 비타민 C 생성 O. 실험군 3(전구 물질 C): 비타민 C 생성 O.

- 효소 2 결함 시: 전구 물질 A → 전구 물질 B 전환 불가. 대조군: X. 실험군 1: X. 실험군 2: O. 실험군 3: O.

- 효소 3 결함 시: 전구 물질 B → 전구 물질 C 전환 불가. 대조군: X. 실험군 1: X. 실험군 2: X. 실험군 3: O.

- 효소 4 결함 시: 전구 물질 C → 비타민 C 전환 불가. 대조군: X. 실험군 1: X. 실험군 2: X. 실험군 3: X.

결함 효소를 기준으로 해당 효소 이후의 전구 물질을 직접 공급하면 비타민 C가 생성되고, 이전 전구 물질을 공급하면 생성되지 않는다. 효소 4 결함 시에는 어떤 전구 물질을 공급해도 최종 단계가 작동하지 않으므로 비타민 C가 생성되지 않는다.

1-2. 효소 1~3 소실 vs 효소 4 소실의 파급효과

(2)

일반적으로 물질대사의 전구 물질들은 여러 다른 대사 과정에도 사용될 수 있다.

효소 1 소실: 포도당에서 전구 물질 A로의 전환이 차단된다. 포도당은 해당과정, TCA 회로, 오탄당 인산 경로 등 수많은 대사 과정의 기질이므로 포도당 자체는 축적되지 않는다. 그러나 전구 물질 A, B, C가 모두 생성되지 않으므로, 이들이 관여하는 다른 대사 경로도 영향을 받아 세포에 광범위한 대사 장애가 발생한다.

효소 2 소실: 전구 물질 A에서 B로의 전환이 차단된다. 전구 물질 A가 축적되고 B, C는 생성되지 않는다. A의 축적이 다른 대사 경로에 영향을 줄 수 있고, B와 C가 관여하는 대사 경로에도 장애가 발생한다.

효소 3 소실: 전구 물질 B에서 C로의 전환이 차단된다. B가 축적되고 C가 생성되지 않는다. B의 축적과 C 부족이 관련 대사 경로에 영향을 준다.

효소 4 소실: 전구 물질 C에서 비타민 C로의 전환만 차단된다. 전구 물질 A, B, C는 모두 정상적으로 생성되므로, 이들이 관여하는 다른 대사 경로는 정상적으로 작동한다. 비타민 C만 합성되지 않을 뿐, 다른 물질대사에는 최소한의 영향만 미친다.

따라서 효소 4의 소실이 세포의 전체 물질대사에 미치는 파급효과가 가장 작다. 효소 1~3이 소실되면 중간 전구 물질이 생성되지 않아 다른 대사 경로에도 연쇄적으로 영향을 미치지만, 효소 4는 대사 경로의 마지막 단계이므로 상류의 전구 물질 생산에 영향을 주지 않는다. 이것이 진화적으로 효소 4의 유전자가 소실된 이유를 설명한다.

1-3. 하디-바인베르크 법칙과 대립유전자 빈도

(3)

하디-바인베르크 법칙:

무한히 큰 멘델 집단에서 돌연변이, 자연선택, 유전자 이동, 유전적 부동, 비무작위 교배가 없을 때, 세대가 지나도 대립유전자 빈도와 유전자형 빈도가 변하지 않고 일정하게 유지된다는 법칙이다. 대립유전자 G의 빈도를 p, g의 빈도를 q라 하면 p + q = 1이고, 유전자형 빈도는 GG = \(p^2\), Gg = \(2pq\), gg = \(q^2\)이다.

대립유전자 G가 하나만 있어도 효소 4가 정상 기능하므로, G는 g에 대해 완전 우성이다.

계통수 분석:

사람(X) - 종 A(O) - 종 B(O) - 종 C(O)

사람: 모든 개체가 비타민 C 합성 불가(X) → 모든 개체의 유전자형이 gg → q = 1, p = 0

종 C: 모든 개체가 비타민 C 합성 가능(O) → gg인 개체가 없다. 모든 개체가 GG 또는 Gg → p = 1, q = 0 (순수하게 모든 개체가 합성 가능하면 g 대립유전자가 존재하지 않을 수 있음)

종 B: 모든 개체가 합성 가능(O) → 종 C와 마찬가지로 p = 1, q = 0이거나, g가 존재하더라도 gg 개체가 없을 수 있다. 모든 개체가 합성 가능이므로 gg = 0, 즉 \(q^2\) = 0 → q = 0, p = 1

종 A: 모든 개체가 합성 가능(O) → 위와 동일하게 p = 1, q = 0

단, 종 A의 일부 개체가 Gg(보인자)일 수 있다면 q > 0이지만 \(q^2\) = 0이므로 관찰적으로는 모두 합성 가능하다. 그러나 제시문에서 "모든 개체"가 합성 가능하다고 하였고, 멘델 집단에서 \(q^2\) = 0이 되려면 q = 0이어야 한다.

따라서:

- 사람: p = 0, q = 1

- 종 A: p = 1, q = 0

- 종 B: p = 1, q = 0

- 종 C: p = 1, q = 0

1-4. 효소 4 소실 생물의 이점

(4)

비타민 C 합성에는 포도당이 원료로 사용된다. 효소 4가 소실되어 비타민 C를 합성하지 못하는 생물은 포도당을 비타민 C 합성에 사용하지 않으므로, 그만큼의 포도당을 다른 용도에 활용할 수 있다.

포도당은 세포의 주된 에너지원으로 해당과정과 세포호흡을 통해 ATP를 생산하는 데 사용되며, 세포 구성 물질(아미노산, 지방, 핵산)의 합성을 위한 탄소 골격으로도 사용된다. 비타민 C 합성에 소비되던 포도당이 절약되면:

- 에너지 생산 효율이 높아져 생존과 번식에 유리하다.

- 성장과 발달에 더 많은 자원을 투입할 수 있다.

- 비타민 C를 음식(과일, 채소)에서 충분히 섭취할 수 있는 환경이라면, 합성에 드는 에너지와 자원을 절약하는 것이 적응적으로 유리하다.

1-5. 비타민 C 비합성 종의 종분화 과정

(5)

비타민 C를 합성하지 못하는 종의 종분화 과정을 유전자풀 변화 요인에 근거하여 유추하면:

공통 조상 개체군에서 효소 4 유전자에 기능 상실 돌연변이(g)가 발생한다. 이 돌연변이는 비타민 C가 풍부한 식이 환경(과일, 채소)에서는 생존에 불이익이 없으며, 오히려 포도당 절약 효과로 약간의 이점을 가질 수 있다.

유전적 부동(genetic drift): 특히 소규모 개체군에서 우연에 의해 g 대립유전자의 빈도가 증가할 수 있다(병목 효과, 창시자 효과).

자연선택: 비타민 C가 풍부한 식이를 하는 환경에서 g 대립유전자를 가진 개체가 포도당 절약에 의한 적응적 이점을 가져 양성 선택될 수 있다.

지리적 격리: 개체군이 지리적으로 격리되어 유전자 이동이 차단되면, 격리된 각 개체군에서 g 대립유전자의 빈도가 독립적으로 변화한다.

이러한 요인들이 복합적으로 작용하여 일부 개체군에서는 g 대립유전자가 고정(q = 1)되어 비타민 C 합성 능력을 완전히 상실하고, 이 개체군이 충분히 오래 격리되어 생식적 격리가 발생하면 새로운 종으로 분화된다.

1-6. 트립토판 영양 요구주 실험 설계

(6)

"트립토판이 풍부한 환경에서는 트립토판을 합성하지 못하는 종이 생존에 더 유리하다"는 가설을 검증하기 위한 실험:

실험 설계:

균주 준비:

- 야생형 대장균(트립토판 자체 합성 가능)

- 트립토판 영양 요구주(트립토판 합성 불가, 외부 공급 필요)

- 두 균주를 구별하기 위해 각각 다른 항생제 내성 표지를 부여하거나 형태적 차이를 이용한다.

실험군 (트립토판 풍부 환경):

트립토판이 충분히 포함된 최소배지에 야생형과 영양 요구주를 동일한 수로 혼합하여 접종한다. 일정 기간 배양 후 각 균주의 개체 수(콜로니 수)를 측정한다.

대조군 (트립토판 부족 환경):

트립토판이 포함되지 않은 최소배지(포도당과 질소원만 포함)에 동일하게 두 균주를 혼합하여 접종하고 배양 후 개체 수를 측정한다.

예상 결과:

- 가설이 맞다면: 트립토판 풍부 환경에서 영양 요구주가 야생형보다 개체 수가 많거나 비율이 증가한다(트립토판 합성에 드는 에너지와 자원을 절약하여 성장에 투자하므로).

- 트립토판 부족 환경에서는 영양 요구주가 트립토판을 합성할 수 없으므로 생장이 저해되어 야생형이 우세하다.

- 두 환경에서의 경쟁 결과 차이가 통계적으로 유의하면 가설이 지지된다.
