해설 2

역학적 에너지 보존: 절벽에서 던지기

높은 절벽(높이 \(H\))에서 같은 크기의 처음 속력 \(v_0\)로 세 가지 방법으로 던짐:

(1) 연직 위로 (2) 연직 아래로 (3) 수평으로

역학적 에너지 보존: \(\dfrac{1}{2}mv_0^2 + mgH = \dfrac{1}{2}mv_f^2\)

여기서 \(v_f\)는 지면에 닿을 때의 속력(크기)이다.

핵심: 역학적 에너지 보존 법칙에서 처음 속력 \(v_0\)와 높이 \(H\)가 세 경우 모두 동일하므로, 지면에 닿을 때의 속력 \(v_f\)도 세 경우 모두 동일하다.

\[v_f = \sqrt{v_0^2 + 2gH} \quad \text{(세 경우 모두 같음)}\]

이제 속도의 수직 방향 성분의 절댓값 비교:

(1) 연직 위로 던진 경우:

지면 도달 시 운동은 연직 방향뿐 (수평 성분 없음)

\(|v_{y1}| = v_f = \sqrt{v_0^2 + 2gH}\)

(2) 연직 아래로 던진 경우:

지면 도달 시 운동은 연직 방향뿐 (수평 성분 없음)

\(|v_{y2}| = v_f = \sqrt{v_0^2 + 2gH}\)

(3) 수평으로 던진 경우:

수평 성분: \(v_x = v_0\) (일정)

전체 속력: \(v_f = \sqrt{v_0^2 + 2gH}\)

수직 성분: \(|v_{y3}| = \sqrt{v_f^2 - v_x^2} = \sqrt{v_0^2 + 2gH - v_0^2} = \sqrt{2gH}\)

비교: \(|v_{y1}| = |v_{y2}| = \sqrt{v_0^2 + 2gH} > |v_{y3}| = \sqrt{2gH}\)

따라서 \(|v_{y1}| = |v_{y2}| > |v_{y3}|\)

이유: (1)과 (2)는 처음 속도가 모두 수직 방향이므로 운동 에너지 전부가 수직 성분에 기여한다. (3)은 수평 성분 \(v_0\)가 유지되므로 전체 에너지 중 일부가 수평 운동에 사용되어 수직 성분이 상대적으로 작아진다.
