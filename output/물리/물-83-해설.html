해설 2

영의 실험을 통해 빛의 회절과 간섭이 일어나는 조건 알기.

2-가

가. 이중 슬릿의 간섭실험에서

보강 간섭의 조건: \(|S_1P - S_2P| = m\lambda\), \(m = 0, 1, 2, 3, \ldots\)

상쇄 간섭의 조건: \(|S_1P - S_2P| = \lambda\left(m + \frac{1}{2}\right)\), \(m = 0, 1, 2, 3, \ldots\)

스크린까지의 거리가 슬릿 간격과 파장에 비해 충분히 먼 경우 근사적으로 \(\sin\theta = \tan\theta = x/L\) 로 볼 수 있다. 따라서 \(|S_1P - S_2P| = d\sin\theta = d\frac{x}{L}\) 로 계산된다.

슬릿 간격이 \(d=0.1\) mm, 슬릿에서 스크린까지의 거리는 \(L=1\) m 이고 중심으로부터 세번째 보강 간섭과 상쇄 간섭무늬까지의 거리 \(\Delta x\)는

\[\Delta x = \frac{L}{d} m\lambda \ (m=3) = 600 \times 10^{-9}\,m \times \frac{1\,m}{0.1 \times 10^{-3}\,m} \times 3 = 1.8 \times 10^{-2}\,m \ (\text{보강간섭})\]
\[\Delta x = \frac{L}{d}(m + \frac{1}{2})\lambda \ (m=2) = 600 \times 10^{-9}\,m \times \frac{1\,m}{0.1 \times 10^{-3}\,m} \times \frac{5}{2} = 1.5 \times 10^{-2}\,m \ (\text{상쇄간섭})\]

2-나

나. \(\Delta x\) 는 이중 슬릿에 입사하는 빛의 파장 \(\lambda\)에 비례함으로, 레이저 파장을 장파장으로 바꾸면 가)의 \(\Delta x\)는 파장의 비만큼 커질 것이고, 단파장으로 바꾸면 \(\Delta x\)는 파장의 비만큼 작아질 것이다.

2-다

다. 이중 슬릿의 폭이 커지면 그림과 같이 단일 슬릿에서 보이는 회절간섭 패턴과 이중 슬릿에서 보이는 간섭 패턴이 동시에 겹쳐 보이는 형태로 나타날 것이다.

슬릿에서 간격이 넓어지면 \(\Delta x = \frac{L}{d} m\lambda \ (or \ \frac{L}{d}(m+\frac{1}{2})\lambda)\) 에서 d 값이 커지는 경우이므로 반복되는 패턴 무늬간의 간격이 좁아지는 형태로 생길 것이다.

2-라

라. 결론은 "아무런 변화가 없다" 이다.

레이저에서 나오는 전자기파의 전기장(혹은 자기장)이 진행 방향에 대해서 진동하는 방향이 편광상태를 결정한다. 처음 슬릿에 들어오는 전자기파의 편광상태를 바꾸더라도 이중슬릿으로 지나가는 전자기파의 편광상태는 상대적으로 아무런 변화가 없다. 따라서 스크린에 생기는 간섭무늬 패턴은 이중슬릿을 지나온 전자기파의 상대적인 위상차이에만 관계가 있으므로, 간섭무늬 패턴은 레이저의 편광상태를 바꾸기 전과 후가 달라지지 않는다.
