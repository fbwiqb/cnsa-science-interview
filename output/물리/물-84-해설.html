해설 3

아르키메데스 법칙과 힘의 평형을 이용하는 문제

가. 부력의 크기를 구하는 문제

나. 힘의 평형을 이용하여 구에 작용하는 힘을 구하고, 줄의 장력을 구하는 문제

다. 유체의 밀도와 물체의 밀도를 비교하여, 물체가 뜨는지 또는 가라앉는지를 결정하는 문제

라. 깊이에 따른 유체의 압력에 관한 문제

3-가

가. 구의 부피는

\[V_{ball} = \frac{4}{3}\pi r^3 = \frac{4}{3} \times 3 \times (0.2)^3 = 0.032\,m^3\]

부력은 구가 밀어낸 부피만큼의 물의 무게이므로

\[F_B = \rho_w V_{ball} g = 1000\,kg/m^3 \times 0.032\,m^3 \times 10\,N/kg = 320\,N\]

3-나

나. 구에 작용하는 힘을 구하면

부력 = 구의 무게 + 줄의 장력

구의 무게는

\[F_g = m_{ball} g = 27\,kg \times 10\,N/kg = 270\,N\]

따라서, 줄의 장력 = 부력 - 공의 무게 = 320N - 270N = 50N

3-다

다. 부력이 구의 무게보다 크므로 수면 위로 떠오른다

구 전체부피의 \(x\), 즉 \(x \cdot V_{ball}\) 만큼 수면 아래 잠겨 있다면 힘의 평형에 의해

\[m_{ball} g \ (= \rho_{ball} V_{ball} g) = x \cdot V_{ball} \rho_w g\]
\[x \left(= \frac{\rho_{ball}}{\rho_w}\right) = \frac{m_{ball}}{V_{ball} \times \rho_w} = \frac{27\,kg}{0.032\,m^3 \times 1000\,kg/m^3} = \frac{27}{32}\]

잠겨 있는 부분이 전체부피의 \(\frac{27}{32}\) 이므로, 수면 밖으로 나와 있는 부분은 전체부피의 \(\frac{5}{32}\) 가 된다

3-라

라. 표를 보면 \(y = \frac{3}{4}D\)일 때 잠겨 있는 부분이 전체 부피의 \(\frac{27}{32}\)이 됨을 유추할 수 있다. 물 속 깊이 \(h\) 에서의 압력 \(p = p_0 + \rho_w g h\)이고, 여기서 \(h = \frac{3}{4}D = \frac{3}{4} \times 0.4 = 0.3\,m\) 이다. 여기 서 \(p_0\) 는 물 표면위의 압력, 즉 대기압을 나타낸다. 따라서, 가장 아랫점과 대기압과의 차이는

\[p - p_0 = \rho_w g h = 1000\,kg/m^3 \times 10\,N/kg \times 0.3\,m = 3000\,N/m^2 = 3000\,Pa\]
