단순 조화 진동과 위상 공간 분석

(3-1) 역학적 에너지 관계와 특수 위치

단순 조화 진동계에서 역학적 에너지 \(E\)는 운동 에너지 \(E_k\)와 위치 에너지 \(E_s\)의 합으로 항상 일정하다.

\[E = E_k + E_s = \frac{p^2}{2m} + \frac{1}{2}kx^2\]

최대 운동 에너지는 위치 에너지가 0일 때(\(x = 0\), 평형점) 달성된다.

\[\boxed{E_{k,\max} = E}\]

변위가 최대인 위치는 운동 에너지가 0일 때이므로 \(E = \frac{1}{2}kx_{\max}^2\)에서 다음을 얻는다.

\[\boxed{x_{\max} = \pm\sqrt{\frac{2E}{k}}}\]

평형점에서의 거리는 \(\sqrt{\frac{2E}{k}}\)이다.

\(E_s = E_k\)인 위치를 구하면, \(E_s = E_k = \frac{E}{2}\)이므로 다음과 같다.

\[\frac{1}{2}kx^2 = \frac{E}{2}\]
\[x^2 = \frac{E}{k}\]
\[\boxed{x = \pm\sqrt{\frac{E}{k}}}\]

(3-2) 운동량 함수와 시간 평균 운동 에너지

변위가 \(x(t) = \sqrt{\frac{2E}{k}}\cos\omega t\)로 주어지므로, 운동량은 다음과 같다.

\[p(t) = m\frac{dx}{dt} = -m\omega\sqrt{\frac{2E}{k}}\sin\omega t\]

\(\omega = \sqrt{k/m}\)이므로 \(m\omega = m\sqrt{k/m} = \sqrt{mk}\)를 대입하면 다음과 같다.

\[\boxed{p(t) = -\sqrt{2mE}\sin\omega t}\]

시간에 대한 평균 운동 에너지를 계산한다.

\[\langle E_k \rangle = \frac{1}{T}\int_0^T E_k(t)\,dt = \frac{1}{T}\int_0^T \frac{p(t)^2}{2m}\,dt\]
\[= \frac{1}{T}\int_0^T \frac{2mE\sin^2\omega t}{2m}\,dt = \frac{E}{T}\int_0^T \sin^2\omega t\,dt\]

\(\sin^2\omega t\)의 한 주기 평균은 \(\frac{1}{2}\)이므로 다음을 얻는다.

\[\boxed{\langle E_k \rangle = \frac{E}{2}}\]

이는 전체 역학적 에너지의 \(\frac{1}{2}\)에 해당하며, 비리얼 정리(virial theorem)와 일치한다.

(3-3) 위상 공간에서의 면적과 물리적 의미

위상 공간 \((x, p)\)에서 에너지 \(E\)에 해당하는 궤적은 다음과 같다.

\[\frac{x^2}{2E/k} + \frac{p^2}{2mE} = 1\]

이는 반장축 \(a = \sqrt{2E/k}\), 반단축 \(b = \sqrt{2mE}\)인 타원이다. 타원의 면적은 다음과 같다.

\[A_E = \pi ab = \pi\sqrt{\frac{2E}{k}}\cdot\sqrt{2mE} = \frac{2\pi E}{\omega}\]

여기서 \(\omega = \sqrt{k/m}\)을 사용하였다. 따라서 다음과 같다.

\[A_E = \frac{2\pi E}{\omega} = ET\]

에너지가 \(E\)와 \(E + \Delta E\) 사이의 면적은 다음과 같다.

\[A_{E+\Delta E} - A_E = \frac{2\pi}{\omega}\Delta E = T\Delta E\]

이 면적의 물리적 의미는 에너지가 \(E\)와 \(E + \Delta E\) 사이에 있는 미시 상태의 수에 비례하는 양이다. 위상 공간에서 두 에너지 곡선 사이의 면적은 해당 에너지 범위 내에서 계가 접근할 수 있는 상태의 수, 즉 상태 밀도(density of states)에 대응한다. \(\Delta E\)가 충분히 작을 때, 이 면적은 \(T\Delta E\)로 주기 \(T\)와 에너지 변화량 \(\Delta E\)의 곱이 된다.

(3-4) 최대 위상 면적 조건과 온도

5개의 독립적인 단순 진동계의 에너지가 각각 \(e_1, e_2, \ldots, e_5\)이고, 에너지 \(e_n\)에 대한 위상 면적이 \(\Omega_n\)으로 주어진다. (3-3)의 결과에 의해 \(\Omega_n \propto e_n\)이다.

전체 위상 면적은 다음과 같다.

\[\Omega_{\text{tot}} = \Omega_1 \cdot \Omega_2 \cdots \Omega_5\]

전체 에너지 조건 \(e_1 + e_2 + \cdots + e_5 = S\)를 만족하면서 \(\Omega_{\text{tot}}\)을 최대화하는 조건을 구한다. \(\Omega_n \propto e_n\)이므로 다음을 최대화한다.

\[\Omega_{\text{tot}} \propto e_1 \cdot e_2 \cdots e_5\]

라그랑주 승수법을 적용한다. \(\ln\Omega_{\text{tot}} = \sum_{n=1}^{5}\ln e_n\)을 \(\sum e_n = S\) 조건 하에서 최대화하면 다음과 같다.

\[\frac{\partial}{\partial e_n}\left(\sum_{i=1}^{5}\ln e_i - \lambda\sum_{i=1}^{5}e_i\right) = 0\]
\[\frac{1}{e_n} = \lambda \quad \text{(모든 } n \text{에 대해)}\]

따라서 최대 조건은 모든 진동계의 에너지가 같을 때이다.

\[\boxed{e_1^* = e_2^* = e_3^* = e_4^* = e_5^* = \frac{S}{5}}\]

\(\Omega(S)\)와 절대온도 \(T\)의 관계는 통계역학의 볼츠만 관계에서 유도된다. 엔트로피 \(\mathcal{S}\)를 다음과 같이 정의하면

\[\mathcal{S} = k_B \ln\Omega\]

온도와의 관계는 다음과 같다.

\[\boxed{\frac{1}{T} = \frac{\partial \mathcal{S}}{\partial E} = k_B \frac{\partial \ln\Omega}{\partial E}}\]

즉, \(\frac{1}{k_BT} = \frac{d\ln\Omega}{dE}\)이다.
