2-1. 이중슬릿 간섭무늬의 간격

입자 X의 드브로이 파장:

\[\lambda = \frac{h}{p} = \frac{h}{mv_0}\]

이중슬릿 간섭의 보강간섭 조건: \(d\sin\theta = n\lambda\) (\(n\)은 정수)

\(L \gg d\)이므로 \(\sin\theta \approx \tan\theta = \frac{y}{L}\)로 근사할 수 있다.

이웃한 보강간섭 지점 간의 간격:

\[x = \frac{\lambda L}{d} = \frac{hL}{mv_0 d}\]
\[\boxed{x = \frac{hL}{mv_0 d}}\]

2-2. 전기장 통과 후 간섭무늬 변화

입자 X(전하량 \(Q > 0\))가 간격 \(s\)인 평행판 축전기를 통과한다. 전기장 \(E\)가 이동 방향과 같은 방향으로 걸려 있으므로, 입자에 작용하는 힘은 \(F = QE\) (이동 방향)이다.

축전기 통과 후의 속력:

에너지 보존 (또는 일-에너지 정리):

\[\frac{1}{2}mv'^2 = \frac{1}{2}mv_0^2 + QEs\]
\[v' = \sqrt{v_0^2 + \frac{2QEs}{m}}\]

간섭무늬 간격 변화:

새로운 드브로이 파장:

\[\lambda' = \frac{h}{mv'}\]

새로운 보강간섭 간격:

\[x' = \frac{\lambda' L}{d} = \frac{hL}{mv'd} = \frac{hL}{d\sqrt{m^2v_0^2 + 2mQEs}}\]
\[\boxed{x' = \frac{hL}{d\sqrt{m^2v_0^2 + 2mQEs}}}\]

전기장에 의해 입자가 가속되어 \(v' > v_0\)이므로 \(\lambda' < \lambda\)이고, 따라서 \(x' < x\)이다. 간섭무늬의 간격이 줄어든다.

2-3. 입자 X의 등속 원운동 조건

(a) UFO까지의 거리 $R$

입자 X가 지구 주위를 등속 원운동하려면, 중력이 구심력을 제공해야 한다:

\[\frac{GM_E m}{R^2} = \frac{mv_0^2}{R}\]
\[\frac{GM_E}{R} = v_0^2\]
\[\boxed{R = \frac{GM_E}{v_0^2}}\]

(b) 입자가 방출되는 방향

등속 원운동을 하려면 속도는 궤도의 접선 방향, 즉 반지름에 수직해야 한다. UFO가 궤도 위의 한 점이므로, 입자는 지구 중심과 UFO를 잇는 직선에 수직인 방향으로 방출되어야 한다.

2-4. 입자 X가 UFO로 돌아오는 최소 시간

UFO에서 방출 방향은 2-3(b)와 같이 반지름에 수직이지만, 속력이 \(v_0\)와 다르다.

입자가 지구에 충돌하거나 통과하지 않고 UFO로 돌아와야 한다. 이는 타원 궤도의 경우이다.

궤도 분석:

UFO 위치에서 속력 \(v\)로 접선 방향으로 방출된 입자는 타원(또는 원, 포물선, 쌍곡선) 궤도를 따른다. UFO로 돌아오려면 닫힌 궤도(타원)여야 한다.

UFO 위치(반지름 \(R\))에서 접선 방향 속도 \(v\)일 때:

- \(v = v_0 = \sqrt{GM_E/R}\): 원궤도

- \(v < v_0\): UFO가 원일점인 타원궤도

- \(v_0 < v < \sqrt{2}v_0\): UFO가 근일점인 타원궤도

지구에 충돌하지 않으려면 근일점 거리가 \(R_E\)보다 커야 한다.

UFO가 원일점인 경우 ($v < v_0$):

각운동량 보존: \(Rv = r_{\min}v_{\max}\)

에너지 보존:

\[\frac{1}{2}v^2 - \frac{GM_E}{R} = \frac{1}{2}v_{\max}^2 - \frac{GM_E}{r_{\min}}\]

\(v_{\max} = \frac{Rv}{r_{\min}}\) 대입:

\[\frac{1}{2}v^2 - \frac{GM_E}{R} = \frac{R^2v^2}{2r_{\min}^2} - \frac{GM_E}{r_{\min}}\]

지구에 충돌하지 않는 조건: \(r_{\min} \geq R_E\)

충돌 경계: \(r_{\min} = R_E\)

각운동량 보존으로부터:

\[v_{\max} = \frac{Rv}{R_E}\]

에너지 보존:

\[\frac{1}{2}v^2 - \frac{GM_E}{R} = \frac{R^2v^2}{2R_E^2} - \frac{GM_E}{R_E}\]
\[\frac{1}{2}v^2\left(1 - \frac{R^2}{R_E^2}\right) = \frac{GM_E}{R_E} - \frac{GM_E}{R} = GM_E\frac{R - R_E}{RR_E}\]

\(R > R_E\)이므로 왼변을 정리하면:

\[\frac{v^2}{2} \cdot \frac{-(R + R_E)(R - R_E)}{R_E^2} = GM_E\frac{R - R_E}{RR_E}\]

\(R \neq R_E\)이므로 \(R - R_E\)로 나누면:

\[-\frac{v^2(R + R_E)}{2R_E^2} = \frac{GM_E}{RR_E}\]
\[v^2 = \frac{2GM_E R_E}{R(R + R_E)}\]

이것이 근일점이 정확히 \(R_E\)가 되는 속력이다.

최소 시간으로 돌아오려면, 타원의 주기가 최소여야 한다. 주기는 장반경에 의해 결정된다:

\[T = 2\pi\sqrt{\frac{a^3}{GM_E}}\]

타원의 장반경:

\[a = \frac{R + r_{\min}}{2}\]

지구에 부딪히지 않는 조건에서 \(r_{\min} \geq R_E\)이므로, 주기를 최소화하려면 \(r_{\min}\)을 최소화, 즉 \(r_{\min} = R_E\)로 설정한다.

\[a = \frac{R + R_E}{2}\]

타원 궤도의 주기:

\[T_{\text{orbit}} = 2\pi\sqrt{\frac{a^3}{GM_E}} = 2\pi\sqrt{\frac{(R + R_E)^3}{8GM_E}}\]

입자가 UFO에서 출발하여 UFO로 돌아오는 시간은 한 바퀴, 즉 \(T_{\text{orbit}}\)이다.

\[\boxed{T_{\min} = 2\pi\sqrt{\frac{(R + R_E)^3}{8GM_E}} = \frac{\pi(R + R_E)}{2}\sqrt{\frac{R + R_E}{2GM_E}}}\]

또는 정리하면:

\[\boxed{T_{\min} = \pi\sqrt{\frac{(R + R_E)^3}{2GM_E}}}\]
