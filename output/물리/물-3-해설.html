2-1. 3차원 우주 U와 2차원 우주 U'에서의 에너지

3차원 우주 U:

원운동 조건: \(\frac{GMm}{R^2} = \frac{mv^2}{R}\) → \(v^2 = \frac{GM}{R}\)

운동 에너지: \(K = \frac{1}{2}mv^2 = \frac{GMm}{2R}\)

중력 퍼텐셜 에너지: \(U = -\frac{GMm}{R}\) (\(r = \infty\)에서 \(U = 0\))

(1) 행성 1($R$)과 행성 2($10R$)의 운동 에너지 차이:

\[K_1 - K_2 = \frac{GMm}{2R} - \frac{GMm}{20R} = \frac{GMm}{2R}\left(1 - \frac{1}{10}\right) = \frac{9GMm}{20R}\]
\[\boxed{K_1 - K_2 = \frac{9GMm}{20R}}\]

(2) 중력 퍼텐셜 에너지 차이:

\[U_1 - U_2 = -\frac{GMm}{R} + \frac{GMm}{10R} = -\frac{9GMm}{10R}\]
\[\boxed{U_1 - U_2 = -\frac{9GMm}{10R}}\]

2차원 우주 U':

힘: \(F' = G'\frac{Mm}{r}\)

원운동 조건: \(\frac{G'Mm}{R} = \frac{mv^2}{R}\) → \(v^2 = G'M\) (반지름에 무관!)

(3) 운동 에너지 차이:

\[K_1' = K_2' = \frac{1}{2}mG'M\]
\[\boxed{K_1' - K_2' = 0}\]

2차원 우주에서 원궤도의 운동 에너지는 반지름에 무관하다.

(4) 중력 퍼텐셜 에너지 차이:

2차원에서 힘: \(F' = \frac{G'Mm}{r}\)

중력 퍼텐셜 에너지: 문제에서 "중력과 크기는 같고 방향이 반대인 힘이 \(r_0\)부터 \(r\)까지 해준 일"을 퍼텐셜 에너지로 정의하므로:

\[U'(r) = \int_{r_0}^r \frac{G'Mm}{r'}\,dr' = G'Mm\ln\frac{r}{r_0}\]

\(r\)이 커질수록 \(U'\)이 커지므로 물리적으로 올바르다 (인력이므로 멀수록 퍼텐셜 에너지가 높다).

참고: 2차원에서 \(1/r\) 힘의 퍼텐셜은 로그 함수이므로 \(r \to \infty\)에서 발산한다. 따라서 \(r = \infty\)에서 \(U = 0\)으로 놓을 수 없고, 유한한 기준점 \(r_0\)에서 \(U = 0\)으로 잡아야 한다.

행성 1과 행성 2의 퍼텐셜 에너지 차이:

\[U_1' - U_2' = G'Mm\ln\frac{R}{r_0} - G'Mm\ln\frac{10R}{r_0} = G'Mm\ln\frac{R}{10R} = -G'Mm\ln 10\]
\[\boxed{U_1' - U_2' = -G'Mm\ln 10}\]

행성 1이 더 가까우므로 퍼텐셜 에너지가 더 낮다.

2-2. 2차원 우주에서의 케플러 제3법칙

3차원: \(v^2 = GM/R\) → \(T = \frac{2\pi R}{v} = 2\pi R\sqrt{R/(GM)} = \frac{2\pi}{\sqrt{GM}}R^{3/2}\)

\(T^2 \propto R^3\) (케플러 제3법칙)

2차원: \(v^2 = G'M\) (반지름에 무관)

\[T = \frac{2\pi R}{v} = \frac{2\pi R}{\sqrt{G'M}}\]
\[T^2 = \frac{4\pi^2}{G'M}R^2\]
\[\boxed{T^2 \propto R^2 \quad \text{(또는 } T \propto R\text{)}}\]

2차원 우주에서 공전 주기의 제곱은 궤도 반지름의 제곱에 비례한다. (3차원에서는 \(R^3\)에 비례)

2-3. 2차원 우주에서의 보어 모형

보어 양자 조건: \(m_e v r = n\hbar\) (\(\hbar = h/(2\pi)\))

2차원에서 원운동 조건: \(\frac{k'q_e^2}{r} = \frac{m_e v^2}{r}\) → \(v^2 = k'q_e^2/m_e\) (반지름 무관!)

\(v = \sqrt{k'q_e^2/m_e}\) (모든 궤도에서 같은 속력)

양자 조건에서:

\[m_e v r_n = n\hbar\]
\[r_n = \frac{n\hbar}{m_e v} = \frac{n\hbar}{m_e}\sqrt{\frac{m_e}{k'q_e^2}} = \frac{n\hbar}{\sqrt{m_e k'q_e^2}}\]

(1)

\[\boxed{r_n' = \frac{n\hbar}{\sqrt{m_e k' q_e^2}} = n \cdot r_1'}\]

여기서 \(r_1' = \frac{\hbar}{\sqrt{m_e k' q_e^2}}\)

에너지:

운동 에너지: \(K_n = \frac{1}{2}m_e v^2 = \frac{k'q_e^2}{2}\) (모든 \(n\)에서 같음)

퍼텐셜 에너지: 전기력도 \(1/r\) 힘이므로 중력의 경우와 마찬가지로 로그 퍼텐셜이다.

전자에 작용하는 전기력은 인력(\(-k'q_e^2/r\), 중심 방향)이므로, 기준점 \(r_0\)에서 \(U = 0\)으로 잡으면:

\[U(r) = \int_{r_0}^{r} \frac{k'q_e^2}{r'}\,dr' = k'q_e^2 \ln\frac{r}{r_0}\]

기준점을 \(r_0 = r_1'\)로 잡으면:

\[U_n = k'q_e^2 \ln\frac{r_n}{r_1'} = k'q_e^2 \ln n\]

전체 에너지:

\[E_n = K_n + U_n = \frac{k'q_e^2}{2} + k'q_e^2 \ln n\]
\[\boxed{E_n' = \frac{k'q_e^2}{2} + k'q_e^2\ln n}\]

\(E_1' = \frac{k'q_e^2}{2}\)이고, \(n\)이 증가하면 에너지도 증가한다.

(2) 전이 시 방출되는 빛의 파장

\(n_2 \to n_1\) 전이 (\(n_2 > n_1\)):

\[\Delta E = E_{n_2}' - E_{n_1}' = k'q_e^2\ln\frac{n_2}{n_1}\]
\[\frac{hc}{\lambda} = k'q_e^2\ln\frac{n_2}{n_1}\]
\[\boxed{\lambda = \frac{hc}{k'q_e^2\ln(n_2/n_1)}}\]

2-4. 2차원 수소 원자의 이온화와 결합

(1) 바닥상태 이온화 에너지:

전자를 \(n = 1\)에서 \(n = \infty\)로 보내는 데 필요한 에너지:

\[E_\infty - E_1 = \lim_{n \to \infty}\left(\frac{k'q_e^2}{2} + k'q_e^2\ln n\right) - \frac{k'q_e^2}{2} = \lim_{n \to \infty} k'q_e^2\ln n = \infty\]

이온화 에너지가 무한대이다!

\[\boxed{E_{\text{ion}} = \infty}\]

2차원 우주에서는 수소 원자의 전자를 완전히 떼어내는 것이 불가능하다 (로그 퍼텐셜이 무한히 증가하므로).

(2) 원자 결합의 차이:

3차원 우주에서는 이온화 에너지가 유한(13.6 eV)하여 원자가 이온화될 수 있고, 원자들 사이에 전자를 공유하거나 이동시켜 화학 결합을 형성할 수 있다.

2차원 우주에서는 이온화 에너지가 무한대이므로, 전자를 원자에서 떼어내는 것이 불가능하다. 따라서:

- 이온 결합은 불가능하다 (전자를 완전히 이동시킬 수 없음).

- 공유 결합도 형성하기 어렵다.

- 원자는 다른 원자와 결합하지 못하고, 화학 반응이 일어나기 매우 어렵다.

2-5. 퍼텐셜 에너지 함수의 근본적 차이

3차원 우주 U: 퍼텐셜 에너지 \(U(r) = -\frac{kq_e^2}{r}\)

- \(r \to 0\): \(U \to -\infty\) (깊은 우물)

- \(r \to \infty\): \(U \to 0\) (유한값에 수렴)

- 퍼텐셜 에너지가 유한한 벽(\(U = 0\))에 접근하므로, 충분한 에너지를 가진 전자는 탈출할 수 있다.

2차원 우주 U': 퍼텐셜 에너지 \(U'(r) = k'q_e^2\ln(r/r_0)\)

- \(r \to 0\): \(U' \to -\infty\) (깊은 우물)

- \(r \to \infty\): \(U' \to +\infty\) (무한히 높은 벽)

- 퍼텐셜 에너지가 무한히 증가하므로, 전자가 아무리 많은 에너지를 가져도 무한히 멀리 탈출할 수 없다.

근본적 원인: \(1/r^2\) 힘(3차원)의 퍼텐셜은 \(-1/r\)로 유한한 값에 수렴하지만, \(1/r\) 힘(2차원)의 퍼텐셜은 \(\ln r\)로 무한히 발산한다. 이 로그 발산이 2차원 우주에서의 이온화 불가능과 결합 불가능의 근본 원인이다.

2-6. 2차원에서 전기력이 거리에 반비례하는 이유

가우스 법칙의 관점:

3차원에서 점전하 \(q\)로부터 나오는 전기력선의 총 수는 \(q/\varepsilon_0\)로 일정하다. 반지름 \(r\)인 구면의 표면적은 \(4\pi r^2\)이므로, 단위 면적당 전기력선 수(전기장 크기)는:

\[E \propto \frac{1}{r^2}\]

따라서 전기력: \(F \propto \frac{1}{r^2}\)

2차원에서: 점전하에서 나오는 전기력선은 2차원 평면 위에 퍼진다. 반지름 \(r\)인 원의 둘레는 \(2\pi r\)이다. 전기력선의 총 수가 일정하므로, 단위 길이당 전기력선 수(전기장 크기)는:

\[E' \propto \frac{1}{r}\]

따라서 전기력: \(F' \propto \frac{1}{r}\)

3차원에서는 힘이 닫힌 곡면(구)의 면적(\(r^2\)에 비례)에 반비례하고, 2차원에서는 닫힌 곡선(원)의 둘레(\(r\)에 비례)에 반비례한다. 이것이 차원에 따라 힘의 거리 의존성이 달라지는 근본 원인이다.
