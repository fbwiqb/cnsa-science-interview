1-1. 두 매질에서의 이중슬릿 간섭

슬릿 사이 거리 \(d\), 슬릿-스크린 거리 \(L\). 슬릿~스크린 사이 매질의 굴절률 \(n_2\).

매질 속에서 빛의 파장: \(\lambda_2 = \frac{\lambda}{n_2}\)

이중슬릿 간섭에서, 경로차가 파장의 정수배이면 보강간섭, 반정수배이면 상쇄간섭.

슬릿과 스크린 사이의 매질이 \(n_2\)이므로, 실효 파장은 \(\lambda/n_2\)이다.

보강 간섭 조건: \(d\sin\theta = m\frac{\lambda}{n_2}\) (\(m = 0, \pm 1, \pm 2, \ldots\))

\(\sin\theta \approx \frac{y}{L}\) 근사:

\[\Delta y = \frac{\lambda L}{n_2 d}\]

상쇄 간섭 조건: \(d\sin\theta = \left(m + \frac{1}{2}\right)\frac{\lambda}{n_2}\)

인접한 상쇄 간섭 무늬 사이 거리:

\[\Delta y' = \frac{\lambda L}{n_2 d}\]

따라서 보강과 상쇄 간섭의 간격은 같다:

\[\boxed{\Delta y = \Delta y' = \frac{\lambda L}{n_2 d}}\]

참고: 단일 슬릿~이중 슬릿 사이의 매질(\(n_1\))은 두 슬릿에 도달하는 빛의 위상에 영향을 준다. 그러나 단일 슬릿이 중앙에 있으면 두 슬릿까지의 광경로가 같으므로, 위상차에 영향을 주지 않는다. 따라서 간섭 패턴은 \(n_2\)에만 의존한다.

1-2. 가변 굴절률 매질에서의 밝은 무늬 수

\(n(\theta)\)가 \(\theta\)에 따라 변하고, \(\lambda = \frac{\pi}{20}d\).

반원 스크린에서, 각도 \(\theta\)에서의 보강간섭 조건:

\[d\sin\theta = m\frac{\lambda}{n(\theta)} = m\frac{\pi d}{20 n(\theta)}\]
\[20 n(\theta)\sin\theta = m\pi\]
\[m = \frac{20 n(\theta)\sin\theta}{\pi}\]

밝은 무늬의 수는 \(m\)이 양의 정수가 되는 \(\theta\) 값의 수이다.

풀이 방법: \(\theta = 0\)에서 \(\theta = \pi/2\)까지 \(f(\theta) = n(\theta)\sin\theta\)의 범위를 구하고, \(\frac{20}{\pi}f(\theta)\)가 정수가 되는 점의 수를 센다. \(\theta < 0\) 쪽도 대칭이므로 같은 수의 무늬가 존재한다. 중앙 밝은 무늬(\(m = 0\))를 포함하여 양쪽의 밝은 무늬를 모두 합산한다.

\(f(\theta)\)의 최댓값을 \(f_{\max}\)라 하면, 한쪽에서 가능한 \(m\)의 최대 정수는 \(\lfloor\frac{20f_{\max}}{\pi}\rfloor\)이다.

(정확한 답은 \(n(\theta)\) 함수의 구체적 형태에 의존한다.)

1-3. 평면 스크린에서의 밝은 무늬 간격

(1) $\Delta y_2$ 계산

가변 굴절률 매질에서 \(n\)번째 밝은 무늬의 위치:

\[d\sin\theta_n = n\frac{\lambda}{n(\theta_n)}\]

평면 스크린에서 \(y_n = L\tan\theta_n\)

\(\Delta y_n = y_n - y_{n-1} = L(\tan\theta_n - \tan\theta_{n-1})\)

\(n(\theta)\)의 형태에 따라 구체적으로 계산해야 한다.

(2) $\Delta y_n$의 변화

일반적으로, 스크린 중앙에서 멀어질수록 \(\theta\)가 커지고 \(\tan\theta\)가 급격히 증가한다. 따라서 \(\Delta y_n\)은 스크린 중앙에서 멀어질수록 증가한다.

이는 \(\frac{d(\tan\theta)}{d\theta} = \sec^2\theta\)가 \(\theta\)에 따라 단조증가하기 때문이다.

또한 \(n(\theta)\)가 \(\theta\)에 따라 증가하면, \(\frac{\lambda}{n(\theta)}\)가 감소하여 각도 간격은 줄어들지만, \(\tan\theta\)의 증가 효과가 더 지배적이므로 \(\Delta y_n\)은 여전히 증가한다.

1-4. 단진동하는 음원의 도플러 효과

(1) 음원의 속도

\[x(t) = A\sin\left(\sqrt{\frac{k}{m}}t\right)\]
\[\boxed{v(t) = \frac{dx}{dt} = A\sqrt{\frac{k}{m}}\cos\left(\sqrt{\frac{k}{m}}t\right)}\]

(2) $\Delta y$를 $t$의 함수로 표현

음원이 단진동하면서 이중슬릿 방향으로 운동하므로, 도플러 효과에 의해 음파의 파장이 변한다.

음원이 속도 \(v_s(t)\)로 슬릿 방향으로 접근할 때 (\(v_s > 0\)이면 접근), 슬릿에 도달하는 음파의 파장은:

\[\lambda'(t) = \lambda\left(1 - \frac{v_s(t)}{V}\right) = \lambda\left(1 - \frac{A\sqrt{\frac{k}{m}}\cos\left(\sqrt{\frac{k}{m}}t\right)}{V}\right)\]

슬릿과 스크린 사이에서 음파의 파장은 \(\lambda'(t)\)이므로, 인접한 보강간섭 극대점 사이의 거리는:

\[\boxed{\Delta y(t) = \frac{\lambda'(t) L}{d} = \frac{\lambda L}{d}\left(1 - \frac{A\sqrt{\frac{k}{m}}\cos\left(\sqrt{\frac{k}{m}}t\right)}{V}\right)}\]

\(v_{\max} = A\sqrt{k/m}\)으로 놓으면:

\[\Delta y(t) = \frac{\lambda L}{d}\left(1 - \frac{v_{\max}\cos\left(\sqrt{\frac{k}{m}}t\right)}{V}\right)\]

(3) $\Delta y$의 한 주기 내 변화

\(\omega = \sqrt{k/m}\)으로 놓으면 \(v_s(t) = v_{\max}\cos(\omega t)\)이다.

- \(t = 0\): \(v_s = v_{\max}\) (슬릿에 접근), \(\Delta y\)가 최소 \(= \frac{\lambda L}{d}(1 - v_{\max}/V)\)

- \(t = T/4\): \(v_s = 0\) (정지), \(\Delta y = \frac{\lambda L}{d}\) (정지 시의 값)

- \(t = T/2\): \(v_s = -v_{\max}\) (슬릿에서 멀어짐), \(\Delta y\)가 최대 \(= \frac{\lambda L}{d}(1 + v_{\max}/V)\)

- \(t = 3T/4\): \(v_s = 0\), 다시 \(\Delta y = \frac{\lambda L}{d}\)

- \(t = T\): 한 주기 완료, \(\Delta y\)가 다시 최소

\(\Delta y\)는 주기 \(T\)로 코사인 형태의 진동을 한다. 음원이 슬릿에 접근할 때 파장이 짧아져 \(\Delta y\)가 줄고, 멀어질 때 파장이 길어져 \(\Delta y\)가 늘어난다.

1-5. 광전효과와 도플러 효과

(1) 검출되는 입자는 광전자(전자)이다.

금속에 빛이 입사하면 광전효과에 의해 전자가 튀어나온다. 음원이 금속 스크린을 향해 다가갈 때, 도플러 효과에 의해 관측 진동수가 증가하여 광전효과가 일어난다.

\(t = 0\)에서 검출되고, \(t = T/8\)부터 검출되지 않는다.

\(t = 0\)에서 음원의 속도: \(v(0) = A\sqrt{k/m} = \frac{c}{2\sqrt{2}}\)

음원이 금속 스크린을 향해 \(x\) 방향으로 단진동한다.

도플러 효과에 의한 관측 진동수: \(f' = \frac{c}{c - v_s}f_0\)

\(t = 0\): \(x = 0\), \(v = A\sqrt{k/m} = \frac{c}{2\sqrt{2}}\) (스크린을 향해)

도플러 효과: \(f' = \frac{c}{c - v_s(t)}f_0\)

\(t = T/8\): \(v_s = \frac{c}{2\sqrt{2}}\cos(\pi/4) = \frac{c}{2\sqrt{2}} \cdot \frac{1}{\sqrt{2}} = \frac{c}{4}\)

\(t = T/8\)에서 광전효과가 멈추므로, 이때의 진동수가 문턱 진동수와 같다:

\[f_{\text{threshold}} = \frac{c}{c - c/4}f_0 = \frac{4}{3}f_0\]

\(t = 0\)에서 진동수: \(f'_0 = \frac{c}{c - c/(2\sqrt{2})}f_0 = \frac{1}{1 - 1/(2\sqrt{2})}f_0 = \frac{2\sqrt{2}}{2\sqrt{2}-1}f_0\)

광전효과가 \(t = T/8\)부터 멈추므로, 진동수가 문턱 이하로 내려가는 것이다.

(2) 다시 검출되기 시작하는 시간

진동수가 다시 문턱 진동수 이상이 되는 시간을 구한다.

\(v_s(t) = \frac{c}{2\sqrt{2}}\cos\left(\frac{2\pi t}{T}\right)\)

문턱 조건: \(f' = f_{\text{threshold}}\)

\[\frac{c}{c - v_s} = \frac{c}{c - c/4} = \frac{4}{3}\]
\[c - v_s = \frac{3c}{4}\]
\[v_s = \frac{c}{4}\]
\[\frac{c}{2\sqrt{2}}\cos\left(\frac{2\pi t}{T}\right) = \frac{c}{4}\]
\[\cos\left(\frac{2\pi t}{T}\right) = \frac{\sqrt{2}}{2} = \frac{1}{\sqrt{2}}\]
\[\frac{2\pi t}{T} = \pm\frac{\pi}{4} + 2n\pi\]

\(t = T/8\) (이미 알고 있는 해)와 \(t = 7T/8\) (다음 해).

\(T/8 < t < 7T/8\) 구간에서 \(\cos\)가 \(1/\sqrt{2}\) 미만이므로 광전효과 없음.

\[\boxed{t = \frac{7T}{8}}\]

(3) 최대 운동 에너지

광전자의 최대 운동 에너지는 입사 광자의 에너지가 최대일 때 발생한다. 이는 도플러 이동이 최대, 즉 \(v_s\)가 최대인 \(t = 0\)일 때이다.

\[v_{s,\max} = \frac{c}{2\sqrt{2}}\]

최대 진동수:

\[f'_{\max} = \frac{c}{c - c/(2\sqrt{2})}f_0 = \frac{2\sqrt{2}}{2\sqrt{2} - 1}f_0\]

광전효과: \(E_k = hf' - W\) (일함수 \(W = hf_{\text{threshold}}\))

\[W = hf_{\text{threshold}} = h \cdot \frac{4}{3}f_0 = \frac{4hf_0}{3} = \frac{4hc}{3\lambda}\]
\[E_{k,\max} = hf'_{\max} - W = \frac{2\sqrt{2}hf_0}{2\sqrt{2}-1} - \frac{4hf_0}{3}\]
\[= hf_0\left(\frac{2\sqrt{2}}{2\sqrt{2}-1} - \frac{4}{3}\right) = \frac{hc}{\lambda}\left(\frac{2\sqrt{2}}{2\sqrt{2}-1} - \frac{4}{3}\right)\]
\[= \frac{hc}{\lambda} \cdot \frac{6\sqrt{2} - 4(2\sqrt{2}-1)}{3(2\sqrt{2}-1)} = \frac{hc}{\lambda} \cdot \frac{6\sqrt{2} - 8\sqrt{2} + 4}{3(2\sqrt{2}-1)}\]
\[= \frac{hc}{\lambda} \cdot \frac{4 - 2\sqrt{2}}{3(2\sqrt{2}-1)} = \frac{hc}{\lambda} \cdot \frac{2(2-\sqrt{2})}{3(2\sqrt{2}-1)}\]

유리화: \(\frac{2(2-\sqrt{2})}{3(2\sqrt{2}-1)} \cdot \frac{2\sqrt{2}+1}{2\sqrt{2}+1} = \frac{2(2-\sqrt{2})(2\sqrt{2}+1)}{3(8-1)} = \frac{2(2-\sqrt{2})(2\sqrt{2}+1)}{21}\)

\((2-\sqrt{2})(2\sqrt{2}+1) = 4\sqrt{2} + 2 - 2\cdot 2 - \sqrt{2} = 4\sqrt{2} + 2 - 4 - \sqrt{2} = 3\sqrt{2} - 2\)

\[E_{k,\max} = \frac{hc}{\lambda} \cdot \frac{2(3\sqrt{2}-2)}{21}\]
\[\boxed{E_{k,\max} = \frac{2(3\sqrt{2}-2)hc}{21\lambda}}\]
