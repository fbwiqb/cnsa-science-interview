상자 밀기와 당기기의 비교 — 힘의 분해와 마찰력

주어진 조건:

- 상자의 질량: \(m\), 중력가속도: \(g\)

- 힘의 크기: \(F\), 바닥면과의 각도: \(\theta\) (미는 경우와 당기는 경우 동일)

(1) 마찰력이 없을 때 수평 방향 가속도 비교

미는 경우: 힘 \(F\)를 각도 \(\theta\)로 아래쪽 방향으로 가하므로, 수평 성분은 \(F\cos\theta\)이다.

\[ma_{\text{밀기}} = F\cos\theta \implies a_{\text{밀기}} = \frac{F\cos\theta}{m}\]

당기는 경우: 힘 \(F\)를 각도 \(\theta\)로 위쪽 방향으로 가하므로, 수평 성분은 역시 \(F\cos\theta\)이다.

\[ma_{\text{당기기}} = F\cos\theta \implies a_{\text{당기기}} = \frac{F\cos\theta}{m}\]

마찰력이 없을 때, 두 경우의 수평 방향 가속도는 같다.

\[\boxed{a_{\text{밀기}} = a_{\text{당기기}} = \frac{F\cos\theta}{m}}\]

(2) 수직 항력

미는 경우: 수직 방향 힘의 평형에서, 힘 \(F\)의 수직 성분이 아래로 작용하므로:

\[N_{\text{밀기}} = mg + F\sin\theta\]
\[\boxed{N_{\text{밀기}} = mg + F\sin\theta}\]

당기는 경우: 수직 방향 힘의 평형에서, 힘 \(F\)의 수직 성분이 위로 작용하므로:

\[N_{\text{당기기}} = mg - F\sin\theta\]
\[\boxed{N_{\text{당기기}} = mg - F\sin\theta}\]

(3) 마찰력이 있을 때 수평 방향 가속도 비교

운동 마찰력의 크기는 \(f = \mu N\)이므로:

미는 경우:

\[ma_{\text{밀기}} = F\cos\theta - \mu N_{\text{밀기}} = F\cos\theta - \mu(mg + F\sin\theta)\]
\[a_{\text{밀기}} = \frac{F\cos\theta - \mu(mg + F\sin\theta)}{m}\]

당기는 경우:

\[ma_{\text{당기기}} = F\cos\theta - \mu N_{\text{당기기}} = F\cos\theta - \mu(mg - F\sin\theta)\]
\[a_{\text{당기기}} = \frac{F\cos\theta - \mu(mg - F\sin\theta)}{m}\]

두 가속도의 차이:

\[a_{\text{당기기}} - a_{\text{밀기}} = \frac{2\mu F\sin\theta}{m} > 0\]

따라서 당기는 경우의 가속도가 미는 경우보다 크다.

\[\boxed{a_{\text{당기기}} > a_{\text{밀기}}}\]

이는 당기는 경우 수직 항력이 작아져서 마찰력도 작아지기 때문이다.
