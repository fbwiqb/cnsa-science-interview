[물리I-i] 평균 속도 문제

(가) 시간 절반씩 \(v_1\), \(v_2\)로 이동:

전체 시간 \(T\), 절반 시간 \(T/2\)씩

총 변위 \(= v_1(T/2) + v_2(T/2) = (v_1 + v_2)T/2\)

평균 속도 \(v_a = \text{총 변위}/T = (v_1 + v_2)/2\)

(나) 거리 절반씩 \(v_1\), \(v_2\)로 이동:

총 거리 \(d\), 절반 \(d/2\)씩

시간: \(t_1 = (d/2)/v_1\), \(t_2 = (d/2)/v_2\)

총 시간 \(= d/(2v_1) + d/(2v_2) = d(v_1 + v_2)/(2v_1 v_2)\)

평균 속도 \(v_b = d / \text{총시간} = 2v_1 v_2/(v_1 + v_2)\)

비교: \(v_a = (v_1 + v_2)/2\) (산술평균), \(v_b = 2v_1 v_2/(v_1 + v_2)\) (조화평균)

산술평균 \(\geq\) 조화평균 이므로 \(v_a \geq v_b\) (등호는 \(v_1 = v_2\)일 때)

(다) 정지 상태에서 시간 절반씩 가속도 \(a\), \(-a\):

시간 \(T/2\) 동안 가속도 \(a\): 최종 속도 \(= a(T/2)\), 변위 \(s_1 = (1/2)a(T/2)^2\)

다음 \(T/2\) 동안 가속도 \(-a\), 초기속도 \(aT/2\):

변위 \(s_2 = a(T/2)(T/2) + (1/2)(-a)(T/2)^2\)

총 변위 \(= s_1 + s_2 = (1/2)a_1 T^2/4 + a_1 T^2/4 + (1/2)a_2 T^2/4\)

\(= aT^2(3/8) + (-a)T^2/8 = T^2(3a_1 + a_2)/8\)

평균 속도 \(v_c = \text{총 변위}/T = T(3a_1 + a_2)/8\)

(라) 정지 상태에서 거리 절반씩 가속도 \(a_1\), \(a_2\):

처음 \(L/2\) 가속도 \(a_1\): \(v_1^2 = 2a_1(L/2) = a_1 L\), \(v_1 = \sqrt{a_1 L}\)

나머지 \(L/2\) 가속도 \(a_2\), 초기속도 \(v_1\):

\(v_f^2 = v_1^2 + 2a_2(L/2) = (a_1 + a_2)L\)

\(v_f = \sqrt{(a_1 + a_2)L}\)

총 시간: \(t_1 = v_1/a_1 = \sqrt{L/a_1}\), 처음 구간 시간

나머지 구간: \(t_2 = (v_f - v_1)/a_2\)

평균 속도 \(v_d = L/(t_1 + t_2)\)

[물리I-ii] 이상기체 방울의 깊이 변화

보일의 법칙: \(PV = \text{일정}\) (온도 일정)

초기: 수면으로부터 깊이 \(h_1 = 10\,\text{m}\), 부피 \(V_1 = 4\,\text{cm}^3\)

초기 압력: \(P_1 = \text{대기압} + \text{수압} = 1.0 \times 10^5 + \rho g h_1\)

\(= 1.0 \times 10^5 + 1000 \times 10 \times 10 = 1.0 \times 10^5 + 1.0 \times 10^5 = 2.0 \times 10^5\,\text{N/m}^2\) = 2기압

나중: 부피 \(V_2 = 5\,\text{cm}^3\)

\(P_1 V_1 = P_2 V_2\)에서

\(P_2 = P_1 V_1/V_2 = 2.0 \times 10^5 \times 4/5 = 1.6 \times 10^5\,\text{N/m}^2\)

나중 깊이: \(P_2 = \text{대기압} + \rho g h_2\)

\(1.6 \times 10^5 = 1.0 \times 10^5 + 1000 \times 10 \times h_2\)

\(0.6 \times 10^5 = 10000 \cdot h_2\)

\(h_2 = 6\,\text{m}\)

따라서 수면으로부터 6 m 깊이에 있다. (위로 4 m 올라감)
