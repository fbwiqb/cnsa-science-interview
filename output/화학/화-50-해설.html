해설 3

\(\text{Na}^+\)나 \(\text{F}^-\)는 수용액에서 산화, 환원 되기 어렵다. 그렇기 때문에 \(\text{H}_2\text{O}\)의 전기 분해 반응 일어나며, 이 때 반쪽 반응식은 아래와 같다.

산화 전극: \(2\text{H}_2\text{O(l)} \rightarrow \text{O}_2\text{(g)} + 4\text{H}^+\text{(aq)} + 4\text{e}^-\)

환원 전극: \(4\text{H}_2\text{O(l)} + 4\text{e}^- \rightarrow 2\text{H}_2\text{(g)} + 4\text{OH}^-\text{(aq)}\)

전체 반응: \(2\text{H}_2\text{O(l)} \rightarrow 2\text{H}_2\text{(g)} + \text{O}_2\text{(g)}\)

3,860초 동안 0.2 A의 전류를 흘려준 경우 전체 흘려준 전하량은 \(3{,}860 \times 0.2 = 772\) C이며, 이는 \(772/96{,}500 = 1/125 = 0.008\) F가 된다.

위에서 구한 물 분해 반응식을 보면 전자 4몰 당, 물 분자 2몰이 소모되고, 수소 기체 2몰과 산소 기체 1몰이 생성되므로, 전기 분해 반응 이후에 물은 \(0.008/4 \times 2 = 0.004\) 몰이 분해되었고, 수소 기체는 \(0.008/4 \times 2 = 0.004\) 몰, 산소 기체는 \(0.008/4 = 0.002\) 몰이 생성되었다.

이 때 오른쪽 수용액 상부 부분의 기체 압력 변화를 살펴보면 아래와 같다.

- 전기분해 반응 전: \(P_1 = 1\) 기압, \(V_1 = 48\) mL

- \(\rightarrow n_1 = P_1V_1/RT = (1 \times 0.048)/(0.08 \times 300) = 0.002\) 몰

- 전기분해 반응 후: 수소 0.004몰과 산소 0.002몰이 생성되므로

- \(V_2 \approx V_1 = 48\) mL, \(n_2 = n_1 + 0.006 = 0.008\) 몰

- \(\rightarrow P_2 = n_2RT/V_2 = (0.008 \times 0.08 \times 300)/(0.048) = 4\) 기압

최종적으로 양 쪽 유리관의 높이가 같으므로 대기압에서 추가된 압력만큼이 삼투압과 같아야 한다 (즉, \(\Pi = 4 - 1 = 3\) 기압). 반트 호프 법칙으로부터 최종 반응 후의 오른쪽 수용액의 용질 전체농도는 \(C = \Pi/RT = 3/(0.08 \times 300) = 1/8 = 0.125\) M이다.

삼투압은 용액의 총괄성을 가지므로, 용질의 개수에 의존한다. NaF가 모두 이온화되어 NaF 1몰당 용질 2몰을 생성한다고 가정하면,

NaF의 몰 농도 = \(0.125/2 = 0.0625\) (=1/16) M이다.
