리튬-공기 전지

2-1. 표준 전지 전위 계산

표준 전지 전위는 다음과 같이 계산된다:

\[E°_{\text{cell}} = E°_{\text{cathode}} - E°_{\text{anode}}\]

(-)극(산화극)의 반쪽 반응: \(\text{Li(s)} \rightarrow \text{Li}^+\text{(aq)} + \text{e}^-\), 이 반쪽의 표준 환원 전위는 \(E° = -3.05\) V이다.

반응 A만 일어나는 경우:

(+)극(환원극): \(\text{Li}^+\text{(aq)} + \text{e}^- + \frac{1}{2}\text{O}_2\text{(g)} \rightarrow \frac{1}{2}\text{Li}_2\text{O}_2\text{(s)}\), \(E° = -0.09\) V

\[E°_{\text{cell}} = E°_{(+)} - E°_{(-)} = (-0.09) - (-3.05) = 2.96 \text{ V}\]

반응 B만 일어나는 경우:

(+)극(환원극): \(\text{Li}^+\text{(aq)} + \text{e}^- + \frac{1}{4}\text{O}_2\text{(g)} \rightarrow \frac{1}{2}\text{Li}_2\text{O}\text{(s)}\), \(E° = -0.14\) V

\[E°_{\text{cell}} = E°_{(+)} - E°_{(-)} = (-0.14) - (-3.05) = 2.91 \text{ V}\]

2-2. 석출된 $\text{Li}_2\text{O}_2$와 $\text{Li}_2\text{O}$의 몰 수

원자량: Li = 7, C = 12, O = 16

분자량: \(\text{Li}_2\text{O}_2 = 2(7) + 2(16) = 46\) g/mol, \(\text{Li}_2\text{O} = 2(7) + 16 = 30\) g/mol

실험 결과 분석:

- 초기 (+)극 질량: 40 g (탄소 막대 + \(\text{Li}_2\text{O}_2\) + \(\text{Li}_2\text{O}\))

- 420 \(°\)C에서 반응 (1): \(\text{Li}_2\text{O}_2\text{(s)} \rightarrow \text{Li}_2\text{O(s)} + \frac{1}{2}\text{O}_2\text{(g)}\)

이 반응으로 \(\text{Li}_2\text{O}_2\)가 모두 분해되어 \(\text{Li}_2\text{O}\)로 전환된다. 질량 감소는 방출된 \(\text{O}_2\)의 질량에 해당한다.

420 \(°\)C 전후 질량 변화: \(40 - 32 = 8\) g 감소

\(\text{Li}_2\text{O}_2\) \(n\)몰이 분해되면 \(\frac{n}{2}\)몰의 \(\text{O}_2\)가 방출되므로:

\[\frac{n}{2} \times 32 = 8 \quad \Rightarrow \quad n = 0.5 \text{ mol}\]

따라서 \(\text{Li}_2\text{O}_2 = 0.5\) mol이었다.

- 800 \(°\)C에서 반응 (3): \(\text{C(s)} + \text{O}_2\text{(g)} \rightarrow \text{CO}_2\text{(g)}\)

800 \(°\)C에서는 탄소 막대가 공기 중의 산소와 반응하여 연소된다. 이때 \(\text{Li}_2\text{O}\)는 1100 \(°\)C에서야 녹으므로 800 \(°\)C에서는 변화가 없다.

800 \(°\)C 전후 질량 변화: \(32 - 18 = 14\) g 감소

이 14 g은 연소된 탄소의 질량이다. (기체인 \(\text{CO}_2\)가 빠져나가고 공기 중 \(\text{O}_2\)가 반응에 참여하므로 질량 감소는 탄소 질량에 해당)

그런데 최종적으로 18 g이 남았다. 이는 \(\text{Li}_2\text{O}\)의 질량이다.

420 \(°\)C 이후 존재하는 \(\text{Li}_2\text{O}\)는 두 가지 기원이 있다:

- 원래 석출된 \(\text{Li}_2\text{O}\): \(m\) mol

- \(\text{Li}_2\text{O}_2\)가 분해되어 생성된 \(\text{Li}_2\text{O}\): 0.5 mol

32 g에서 \(\text{Li}_2\text{O}\)와 탄소 막대가 있다.

\[32 = \text{Li}_2\text{O 질량} + \text{탄소 질량}\]

탄소 질량 = 14 g

\[\text{Li}_2\text{O 질량} = 32 - 14 = 18 \text{ g}\]

\(\text{Li}_2\text{O}\)의 총 몰 수 = \(\frac{18}{30} = 0.6\) mol

이 중 \(\text{Li}_2\text{O}_2\) 분해로 생성된 것이 0.5 mol이므로, 원래 석출된 \(\text{Li}_2\text{O}\)는:

\[0.6 - 0.5 = 0.1 \text{ mol}\]

따라서 석출된 \(\text{Li}_2\text{O}_2\)는 0.5 mol, \(\text{Li}_2\text{O}\)는 0.1 mol이다.

2-3. 산소 기체의 몰 수와 평균 전류

(-)극에서 리튬 금속 1.4 g이 소모되었다.

\[n_{\text{Li}} = \frac{1.4}{7} = 0.2 \text{ mol}\]

\(\text{Li}_2\text{O}_2\)와 \(\text{Li}_2\text{O}\)의 몰 수 비가 9:1이므로, \(\text{Li}_2\text{O}_2 = 9k\) mol, \(\text{Li}_2\text{O} = k\) mol로 놓자.

각 반응에서 소모되는 Li의 몰 수:

- 반응 A: \(\text{Li}_2\text{O}_2\) 1/2 mol 생성 시 \(\text{Li}^+\) 1 mol 필요 → \(\text{Li}_2\text{O}_2\) \(9k\) mol 생성 시 \(\text{Li}^+\) \(18k\) mol

- 반응 B: \(\text{Li}_2\text{O}\) 1/2 mol 생성 시 \(\text{Li}^+\) 1 mol 필요 → \(\text{Li}_2\text{O}\) \(k\) mol 생성 시 \(\text{Li}^+\) \(2k\) mol

총 Li 소모량:

\[18k + 2k = 20k = 0.2 \quad \Rightarrow \quad k = 0.01\]

따라서 \(\text{Li}_2\text{O}_2 = 0.09\) mol, \(\text{Li}_2\text{O} = 0.01\) mol이다.

(+)극에서 반응한 산소 기체의 몰 수:

- 반응 A에서: \(\text{Li}_2\text{O}_2\) 1/2 mol 생성 시 \(\text{O}_2\) 1/2 mol 필요 → \(0.09\) mol 생성 시 \(\text{O}_2 = 0.09\) mol

- 반응 B에서: \(\text{Li}_2\text{O}\) 1/2 mol 생성 시 \(\text{O}_2\) 1/4 mol 필요 → \(0.01\) mol 생성 시 \(\text{O}_2 = 0.005\) mol

총 산소 기체 몰 수:

\[n_{\text{O}_2} = 0.09 + 0.005 = 0.095 \text{ mol}\]

전하량 계산:

Li 1 mol이 산화되면 전자 1 mol이 이동하므로:

\[Q = n_{\text{Li}} \times F = 0.2 \times 96{,}000 = 19{,}200 \text{ C}\]

평균 전류:

방전 시간 = 800분 = \(800 \times 60 = 48{,}000\)초

\[I = \frac{Q}{t} = \frac{19{,}200}{48{,}000} = 0.4 \text{ A}\]

따라서 반응한 산소 기체는 0.095 mol, 평균 전류는 0.4 A이다.
