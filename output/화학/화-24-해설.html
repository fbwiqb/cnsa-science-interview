요소와 질소산화물의 반응

2-1. 엔탈피 변화 계산

전체 반응식:

\[4\text{NO} + 2(\text{NH}_2)_2\text{CO} + \text{O}_2 \rightarrow 4\text{N}_2 + 2\text{CO}_2 + 4\text{H}_2\text{O}\]

표준 반응 엔탈피는 생성물의 표준 생성 엔탈피 합에서 반응물의 표준 생성 엔탈피 합을 뺀 것이다. (\(\text{N}_2\)와 \(\text{O}_2\)는 홑원소 물질이므로 \(\Delta H_f = 0\))

\[\Delta H_{\text{rxn}} = [4(0) + 2(-350) + 4(-250)] - [4(+100) + 2(-350) + 0]\]
\[= [0 - 700 - 1000] - [400 - 700]\]
\[= -1700 - (-300) = -1400 \text{ kJ}\]

이것은 4 mol의 NO가 반응할 때의 엔탈피 변화이므로, 1 mol의 NO가 반응할 때:

\[\Delta H = \frac{-1400}{4} = -350 \text{ kJ/mol}\]

(문제에서 \(\text{H}_2\text{O(g)}\)의 생성 엔탈피 -250 kJ/mol을 사용했다.)

2-2. NO의 루이스 전자점식

NO 분자는 총 \(5 + 6 = 11\)개의 원자가 전자를 가지며, 홀수 전자 분자이다. 두 가지 주요 공명 구조가 가능하다.

구조 I: N-O 이중결합, 홀전자가 N에 위치

\[:\dot{\text{N}}=\text{O}:\]

- 결합 전자: 4개 (이중결합)

- N: 비공유 전자 3개 (비공유 전자쌍 1개 + 홀전자 1개)

- O: 비공유 전자 4개 (비공유 전자쌍 2개)

- 총: 4 + 3 + 4 = 11개

N 형식전하: \(5 - 3 - \frac{4}{2} = 0\), O 형식전하: \(6 - 4 - \frac{4}{2} = 0\)

구조 II: N-O 삼중결합, 홀전자가 N에 위치

- 결합 전자: 6개 (삼중결합)

- N: 비공유 전자 1개 (홀전자 1개)

- O: 비공유 전자 4개 (비공유 전자쌍 2개)

- 총: 6 + 1 + 4 = 11개

N 형식전하: \(5 - 1 - \frac{6}{2} = +1\), O 형식전하: \(6 - 4 - \frac{6}{2} = -1\)

가장 안정한 구조는 구조 I이다. 구조 I에서는 모든 원자의 형식전하가 0이므로 구조 II(형식전하 +1, -1)보다 안정하다. 실제 NO 분자는 이 두 구조의 공명 혼성체로, 결합 차수는 약 2.5이다. 홀수 전자 분자이므로 어느 구조에서도 모든 원자가 옥텟을 만족시킬 수 없다.

2-3. 산화수 변화와 전자 이동

(1) 산화되는 물질과 환원되는 물질:

반응: \(4\text{NO} + 2(\text{NH}_2)_2\text{CO} + \text{O}_2 \rightarrow 4\text{N}_2 + 2\text{CO}_2 + 4\text{H}_2\text{O}\)

각 물질의 산화수:

- NO에서 N: +2, O: -2

- 요소 \((\text{NH}_2)_2\text{CO}\)에서 N: -3, H: +1, C: +4 (확인: \(2(-3) + 4(+1) + (+4) + (-2) = 0\)... \(-6+4+4-2 = 0\)), O: -2

- \(\text{N}_2\)에서 N: 0

- \(\text{CO}_2\)에서 C: +4, O: -2

산화수 변화:

- NO의 N: +2 → 0 (감소, 환원) → NO가 환원되는 물질

- 요소의 N: -3 → 0 (증가, 산화) → 요소가 산화되는 물질

- C: +4 → +4 (변화 없음)

- O: -2 → -2 (변화 없음, \(\text{O}_2\)에서 O는 0 → -2로 환원)

따라서:

- 산화되는 물질: 요소 (N의 산화수: \(-3 \rightarrow 0\), +3 증가)

- 환원되는 물질: NO (N의 산화수: \(+2 \rightarrow 0\), -2 감소)와 \(\text{O}_2\) (O의 산화수: \(0 \rightarrow -2\), -2 감소)

(2) 전자 이동:

요소 1분자에 N이 2개 있으므로, 요소 1분자당 전자 \(2 \times 3 = 6\)개를 잃는다. 2분자의 요소에서 총 12개의 전자를 잃는다.

NO 1분자의 N이 전자 2개를 얻으므로, 4분자의 NO에서 총 \(4 \times 2 = 8\)개의 전자를 얻는다.

\(\text{O}_2\) 1분자에서 O 2개가 각각 전자 2개씩 얻으므로, 1분자당 4개의 전자를 얻는다.

총 얻는 전자: \(8 + 4 = 12\)개 = 총 잃는 전자 12개. 전자 수가 일치한다.

(3) 산화 환원 반응의 일반적인 특징:

산화 환원 반응에서는 한 물질이 전자를 잃으면(산화) 반드시 다른 물질이 그 전자를 얻는다(환원). 즉, 산화와 환원은 항상 동시에 일어나며, 잃은 전자의 총 수와 얻은 전자의 총 수는 항상 같다. 이것이 전하 보존 법칙의 반영이다.

2-4. 반응 엔탈피와 온도 변화

\[6\text{NO}_2 + 8\text{NH}_3 \rightarrow 7\text{N}_2 + 12\text{H}_2\text{O}\]

이 반응의 엔탈피 변화를 추론한다. 반응물에서 생성물로 갈 때 강한 결합(\(\text{N}\equiv\text{N}\) 삼중결합)이 많이 형성되고, 비교적 약한 결합(NO, NH)이 끊어진다. \(\text{N}_2\)의 삼중결합 에너지는 매우 크므로(약 945 kJ/mol), 생성물 쪽이 에너지적으로 매우 안정하다.

또한 생성 엔탈피를 이용하면 (\(\Delta H_f(\text{NO}_2) \approx +34\) kJ/mol, \(\Delta H_f(\text{NH}_3) = -46\) kJ/mol, \(\Delta H_f(\text{H}_2\text{O}) = -250\) kJ/mol):

\[\Delta H = [7(0) + 12(-250)] - [6(+34) + 8(-46)] = [-3000] - [204 - 368] = -3000 + 164 = -2836 \text{ kJ}\]

따라서 이 반응은 큰 발열 반응(\(\Delta H < 0\))이다. 반응이 일어나면 열이 방출되므로 주위의 온도가 상승한다.

2-5. 부피 감소 시 평형 이동

\[6\text{NO}_2\text{(g)} + 8\text{NH}_3\text{(g)} \rightleftharpoons 7\text{N}_2\text{(g)} + 12\text{H}_2\text{O(g)}\]

반응물 쪽 기체 몰 수: \(6 + 8 = 14\) mol

생성물 쪽 기체 몰 수: \(7 + 12 = 19\) mol

반응 용기의 부피가 줄어들면 압력이 증가한다. 르 샤틀리에 원리에 의해, 계는 압력 증가를 상쇄하는 방향, 즉 기체 몰 수가 감소하는 방향으로 평형이 이동한다.

기체 몰 수가 적은 쪽은 반응물 쪽(14 mol)이므로, 평형은 역반응 방향(왼쪽)으로 이동한다.
