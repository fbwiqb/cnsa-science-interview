요소 분해 반응의 열화학과 평형

1-1. 표준 반응 엔탈피 계산

\[\text{요소}(s) + \text{H}_2\text{O}(l) \rightarrow \text{CO}_2(g) + 2\text{NH}_3(g)\]
\[\Delta H_{\text{rxn}}^o = \sum \Delta H_f^o(\text{생성물}) - \sum \Delta H_f^o(\text{반응물})\]
\[= [\Delta H_f^o(\text{CO}_2) + 2\Delta H_f^o(\text{NH}_3)] - [\Delta H_f^o(\text{요소}) + \Delta H_f^o(\text{H}_2\text{O})]\]
\[= [(-394) + 2(-46)] - [(-334) + (-286)]\]
\[= [-394 - 92] - [-620]\]
\[= -486 + 620 = +134 \text{ kJ/mol}\]

따라서 표준 반응 엔탈피는 +134 kJ/mol (흡열 반응)이다.

1-2. 평균 결합 에너지를 이용한 반응 엔탈피

(1) 기체 상태 요소 분해의 반응 엔탈피:

\[\text{요소}(g) + \text{H}_2\text{O}(g) \rightarrow \text{CO}_2(g) + 2\text{NH}_3(g)\]

요소의 구조: \((\text{NH}_2)_2\text{CO}\)에서 결합은 C=O 1개, C-N 2개, N-H 4개이다.

끊어지는 결합(반응물):

- 요소: C=O (1개) + C-N (2개) + N-H (4개)

- \(\text{H}_2\text{O}\): O-H (2개)

끊어지는 결합 에너지 합:

\[E_{\text{break}} = 1(732) + 2(300) + 4(390) + 2(460) = 732 + 600 + 1560 + 920 = 3812 \text{ kJ}\]

형성되는 결합(생성물):

- \(\text{CO}_2\): C=O (2개)

- \(2\text{NH}_3\): N-H (6개)

형성되는 결합 에너지 합:

\[E_{\text{form}} = 2(732) + 6(390) = 1464 + 2340 = 3804 \text{ kJ}\]

반응 엔탈피:

\[\Delta H = E_{\text{break}} - E_{\text{form}} = 3812 - 3804 = +8 \text{ kJ}\]

(2) 1-1의 값과 다른 이유:

문항 1-1의 반응 엔탈피(+134 kJ)와 결합 에너지로 구한 값(+8 kJ)이 다른 이유는 두 가지이다:

첫째, 문항 1-1은 고체 상태 요소와 액체 상태 물의 반응인 반면, 문항 1-2는 기체 상태 요소와 기체 상태 물의 반응이다. 고체 → 기체(승화), 액체 → 기체(기화)에는 추가적인 에너지가 필요하므로, 요소의 승화열과 물의 기화열만큼 차이가 난다.

둘째, 평균 결합 에너지는 다양한 분자에서의 결합 에너지를 평균낸 값이므로 특정 분자에서의 실제 결합 에너지와 차이가 있다. 특히 요소의 C=O 결합은 공명 구조에 의해 일반적인 C=O 결합과 에너지가 다를 수 있다.

1-3. 평형 이동 분석

\[\text{요소}(g) + \text{H}_2\text{O}(g) \rightleftharpoons \text{CO}_2(g) + 2\text{NH}_3(g)\]

평형 상태에서 모두 1 M: \(K = \frac{[\text{CO}_2][\text{NH}_3]^2}{[\text{요소}][\text{H}_2\text{O}]} = \frac{(1)(1)^2}{(1)(1)} = 1\)

반응물 쪽 기체 몰 수: 2 mol, 생성물 쪽 기체 몰 수: 3 mol

(1) 용기의 부피를 두 배로 늘렸을 경우:

부피가 2배가 되면 모든 농도가 절반이 된다.

\[Q = \frac{(0.5)(0.5)^2}{(0.5)(0.5)} = \frac{0.125}{0.25} = 0.5 < K = 1\]

\(Q < K\)이므로 정반응 방향으로 진행된다. 따라서 요소의 몰 수는 감소하고, $\text{CO}_2$의 몰 수는 증가한다.

또한 기체 몰 수가 증가하는 방향(정반응, 2→3)으로 평형이 이동한다는 르 샤틀리에 원리와도 일치한다.

(2) 1 몰의 $\text{NH}_3$ 기체를 첨가할 경우:

\([\text{NH}_3]\)가 증가하면 \(Q > K\)가 되므로 역반응 방향으로 진행된다. 따라서 요소의 몰 수는 증가하고, $\text{CO}_2$의 몰 수는 감소한다.

(3) 1 몰의 HCl 기체를 첨가할 경우:

HCl은 \(\text{NH}_3\)와 반응한다:

\[\text{NH}_3(g) + \text{HCl}(g) \rightarrow \text{NH}_4\text{Cl}(s)\]

\(\text{NH}_3\)가 소모되어 \([\text{NH}_3]\)가 감소한다. \(Q < K\)가 되므로 정반응 방향으로 진행된다. 따라서 요소의 몰 수는 감소하고, $\text{CO}_2$의 몰 수는 증가한다.

(4) 0.1 몰의 $\text{H}_2\text{O}$와 0.1 몰의 $\text{CO}_2$를 동시에 제거할 경우:

제거 직후: \([\text{H}_2\text{O}] = 0.9\) M, \([\text{CO}_2] = 0.9\) M, 나머지는 변화 없다.

\[Q = \frac{(0.9)(1)^2}{(1)(0.9)} = 1 = K\]

\(Q = K\)이므로 평형 상태가 유지된다. 요소와 $\text{CO}_2$의 몰 수는 변화 없다 (제거 직후의 상태가 새로운 평형).

(5) 0.1 몰의 $\text{H}_2\text{O}$와 0.1 몰의 $\text{NH}_3$를 동시에 제거할 경우:

제거 직후: \([\text{H}_2\text{O}] = 0.9\) M, \([\text{NH}_3] = 0.9\) M, 나머지는 변화 없다.

\[Q = \frac{(1)(0.9)^2}{(1)(0.9)} = \frac{0.81}{0.9} = 0.9 < K = 1\]

\(Q < K\)이므로 정반응 방향으로 진행된다. 따라서 요소의 몰 수는 감소하고, $\text{CO}_2$의 몰 수는 증가한다.
