수소화물의 산화수, 루이스 전자점식, 결합각

문제 1. \(\text{C}_2\), \(\text{N}_2\), \(\text{O}_2\)와 \(\text{H}_2\)의 반응 (C, N, O의 산화수 \(= -1\))

각 원소의 산화수가 \(-1\)이 되는 수소화물을 구한다.

탄소 (산화수 $-1$):

C의 산화수가 \(-1\)이고 H의 산화수가 \(+1\)인 수소화물:

산화수 합 = 0이므로, \(\text{C}_x\text{H}_y\)에서 \(-x + y = 0\), 즉 \(x = y\)

가장 간단한 것: \(\text{C}_2\text{H}_2\) (아세틸렌, 에타인)

\(\text{C}_2\text{H}_2\)에서 C의 산화수: 각 C는 C와 삼중결합(동일 원소, 기여 0)과 H와 단일결합(C > H이므로 기여 \(-1\))

산화수 = \(-1\) ✓

반응식:

\[\text{C}_2 + \text{H}_2 \rightarrow \text{C}_2\text{H}_2\]

(계수 맞춤: 이미 균형)

루이스 전자점식: \(\text{H-C} \equiv \text{C-H}\)

각 C는 삼중결합 1개 + 단일결합(H) 1개, 비공유 전자쌍 없음

질소 (산화수 $-1$):

N의 산화수가 \(-1\)인 수소화물:

\(\text{N}_x\text{H}_y\)에서 \(-x + y = 0\), 즉 \(x = y\)

가장 간단한 것: \(\text{N}_2\text{H}_2\) (다이아젠, \(\text{HN=NH}\))

\(\text{N}_2\text{H}_2\)에서 N의 산화수: N-N 이중결합(동일 원소, 기여 0) + N-H 단일결합(N > H이므로 기여 \(-1\))

산화수 = \(-1\) ✓

반응식:

\[\text{N}_2 + \text{H}_2 \rightarrow \text{N}_2\text{H}_2\]

루이스 전자점식: 각 N은 이중결합(N=N) 1개 + 단일결합(N-H) 1개 + 비공유 전자쌍 1개

산소 (산화수 $-1$):

O의 산화수가 \(-1\)인 수소화물:

\(\text{O}_x\text{H}_y\)에서 \(-x + y = 0\), 즉 \(x = y\)

가장 간단한 것: \(\text{H}_2\text{O}_2\) (과산화수소)

\(\text{H}_2\text{O}_2\)에서 O의 산화수: O-O 단일결합(동일 원소, 기여 0) + O-H 단일결합(O > H이므로 기여 \(-1\))

산화수 = \(-1\) ✓

반응식:

\[\text{O}_2 + \text{H}_2 \rightarrow \text{H}_2\text{O}_2\]

루이스 전자점식: \(\text{H-O-O-H}\), 각 O는 단일결합 2개(O-O, O-H) + 비공유 전자쌍 2개

결합각의 상대적 크기 비교 (전자쌍 반발 원리):

\(\text{C}_2\text{H}_2\): 각 C 주위에 2개의 전자 영역(삼중결합 + 단일결합) → \(sp\) 혼성 → 직선형, 결합각 = \(180°\)

\(\text{N}_2\text{H}_2\): 각 N 주위에 3개의 전자 영역(이중결합 + 단일결합 + 비공유 전자쌍 1개) → \(sp^2\) 혼성 → 굽은형, 결합각 ≈ \(120°\)보다 약간 작음 (비공유 전자쌍의 반발)

\(\text{H}_2\text{O}_2\): 각 O 주위에 4개의 전자 영역(단일결합 2개 + 비공유 전자쌍 2개) → \(sp^3\) 혼성 → H-O-O 결합각 ≈ \(109.5°\)보다 작음 (비공유 전자쌍 2개의 강한 반발), 실제 약 \(94.8°\)

결합각 크기 순서:

\[\boxed{\text{C}_2\text{H}_2\ (180°) > \text{N}_2\text{H}_2\ (\sim 120°) > \text{H}_2\text{O}_2\ (\sim 94.8°)}\]

이유: 중심 원자 주위의 비공유 전자쌍 수가 증가할수록(0 → 1 → 2) 공유 전자쌍을 더 강하게 밀어내어 결합각이 감소한다.

문제 2. \(\text{C}_2\), \(\text{N}_2\), \(\text{O}_2\)와 \(\text{H}_2\)의 반응 (C, N, O의 산화수 \(= -2\))

탄소 (산화수 $-2$):

C의 산화수가 \(-2\)인 수소화물: \(\text{C}_x\text{H}_y\)에서 \(-2x + y = 0\), \(y = 2x\)

가장 간단한 것: \(\text{CH}_2\) → 불안정. 안정한 것: \(\text{C}_2\text{H}_4\) (에틸렌)

\(\text{C}_2\text{H}_4\)에서 C의 산화수: C=C 이중결합(동일 원소, 기여 0) + C-H 2개(각 \(-1\))

산화수 = \(-2\) ✓

반응식:

\[\text{C}_2 + 2\text{H}_2 \rightarrow \text{C}_2\text{H}_4\]

루이스 전자점식: \(\text{H}_2\text{C=CH}_2\), 각 C는 이중결합 1개 + C-H 결합 2개, 비공유 전자쌍 없음

질소 (산화수 $-2$):

N의 산화수가 \(-2\)인 수소화물: \(\text{N}_x\text{H}_y\)에서 \(-2x + y = 0\), \(y = 2x\)

가장 간단한 것: \(\text{N}_2\text{H}_4\) (하이드라진)

\(\text{N}_2\text{H}_4\)에서 N의 산화수: N-N 단일결합(동일 원소, 기여 0) + N-H 2개(각 \(-1\))

산화수 = \(-2\) ✓

반응식:

\[\text{N}_2 + 2\text{H}_2 \rightarrow \text{N}_2\text{H}_4\]

루이스 전자점식: \(\text{H}_2\text{N-NH}_2\), 각 N은 단일결합 3개(N-N, N-H 2개) + 비공유 전자쌍 1개

산소 (산화수 $-2$):

O의 산화수가 \(-2\)인 수소화물: \(\text{H}_2\text{O}\) (물)

\(\text{H}_2\text{O}\)에서 O의 산화수: O-H 2개(각 \(-1\)) → \(-2\) ✓

반응식:

\[\text{O}_2 + 2\text{H}_2 \rightarrow 2\text{H}_2\text{O}\]

루이스 전자점식: \(\text{H-O-H}\), O는 단일결합 2개 + 비공유 전자쌍 2개

브뢴스테드-로우리 염기로 작용할 수 있는 수소화물:

브뢴스테드-로우리 염기는 \(\text{H}^+\)을 받을 수 있는 물질, 즉 비공유 전자쌍을 가진 물질이다.

- \(\text{C}_2\text{H}_4\): 탄소에 비공유 전자쌍이 없다. \(\pi\) 결합의 전자를 이용해 약한 염기로 작용할 수는 있으나, 전형적인 브뢴스테드 염기는 아니다.

- \(\text{N}_2\text{H}_4\): 각 질소에 비공유 전자쌍이 1개씩 있다. \(\text{H}^+\)을 받아 \(\text{N}_2\text{H}_5^+\)를 형성할 수 있으므로 브뢴스테드-로우리 염기로 작용한다.

- \(\text{H}_2\text{O}\): 산소에 비공유 전자쌍이 2개 있다. \(\text{H}^+\)을 받아 \(\text{H}_3\text{O}^+\)를 형성할 수 있으므로 브뢴스테드-로우리 염기로 작용한다.

\[\boxed{\text{N}_2\text{H}_4\text{와}\ \text{H}_2\text{O}}\]
