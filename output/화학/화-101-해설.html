해설 2

[화학I-i] 산화제/환원제 판별 및 원소 X 결정

A = \(\text{X}_2\) (Mg이나 Zn이 묽은 염산과 반응하여 생성)

Mg + 2HCl \(\rightarrow\) \(\text{MgCl}_2\) + \(\text{H}_2\)(g) \(\rightarrow\) A = \(\text{H}_2\), X = H

(1) 2Na + \(\text{H}_2\) \(\rightarrow\) 2NaH: H의 산화수 0 \(\rightarrow\) -1 (환원), \(\text{H}_2\)는 산화제

(2) CuO + \(\text{H}_2\) \(\rightarrow\) Cu + \(\text{H}_2\)O: H의 산화수 0 \(\rightarrow\) +1 (산화), \(\text{H}_2\)는 환원제

(3) \(\text{CH}_2\) + \(\text{H}_2\) \(\rightarrow\) \(\text{CH}_4\) (아세틸렌+수소?): 확인. \(\text{C}_2\text{H}_2\) + \(\text{H}_2\) \(\rightarrow\) \(\text{C}_2\text{H}_4\)

H의 산화수 0 \(\rightarrow\) +1/-1 depending on context

[화학I-ii] 원소 Y 결정

B = \(\text{Y}_2\), Y는 2주기 원소. XCY = HCY 분자에서 Y와 C가 옥텟 충족

HCN: H-C\(\equiv\)N (C와 N 모두 옥텟 충족) \(\rightarrow\) Y = N (질소)

\(\text{Y}_2\) = \(\text{N}_2\)

\(\text{YF}_2\) = \(\text{NF}_2\)... 하지만 \(\text{NF}_3\)가 안정적. \(\text{N}_2\text{F}_2\)라면 F-N=N-F

또한 화합물 \(\text{Y}_2\text{F}_2\)의 루이스 전자점식을 그리시오. (단, Y는 임의의 원소이고, X는 [화학I-i]의 원소 X이다.)

[화학I-iii] Z 원소와 결합각

D = \(\text{YX}_3\) = \(\text{NH}_3\)

\(\text{ZCl}_3\): Z는 2주기 원소, \(\text{BCl}_3\) (붕소)

\(\text{BCl}_3\) + \(\text{NH}_3\) \(\rightarrow\) \(\text{Cl}_3\text{B-NH}_3\)

반응 전 \(\text{BCl}_3\): 평면삼각형(120\(\degree\)), 반응 후 사면체형(약 109.5\(\degree\)) \(\rightarrow\) 감소

반응 전 \(\text{NH}_3\): 삼각뿔형(약 107\(\degree\)), 반응 후 사면체에 가까워짐(약 109.5\(\degree\)) \(\rightarrow\) 증가

[화학I-iv] 반응 후 피스톤 위치

A = \(\text{H}_2\), B = \(\text{N}_2\), D = \(\text{NH}_3\)

\(3\text{H}_2 + \text{N}_2 \rightarrow 2\text{NH}_3\)

a=3, b=1, d=2

반응 전: \(\text{H}_2\)와 \(\text{N}_2\)의 비율과 부피로부터 몰수 계산하여 피스톤 위치 결정

(그림 정보 없이 정확한 계산 불가하나, 기체 몰수 변화에 따른 부피 변화 계산)

반응 전 총 기체 4몰 \(\rightarrow\) 반응 후 2몰 (절반으로 감소)

온도와 압력 일정 시 부피도 절반

[화학I-v] 아미노산의 양쪽성

E = 아미노산 (\(\text{NH}_3\)와 관련 물질에서 생성)

아미노산은 양쪽성 이온(쯔비터 이온)

- 산성 용액: \(-\text{NH}_3^+\), \(-\text{COOH}\) \(\rightarrow\) 양이온 \(\rightarrow\) (-)극으로 이동

- 염기성 용액: \(-\text{NH}_2\), \(-\text{COO}^-\) \(\rightarrow\) 음이온 \(\rightarrow\) (+)극으로 이동

- 중성 용액(등전점): 쯔비터 이온(\(-\text{NH}_3^+\), \(-\text{COO}^-\)) \(\rightarrow\) 알짜 전하 0 \(\rightarrow\) 이동하지 않음
