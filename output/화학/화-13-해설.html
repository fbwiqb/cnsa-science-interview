해설 1

[3번 - 화학] $\text{Fe}_2\text{O}_3$의 환원 반응

반응: \(\text{Fe}_2\text{O}_3\) + 순수한 물질(A 또는 B) \(\rightarrow\) Fe + \(\text{CO}_2\)

[3-i] 순수한 물질의 산화/환원:

\(\text{Fe}_2\text{O}_3\)에서 Fe의 산화수: +3

생성된 Fe에서 산화수: 0

Fe의 산화수가 +3 \(\rightarrow\) 0으로 감소 \(\rightarrow\) \(\text{Fe}_2\text{O}_3\)의 Fe는 환원됨

산화-환원 반응에서 산화와 환원은 동시에 일어나므로, 순수한 물질은 산화된다. (탄소의 산화수가 증가하여 \(\text{CO}_2\)가 됨)

[3-ii] A(탄소 산화수 0)와 B(탄소 산화수 +2) 비교:

물질 A: 탄소 산화수 \(0 \rightarrow\) C (탄소, 원소 상태)

물질 B: 탄소 산화수 \(+2 \rightarrow\) CO (일산화탄소)

A를 사용한 반응: \(2\text{Fe}_2\text{O}_3 + 3\text{C} \rightarrow 4\text{Fe} + 3\text{CO}_2\)

C의 산화수 변화: \(0 \rightarrow +4\) (전자 4개 잃음)

탄소 1몰당 전자 4몰 이동, \(\text{CO}_2\) 1몰 생성 시 Fe 4/3몰 생성

B를 사용한 반응: \(\text{Fe}_2\text{O}_3 + 3\text{CO} \rightarrow 2\text{Fe} + 3\text{CO}_2\)

CO의 산화수 변화: \(+2 \rightarrow +4\) (전자 2개 잃음)

CO 1몰당 전자 2몰 이동, \(\text{CO}_2\) 1몰 생성 시 Fe 2/3몰 생성

동일한 양의 \(\text{CO}_2\)를 발생시킬 때:

A(탄소) 사용: \(\text{CO}_2\) 1몰당 Fe = 4/3몰

B(CO) 사용: \(\text{CO}_2\) 1몰당 Fe = 2/3몰

물질 A를 사용할 때 Fe가 2배 더 많이 생성된다.

[3-iii] B(CO)를 사용하여 Fe 55.85 g 생성 시 압력:

Fe 55.85 g = 1 mol (원자량 55.85)

반응식: \(\text{Fe}_2\text{O}_3 + 3\text{CO} \rightarrow 2\text{Fe} + 3\text{CO}_2\)

Fe 2몰당 \(\text{CO}_2\) 3몰 생성 \(\rightarrow\) Fe 1몰당 \(\text{CO}_2\) 1.5몰

0°C, 1기압에서 22.4 L에 이미 공기 1몰 존재

\(\text{CO}_2\) 1.5몰을 추가 포집

용기 부피 고정(22.4 L), 온도 0°C 일정

총 기체 = 1(공기) + 1.5(\(\text{CO}_2\)) = 2.5 mol

이상기체 법칙: \(P = nRT/V\)

처음: 1기압 = \(1 \times RT/22.4\)

나중: \(P = 2.5 \times RT/22.4 = 2.5\)기압

최종 압력 = 2.5기압
