메탄올 산화 반응의 열화학

1-1. 반응 A와 B의 화학 반응식

반응 A는 메탄올이 완전 연소하여 \(\text{CO}_2\)와 \(\text{H}_2\text{O}\)를 생성하는 반응이다.

\[\text{CH}_3\text{OH(g)} + \frac{3}{2}\text{O}_2\text{(g)} \rightarrow \text{CO}_2\text{(g)} + 2\text{H}_2\text{O(g)}\]

반응 B는 메탄올이 부분 산화되어 폼산(HCOOH)과 \(\text{H}_2\text{O}\)를 생성하는 반응이다. 탄소의 산화수 변화와 수소-산소 균형을 고려하면:

\[\text{CH}_3\text{OH(g)} + \text{O}_2\text{(g)} \rightarrow \text{HCOOH(g)} + \text{H}_2\text{O(g)}\]

1-2. 메탄올과 폼산의 표준 생성 엔탈피

반응 A의 반응 엔탈피는 생성물의 표준 생성 엔탈피 합에서 반응물의 표준 생성 엔탈피 합을 빼서 구할 수 있다.

\[\Delta H_A = \Delta H_f^\circ(\text{CO}_2) + 2\Delta H_f^\circ(\text{H}_2\text{O}) - \Delta H_f^\circ(\text{CH}_3\text{OH}) - \frac{3}{2}\Delta H_f^\circ(\text{O}_2)\]

\(\text{O}_2\)는 홑원소 물질이므로 \(\Delta H_f^\circ = 0\)이다. 대입하면:

\[-670 = (-390) + 2(-240) - \Delta H_f^\circ(\text{CH}_3\text{OH})\]
\[-670 = -870 - \Delta H_f^\circ(\text{CH}_3\text{OH})\]
\[\Delta H_f^\circ(\text{CH}_3\text{OH}) = -870 + 670 = -200 \text{ kJ/mol}\]

반응 B에 대해:

\[\Delta H_B = \Delta H_f^\circ(\text{HCOOH}) + \Delta H_f^\circ(\text{H}_2\text{O}) - \Delta H_f^\circ(\text{CH}_3\text{OH}) - \Delta H_f^\circ(\text{O}_2)\]
\[-420 = \Delta H_f^\circ(\text{HCOOH}) + (-240) - (-200) - 0\]
\[-420 = \Delta H_f^\circ(\text{HCOOH}) - 40\]
\[\Delta H_f^\circ(\text{HCOOH}) = -380 \text{ kJ/mol}\]

따라서 메탄올의 표준 생성 엔탈피는 \(-200\) kJ/mol, 폼산의 표준 생성 엔탈피는 \(-380\) kJ/mol이다.

1-3. 수용액의 pH 계산

반응 B: \(\text{CH}_3\text{OH(g)} + \text{O}_2\text{(g)} \rightarrow \text{HCOOH(g)} + \text{H}_2\text{O(g)}\)

1몰의 메탄올이 반응하면 1몰의 폼산(HCOOH)과 1몰의 물이 생성된다.

생성된 폼산 1몰을 물에 녹여 총 8 L의 수용액을 만들었으므로, 폼산의 초기 농도는:

\[C = \frac{1 \text{ mol}}{8 \text{ L}} = 0.125 \text{ M} = \frac{1}{8} \text{ M}\]

폼산은 약산이므로 이온화 평형을 고려한다.

\[\text{HCOOH} \rightleftharpoons \text{HCOO}^- + \text{H}^+\]
\[K_a = \frac{[\text{H}^+][\text{HCOO}^-]}{[\text{HCOOH}]} = 2 \times 10^{-4}\]

\([\text{H}^+] = x\)라 하면:

\[2 \times 10^{-4} = \frac{x^2}{0.125 - x}\]

\(x \ll 0.125\)라 가정하면:

\[x^2 = 2 \times 10^{-4} \times 0.125 = 2.5 \times 10^{-5}\]
\[x = \sqrt{2.5 \times 10^{-5}} = 5 \times 10^{-3} \text{ M}\]

검증: \(x/C = 5 \times 10^{-3}/0.125 = 0.04 = 4\%\) (5% 미만이므로 근사 유효)

\[\text{pH} = -\log(5 \times 10^{-3}) = -\log 5 + 3 = 3 - 0.7 = 2.3\]

따라서 수용액의 pH는 2.3이다.

1-4. 탄소 원자의 산화수

각 분자에서 탄소의 산화수를 구한다. 수소의 산화수는 +1, 산소의 산화수는 -2로 설정한다.

- 메탄올($\text{CH}_3\text{OH}$): \(x + 3(+1) + (-2) + (+1) = 0\)이므로 \(x + 2 = 0\), \(x = -2\)

- 폼산($\text{HCOOH}$): \(x + (+1) + 2(-2) + (+1) = 0\)이므로 \(x - 2 = 0\), \(x = +2\)

- 이산화 탄소($\text{CO}_2$): \(x + 2(-2) = 0\)이므로 \(x = +4\)

따라서 메탄올의 탄소 산화수는 \(-2\), 폼산의 탄소 산화수는 \(+2\), 이산화 탄소의 탄소 산화수는 \(+4\)이다.

1-5. 이산화 탄소의 환원 반응 중 발열 반응

이산화 탄소(\(\text{CO}_2\))를 환원하여 탄소 원자 1개를 포함하는 생성물을 만드는 반응을 고려한다. 환원제로 \(\text{H}_2\), \(\text{H}_2\text{O}\), \(\text{H}_2\text{O}_2\)만 사용한다.

표에서 탄소 원자 1개를 포함하는 화합물: CO, HCHO, \(\text{CH}_4\), \(\text{CH}_3\text{OH}\)

각 가능한 반응의 반응 엔탈피를 계산한다.

반응 1: $\text{CO}_2 + \text{H}_2 \rightarrow \text{CO} + \text{H}_2\text{O}$

\[\Delta H = (-110) + (-240) - (-390) - 0 = +40 \text{ kJ} \quad (\text{흡열})\]

반응 2: $\text{CO}_2 + 2\text{H}_2 \rightarrow \text{HCHO} + \text{H}_2\text{O}$

\[\Delta H = (-110) + (-240) - (-390) - 0 = +40 \text{ kJ} \quad (\text{흡열})\]

반응 3: $\text{CO}_2 + 4\text{H}_2 \rightarrow \text{CH}_4 + 2\text{H}_2\text{O}$

\[\Delta H = (-70) + 2(-240) - (-390) - 0 = -160 \text{ kJ} \quad (\text{발열})\]

\(\text{H}_2\text{O}_2\)를 환원제로 사용하는 경우도 검토한다.

반응 4: $\text{CO}_2 + \text{H}_2\text{O}_2 \rightarrow \text{HCHO} + \frac{3}{2}\text{O}_2$

원자 균형 확인: C: 1=1, H: 2=2, O: 2+2=1+3 (4=4). 균형 맞음. 그러나 이 반응에서 \(\text{H}_2\text{O}_2\)는 환원제가 아니라 산소를 내놓는 역할을 한다. 탄소의 산화수가 +4에서 0으로 감소하므로 \(\text{CO}_2\)가 환원되지만, \(\text{H}_2\text{O}_2\)의 산소 산화수는 -1에서 0으로 증가하므로 \(\text{H}_2\text{O}_2\)가 산화(환원제 역할)된다.

\[\Delta H = (-110) + 0 - (-390) - (-140) = +420 \text{ kJ} \quad (\text{흡열})\]

반응 5: $\text{CO}_2 + 2\text{H}_2\text{O}_2 \rightarrow \text{CH}_4 + 3\text{O}_2$

원자 균형: C: 1=1, H: 4=4, O: 2+4=6=6. 그러나 이 반응에서 탄소 산화수 +4 → -4, 산소 산화수 -1 → 0.

\[\Delta H = (-70) + 0 - (-390) - 2(-140) = +600 \text{ kJ} \quad (\text{흡열})\]

\(\text{H}_2\text{O}\)를 환원제로 사용하는 경우: \(\text{H}_2\text{O}\)에서 산소의 산화수는 -2이므로 이미 가장 안정한 상태이다. \(\text{H}_2\text{O}\)를 환원제로 사용하면 산소가 산화되어야 하므로, 예를 들어 \(\text{CO}_2 + 2\text{H}_2\text{O} \rightarrow \text{CH}_4 + 2\text{O}_2\)와 같은 반응이 가능하나:

\[\Delta H = (-70) + 0 - (-390) - 2(-240) = +800 \text{ kJ} \quad (\text{흡열})\]

모든 \(\text{H}_2\text{O}\) 환원제 반응은 강한 흡열 반응이다.

또한 메탄올도 탄소 원자 1개를 포함하는 화합물이므로 검토한다(표준 생성 엔탈피는 문제 1-2에서 \(-200\) kJ/mol로 구함):

반응 6: $\text{CO}_2 + 3\text{H}_2 \rightarrow \text{CH}_3\text{OH} + \text{H}_2\text{O}$

\[\Delta H = (-200) + (-240) - (-390) - 0 = -50 \text{ kJ} \quad (\text{발열})\]

따라서 발열 반응은 다음 두 가지이다:

\[\text{CO}_2\text{(g)} + 3\text{H}_2\text{(g)} \rightarrow \text{CH}_3\text{OH(g)} + \text{H}_2\text{O(g)} \qquad \Delta H = -50 \text{ kJ}\]
\[\text{CO}_2\text{(g)} + 4\text{H}_2\text{(g)} \rightarrow \text{CH}_4\text{(g)} + 2\text{H}_2\text{O(g)} \qquad \Delta H = -160 \text{ kJ}\]
