기체의 성질과 전기분해

1-1. 기체의 특성 비교

[그림 1]에서 a~d는 \(\text{H}_2\) 기체, e~h는 \(\text{O}_2\) 기체가 각 용기에 들어있다. 큰 용기는 작은 용기 부피의 2배이다. 기체 분자 수와 용기 크기를 기반으로 분석한다.

그림을 기반으로 각 용기의 분자 수와 용기 크기를 파악한다:

- a: \(\text{H}_2\) 2분자, 작은 용기

- b: \(\text{H}_2\) 2분자, 큰 용기

- c: \(\text{H}_2\) 4분자, 작은 용기

- d: \(\text{H}_2\) 4분자, 큰 용기

- e: \(\text{O}_2\) 2분자, 작은 용기

- f: \(\text{O}_2\) 2분자, 큰 용기

- g: \(\text{O}_2\) 4분자, 작은 용기

- h: \(\text{O}_2\) 4분자, 큰 용기

(1) 압력 (\(P = \frac{nRT}{V}\), 모든 용기가 같은 온도):

압력은 \(n/V\)에 비례한다. 같은 분자 수에 대해 작은 용기가 큰 용기의 2배 압력을 가진다.

- \(n/V\) 비율: a = 2/1V, b = 2/2V = 1/V, c = 4/1V, d = 4/2V = 2/V, e = 2/1V, f = 2/2V = 1/V, g = 4/1V, h = 4/2V = 2/V

같은 \(n/V\)를 가지는 것끼리 같은 압력: b = f < a = d = e = h < c = g

\[b = f < a = d = e = h < c = g\]

(2) 밀도 (\(\rho = \frac{m}{V} = \frac{nM}{V}\)):

밀도는 \(nM/V\)에 비례한다. \(\text{H}_2\)의 분자량 = 2, \(\text{O}_2\)의 분자량 = 32이다.

- a: \(2 \times 2 / V = 4/V\)

- b: \(2 \times 2 / 2V = 2/V\)

- c: \(4 \times 2 / V = 8/V\)

- d: \(4 \times 2 / 2V = 4/V\)

- e: \(2 \times 32 / V = 64/V\)

- f: \(2 \times 32 / 2V = 32/V\)

- g: \(4 \times 32 / V = 128/V\)

- h: \(4 \times 32 / 2V = 64/V\)

\[b < a = d < c < f < e = h < g\]

(3) 분자당 평균 속력:

같은 온도에서 기체 분자의 평균 속력은 분자량에만 의존한다: \(v \propto \sqrt{\frac{T}{M}}\). 온도가 동일하므로 분자량만 고려한다.

- \(\text{H}_2\) (M = 2): a, b, c, d 모두 같은 평균 속력

- \(\text{O}_2\) (M = 32): e, f, g, h 모두 같은 평균 속력

\(\text{H}_2\)의 평균 속력이 \(\text{O}_2\)보다 \(\sqrt{32/2} = 4\)배 빠르다.

\[e = f = g = h < a = b = c = d\]

1-2. 수소와 산소의 반응

(1) 반응 후 용기 내부 압력:

\[2\text{H}_2\text{(g)} + \text{O}_2\text{(g)} \rightarrow 2\text{H}_2\text{O(g)}\]

\(PV = nRT\)에서 \(T = 227°\text{C} = 500\) K, \(V = 20.0\) L

\(\text{H}_2\)의 몰 수: \(n_{\text{H}_2} = \frac{PV}{RT} = \frac{2.00 \times 20.0}{0.08 \times 500} = 1.00\) mol

\(\text{O}_2\)의 몰 수: \(n_{\text{O}_2} = \frac{3.00 \times 20.0}{0.08 \times 500} = 1.50\) mol

반응 비 \(\text{H}_2\):\(\text{O}_2\) = 2:1이므로, \(\text{H}_2\) 1.00 mol은 \(\text{O}_2\) 0.50 mol과 반응한다.

\(\text{H}_2\)가 한계 반응물이다.

반응 후:

- \(\text{H}_2\): 0 mol

- \(\text{O}_2\): \(1.50 - 0.50 = 1.00\) mol

- \(\text{H}_2\text{O}\): 1.00 mol

총 기체 몰 수 = 2.00 mol (227 \(°\)C에서 물은 수증기)

\[P = \frac{nRT}{V} = \frac{2.00 \times 0.08 \times 500}{20.0} = 4.00 \text{ atm}\]

(2) 0 $°$C로 온도를 내렸을 때 생성되는 얼음의 질량:

0 \(°\)C에서 물은 고체(얼음) 상태가 된다. 생성된 \(\text{H}_2\text{O}\) 1.00 mol이 모두 얼음이 된다.

\[m = 1.00 \times 18 = 18.0 \text{ g}\]

(3) 상온에서 반응하지 않는 이유:

수소와 산소의 반응은 열역학적으로 자발적이지만(\(\Delta G < 0\)), 반응의 활성화 에너지(\(E_a\))가 높기 때문이다. 상온에서 기체 분자들의 평균 운동 에너지가 이 활성화 에너지 장벽을 넘기에 충분하지 않아 반응이 일어나지 않는다. 점화(불꽃, 촉매 등)에 의해 활성화 에너지를 공급하면 반응이 시작되고 발열 반응의 에너지로 연쇄적으로 진행된다.

1-3. 물의 전기분해

(1) (+)극과 (-)극에서의 pH 변화:

(+)극(양극, 산화 반응):

\[2\text{H}_2\text{O}(l) \rightarrow \text{O}_2(g) + 4\text{H}^+(aq) + 4e^-\]

\(\text{H}^+\)이 생성되므로 (+)극 인접 부위의 pH는 감소(산성화)한다.

(-)극(음극, 환원 반응):

\[2\text{H}_2\text{O}(l) + 2e^- \rightarrow \text{H}_2(g) + 2\text{OH}^-(aq)\]

\(\text{OH}^-\)이 생성되므로 (-)극 인접 부위의 pH는 증가(염기성화)한다.

(2) 소모된 물의 양:

전체 반응: \(2\text{H}_2\text{O}(l) \rightarrow 2\text{H}_2(g) + \text{O}_2(g)\)

생성 기체의 몰비: \(\text{H}_2\):\(\text{O}_2\) = 2:1, 총 기체 = \(\text{H}_2 + \text{O}_2 = 3\)부분

총 기체 18.0 L, 25 \(°\)C, 1.00 atm에서:

\[n_{\text{total}} = \frac{18.0}{24} = 0.75 \text{ mol}\]

\(n_{\text{H}_2} = 0.50\) mol, \(n_{\text{O}_2} = 0.25\) mol

소모된 물: \(2\text{H}_2\text{O}\) → \(2\text{H}_2\) + \(\text{O}_2\) 비율에서, \(\text{H}_2\) 0.50 mol이 생성되려면 \(\text{H}_2\text{O}\) 0.50 mol이 필요하다.

\[m_{\text{H}_2\text{O}} = 0.50 \times 18 = 9.0 \text{ g}\]

(3) NaI를 넣었을 때의 차이:

[표 1]에서 \(\text{I}^-\)의 산화 전위를 확인한다. \(\text{I}_2(s) + 2e^- \rightarrow 2\text{I}^-(aq)\), \(E° = +0.54\) V이므로, 역반응으로 \(\text{I}^-\)의 산화 전위는 -0.54 V이다.

\(\text{H}_2\text{O}\)의 산화 전위는 -1.23 V (\(\text{O}_2 + 4\text{H}^+ + 4e^- \rightarrow 2\text{H}_2\text{O}\)의 역반응)이다.

\(\text{I}^-\)가 \(\text{H}_2\text{O}\)보다 산화되기 쉬우므로 (+)극에서 \(\text{I}^-\)가 먼저 산화된다:

\[2\text{I}^-(aq) \rightarrow \text{I}_2(s) + 2e^-\]

(+)극에서는 산소 기체 대신 \(\text{I}_2\)가 생성되어 용액이 갈색으로 변한다. (-)극에서는 변화가 없고, 여전히 \(\text{H}_2\) 기체가 발생한다 (\(\text{Na}^+\)는 환원 전위가 매우 낮아(-2.71 V) \(\text{H}_2\text{O}\)보다 환원되기 어렵다).

1-4. 메테인의 연소 평형

\(\text{CH}_4\) 16.0 g = 1.00 mol, \(\text{O}_2\) 64.0 g = 2.00 mol

\(T = 227°\text{C} = 500\) K, \(V = 10.0\) L

초기 압력:

- \(P_{\text{CH}_4} = \frac{1.00 \times 0.08 \times 500}{10.0} = 4.00\) atm

- \(P_{\text{O}_2} = \frac{2.00 \times 0.08 \times 500}{10.0} = 8.00\) atm

완전 연소: \(\text{CH}_4 + 2\text{O}_2 \rightarrow \text{CO}_2 + 2\text{H}_2\text{O}\)

불완전 연소: \(2\text{CH}_4 + 3\text{O}_2 \rightarrow 2\text{CO} + 4\text{H}_2\text{O}\)

완전 연소로 반응한 \(\text{CH}_4\)를 \(a\) mol, 불완전 연소로 반응한 \(\text{CH}_4\)를 \(b\) mol이라 하자.

평형에서의 각 성분의 몰 수 (기체이므로 압력은 몰 수에 비례):

\(n_{\text{CH}_4} = 1 - a - b\)

\(n_{\text{O}_2} = 2 - 2a - \frac{3}{2}b\)

\(n_{\text{CO}_2} = a\)

\(n_{\text{CO}} = b\)

\(n_{\text{H}_2\text{O}} = 2a + 2b\)

\(\frac{0.08 \times 500}{10} = 4\) atm/mol 이므로 압력 = 4 \(\times\) 몰 수 atm이다.

평형 조건:

- \(P_{\text{O}_2} = 1.00\) atm → \(n_{\text{O}_2} = 0.25\) mol → \(2 - 2a - 1.5b = 0.25\)

- \(P_{\text{H}_2\text{O}} = 7.50\) atm → \(n_{\text{H}_2\text{O}} = 1.875\) mol → \(2a + 2b = 1.875\)

두 번째 식에서: \(a + b = 0.9375\)

첫 번째 식에서: \(2a + 1.5b = 1.75\)

연립방정식을 풀면:

\(2a + 1.5b = 1.75\) ... (i)

\(a + b = 0.9375\) → \(2a + 2b = 1.875\) ... (ii)

(ii) - (i): \(0.5b = 0.125\) → \(b = 0.25\)

\(a = 0.9375 - 0.25 = 0.6875\)

각 성분의 몰 수와 압력:

\(n_{\text{CH}_4} = 1 - 0.6875 - 0.25 = 0.0625\) mol → \(P = 0.0625 \times 4 = 0.25\) atm

\(n_{\text{CO}} = 0.25\) mol → \(P = 0.25 \times 4 = 1.00\) atm

\(n_{\text{CO}_2} = 0.6875\) mol → \(P = 0.6875 \times 4 = 2.75\) atm

따라서 평형에서 메테인의 압력은 0.25 atm, 일산화탄소의 압력은 1.00 atm, 이산화탄소의 압력은 2.75 atm이다.
