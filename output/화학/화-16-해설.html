[문제 1-1] 다음 표는 화합물의 기준 끓는점에 대한 자료이다.

화합물\(\text{O}_2\)\(\text{CH}_3\text{OH}\)\(\text{PH}_3\)\(\text{Cl}_2\)
기준 끓는점(K)\(a\)\(b\)\(c\)\(d\)

\(a\)와 \(b\), \(a\)와 \(d\), \(b\)와 \(c\)의 크기를 각각 비교하고, 그 이유를 제시문 (가)를 참고하여 서술하시오. (단, H, C, O, P, Cl의 원자량은 각각 1, 12, 16, 31, 35이다.) (6점)

(평가 기준)

\(a(\text{O}_2) < b(\text{CH}_3\text{OH})\): \(\text{O}_2\)와 \(\text{CH}_3\text{OH}\)의 분자량이 같아서 분산력이 유사하다. \(\text{CH}_3\text{OH}\)는 분자 간 수소 결합을 할 수 있어 \(\text{O}_2\)보다 분자 간 힘이 세다.

\(a(\text{O}_2) < d(\text{Cl}_2)\): \(\text{Cl}_2\)의 분자량이 \(\text{O}_2\)의 분자량보다 커서 \(\text{Cl}_2\)의 분산력이 \(\text{O}_2\)의 분산력보다 크다.

\(b(\text{CH}_3\text{OH}) > c(\text{PH}_3)\): \(\text{PH}_3\)와 \(\text{CH}_3\text{OH}\)의 분자량이 유사하여 분산력이 유사하다. \(\text{PH}_3\)는 쌍극자-쌍극자 힘을 \(\text{CH}_3\text{OH}\)는 쌍극자-쌍극자 힘과 분자 간 수소 결합을 할 수 있다.

[문제 1-2] 다음은 \(\text{A}(g)\)와 \(\text{B}(g)\)가 반응하여 \(\text{C}(g)\)가 생성되는 반응의 화학 반응식과 농도로 정의되는 평형상수(\(K\))이다.

\[\text{A}(g) + 2\text{B}(g) \rightleftharpoons \text{C}(g) \quad K\]

그림 I은 온도 \(T\)에서 강철 용기에 \(\text{A}(g)\)와 \(\text{B}(g)\)를 넣은 초기 상태를 나타낸 것이고, 그림 II는 반응이 진행되어 평형에 도달한 상태를 나타낸 것이다. 그림 II에서 \(\text{A}(g)\) 몰분율은 0.5이다.

제시문 (나)와 (다)를 참고하여 그림 II에서 \(\text{C}(g)\)의 부분 압력과 평형상수(\(K\))를 구하고, 그 과정을 서술하시오. (단, 온도는 \(T\)로 일정하다.) (7점)

(평가 기준)

평형에서 전체 몰: 6몰, 초기 전체 몰수: 8몰

\(\text{A}(g)\) +\(2\text{B}(g)\)\(\rightleftharpoons\)\(\text{C}(g)\)
초기(몰)\(3+x\)\(3+x\)0
변화량(몰)\(-x\)\(-2x\)\(+x\)
평형(몰)3\(3-x\)\(x\)
321

\(2(3+x) = 8\), \(3+x = 4\), \(x = 1\)

\(P_C = X_C \times P_{tot} = 1/6 \times 3\text{atm} = 1/2 = 0.5\;\text{atm}\)

\([\text{A}] = 3/3\), \([\text{B}] = 2/3\), \([\text{C}] = 1/3\)

\[K = \frac{[\text{C}]}{[\text{A}][\text{B}]^2} = \frac{(1/3)}{(1)(2/3)^2} = \frac{3}{4}\]

[문제 1-3] 다음은 \(\text{A}(g)\)와 \(\text{B}(g)\)가 반응하여 \(\text{C}(g)\)가 생성되는 반응의 화학 반응식이다.

\[\text{A}(g) + 2\text{B}(g) \to 2\text{C}(g)\]

그림은 온도 \(T\)에서 강철 용기에 \(\text{A}(g)\)와 \(\text{B}(g)\)를 넣은 초기 상태를 나타낸 것이다.

제시문 (다)를 참고하여 A의 몰분율(\(\chi_A\))을 달리하여 반응을 완결 시켰을 때, \(\chi_A\)에 따른 C의 부분 압력(\(P_C\))의 그래프를 그리고, 최대 \(P_C\)값과 그때의 \(\chi_A\)값을 표시하시오. (단, 온도는 \(T\)로 일정하고, \(\chi_A\)는 \(0 \leq \chi_A \leq 1\)이다.) (7점)

(평가 기준)

\(\chi_A = 1/3\)일 때 \(P_C\)가 최댓값을 가짐.

\(\chi_A \leq 1/3\)일 때, A의 양에 따라 C의 생성량이 결정됨.

\(PV = nRT\), \(n = PV/RT = 2/RT\)

\(\text{A}(g)\) +\(2\text{B}(g)\)\(\to\)\(2\text{C}(g)\)
초기(몰)\(2\chi_A/RT\)\(2(1-\chi_A)/RT\)0
변화량(몰)\(-2\chi_A/RT\)\(-4\chi_A/RT\)\(+4\chi_A/RT\)
반응후(몰)0\(-\)\(4\chi_A/RT\)

\(P_C V = nRT\), \(P_C = nRT/V = (4\chi_A/RT) \times RT = 4\chi_A\)

최대 \(P_C = 4\chi_A = 4 \times 1/3 = 4/3\)

\(\chi_A \geq 1/3\)일 때, 생성되는 C의 양은 B의 양에 의하여 결정된다.

\(\text{A}(g)\) +\(2\text{B}(g)\)\(\to\)\(2\text{C}(g)\)
초기(몰)\(2\chi_A/RT\)\(2(1-\chi_A)/RT\)0
변화량(몰)\(-(1-\chi_A)/RT\)\(-2(1-\chi_A)/RT\)\(+2(1-\chi_A)/RT\)
반응후(몰)\(-\)0\(2(1-\chi_A)/RT\)

\(P_C = nRT/V = 2(1-\chi_A)/RT \times RT = 2(1-\chi_A)\)

\(\chi_A = 1/3\)일 때 \(P_C\)의 최댓값, \(4/3\;\text{atm}\) 표시

\(\chi_A \leq 1/3\)일 때, \(P_C = 4\chi_A\) 식으로 직선을 그림

\(\chi_A \geq 1/3\)일 때, \(P_C = 2(1-\chi_A)\) 식으로 직선을 그림
