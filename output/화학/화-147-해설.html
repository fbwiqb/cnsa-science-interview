유효 핵전하, 이온 반지름, 격자 에너지

(화학 1)

i) \(\text{O}^{2-}\), \(\text{F}^-\), \(\text{Ne}\), \(\text{Na}^+\), \(\text{Mg}^{2+}\)은 모두 전자 수가 10개인 등전자 종이다.

등전자 종에서는 양성자 수(핵전하)가 클수록 전자를 더 강하게 끌어당기므로 반지름이 작아진다.

각 종의 양성자 수: \(\text{O}^{2-}\)(8개), \(\text{F}^-\)(9개), \(\text{Ne}\)(10개), \(\text{Na}^+\)(11개), \(\text{Mg}^{2+}\)(12개)

따라서 반지름이 작은 것부터 차례로:

\[\boxed{\text{Mg}^{2+} < \text{Na}^+ < \text{Ne} < \text{F}^- < \text{O}^{2-}}\]

양성자 수가 많을수록 유효 핵전하가 커서 전자를 더 강하게 잡아당기므로 이온 반지름이 작아진다.

ii) 세 화합물의 격자 에너지 비교

격자 에너지: \(U = -k\dfrac{Q_1 Q_2}{d_1}\)

\(\text{NaCl}\): \(|Q_1 Q_2| = 1\), 이온 간 거리 \(d_1\)

\(\text{MgCl}_2\): \(|Q_1 Q_2| = 2\) (\(\text{Mg}^{2+}\)의 전하가 \(+2\)), \(d_2 < d_1\) (\(\text{Mg}^{2+}\)의 반지름 < \(\text{Na}^+\)의 반지름)

\(\text{MgO}\): \(|Q_1 Q_2| = 4\) (\(\text{Mg}^{2+}\)는 \(+2\), \(\text{O}^{2-}\)는 \(-2\)), \(d_3 < d_2\) (\(\text{O}^{2-}\)의 반지름 < \(\text{Cl}^-\)의 반지름)

격자 에너지의 크기: \(|U(\text{MgO})| > |U(\text{MgCl}_2)| > |U(\text{NaCl})|\)

에너지-거리 좌표평면에서: \(d_3 < d_2 < d_1\)이고 \(E_3 < E_2 < E_1 < 0\)

\((d_3, E_3)\): \(\text{MgO}\) - 가장 왼쪽, 가장 아래 (에너지가 가장 낮음)

\((d_1, E_1)\): \(\text{NaCl}\) - 가장 오른쪽, 가장 위 (에너지가 가장 덜 낮음)

(화학 2)

에너지: \(E = -\dfrac{\alpha}{d} + \dfrac{\beta}{d^2}\)

평형 거리에서 \(\frac{dE}{dd} = 0\): \(\dfrac{\alpha}{d^2} - \dfrac{2\beta}{d^3} = 0 \implies d_i = \dfrac{2\beta}{\alpha}\)

평형 에너지: \(E_i = -\dfrac{\alpha^2}{4\beta}\)

조건: \(1 \leq d_i \leq 2\), \(E_i \geq -1\)

\(\alpha^2 \leq 4\beta\)이고 \(\beta \leq \alpha\) (from \(d_i \geq 1\))

\(\beta = \frac{\alpha^2}{4}\)로 놓으면 \(E_i = -1\), \(d_i = \frac{\alpha}{2}\)

\(d_i \leq 2\)에서 \(\alpha \leq 4\), \(\beta \leq \alpha\)에서 \(\frac{\alpha^2}{4} \leq \alpha\)이므로 \(\alpha \leq 4\)

\(\alpha = 4\)일 때 \(\beta = 4\): \(\alpha + \beta = \boxed{8}\)
