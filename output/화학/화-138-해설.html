해설 1

[반응 계수 및 산화수 변화]

[반응식 1]

N의 개수 : \(a = 2c\)

H의 개수 : \(3a = 2d\)

O의 개수 : \(2b = d\)

\(a : b : c : d = 4 : 3 : 2 : 6\)

[반응식 2]

N의 개수 : \(e = g\)

H의 개수 : \(3e = 2h\)

O의 개수 : \(2f = g + h\)

\(e : f : g : h = 4 : 5 : 4 : 6\)

[반응식 3]

N의 개수 : \(i = k\)

H의 개수 : \(3i = 2l\)

O의 개수 : \(2j = 2k + l\)

\(i : j : k : l = 4 : 7 : 4 : 6\)

따라서 반응 계수(\(b\), \(f\), \(j\))를 구하면 각각 3, 5, 7 이다.

[반응식 1]과 [반응식 2]에서 질소의 산화수 변화는 각각 다음과 같다.

[반응식 1] : \(-3\)에서 \(0\)으로 변하였다.

[반응식 2] : \(-3\)에서 \(+2\)로 변하였다.

[반응식 3]에서 환원된 물질은 산소 기체(\(\text{O}_2\))이다.

해설 2

[기체 생성물 확인 및 $x$ 값]

반응물인 암모니아가 17 g(1몰) 반응한다면 반응별로 각각 14 g(0.5몰), 30 g(1몰), 46 g(1몰)의 기체 분자(\(\text{N}_2\), NO, \(\text{NO}_2\))가 발생할 것이다. 즉 일정양의 암모니아가 반응할 때 생성되는 기체분자(\(\text{N}_2\), NO, \(\text{NO}_2\))의 질량비는 14:30:46 (7:15:23) 이다.

주어진 그래프에서 일정한 양(\(x\) g)의 암모니아가 반응할 때 생성되는 기체 A와 B의 질량비가 46 : 14의 비율로 얻어지므로 이는 각각 \(\text{NO}_2\)와 \(\text{N}_2\)에 해당함을 알 수 있다.

또한 \(x\) g의 암모니아가 반응하였을 때 생성된 \(\text{NO}_2\), \(\text{N}_2\) 기체의 몰수가 각각 1몰, 0.5몰이므로, 암모니아가 1몰 만큼 반응하였음을 알 수 있다.

따라서 기체 A는 \(\text{NO}_2\), B는 \(\text{N}_2\) 이다.

그리고 소모된 암모니아의 질량 \(x\)는 17 g 이다.
