문제 1. HCl, HF, HBr의 에너지-거리 그래프 비교 및 NaF의 이온 결합

HCl, HF, HBr의 결합 에너지와 결합 길이:

분자결합 에너지 (kJ/mol)결합 길이 (pm)
\(\text{HF}\)56792
\(\text{HCl}\)429128
\(\text{HBr}\)366141

HF의 그래프 (HCl 대비):

- 결합 에너지가 더 크다 (\(567 > 429\)): 에너지 최솟값이 HCl보다 더 아래에 위치

- 결합 길이가 더 짧다 (\(92 < 128\)): 최솟값이 HCl보다 왼쪽에 위치

\(\text{F}\)의 전기음성도(4.0)가 \(\text{Cl}\)(3.0)보다 훨씬 크므로, \(\text{H-F}\) 결합의 극성이 매우 커서 결합 에너지가 크다. 또한 \(\text{F}\)의 원자 반지름이 \(\text{Cl}\)보다 작으므로 결합 길이가 짧다.

HBr의 그래프 (HCl 대비):

- 결합 에너지가 더 작다 (\(366 < 429\)): 에너지 최솟값이 HCl보다 위에 위치

- 결합 길이가 더 길다 (\(141 > 128\)): 최솟값이 HCl보다 오른쪽에 위치

\(\text{Br}\)의 전기음성도(2.8)가 \(\text{Cl}\)(3.0)보다 작고, \(\text{Br}\)의 원자 반지름이 \(\text{Cl}\)보다 크므로 결합 길이가 길고 결합 에너지가 작다.

그래프 요약:

- 에너지 최솟값 깊이: \(\text{HF}(-567) < \text{HCl}(-429) < \text{HBr}(-366)\)

- 에너지 최솟값 위치(거리): \(\text{HF}(92) < \text{HCl}(128) < \text{HBr}(141)\) pm

NaF의 이온 결합 에너지-거리 그래프:

\(\text{NaF}\)에서 \(\text{Na}^+\)와 \(\text{F}^-\) 이온의 결합:

에너지가 가장 낮은 지점의 이온 간 거리: 231 pm (제시문에서 주어짐)

이온이 서로 아주 멀리 있을 때의 에너지 값:

중성 \(\text{Na}\)와 \(\text{F}\) 원자가 멀리 떨어져 있을 때의 에너지를 0으로 설정한다.

\(\text{Na}^+(g) + \text{F}^-(g)\)의 에너지는 다음 과정으로부터 계산한다:

\[\text{Na}(g) \rightarrow \text{Na}^+(g) + e^- \quad \Delta E = +496 \text{ kJ/mol (이온화 에너지)}\]
\[\text{F}(g) + e^- \rightarrow \text{F}^-(g) \quad \Delta E = -328 \text{ kJ/mol (전자 친화도)}\]

따라서 중성 원자 기준으로 \(\text{Na}^+(g) + \text{F}^-(g)\)의 에너지:

\[E_{\infty} = 496 - 328 = +168 \text{ kJ/mol}\]

이온이 매우 멀리 있을 때의 에너지는 중성 원자 대비 \(+168\) kJ/mol이다.

그래프에서:

- 이온 간 거리가 매우 클 때: \(E = +168\) kJ/mol

- 이온 간 거리가 231 pm일 때: 에너지가 최소 (가장 안정한 상태)

- 이온 간 거리가 매우 짧을 때: 반발력으로 에너지가 급격히 증가

문제 2. 화합물의 산화수 분석과 극성 비교

\(\text{XYZ}_4\), \(\text{XYZ}_6\), \(\text{XY}_2\text{Z}_6\)에서 X의 산화수:

화합물X의 산화수
\(\text{XYZ}_4\)\(-2\)
\(\text{XYZ}_6\)\(0\)
\(\text{XY}_2\text{Z}_6\)\(+2\)

Y와 Z의 산화수는 일정하다. Y의 산화수를 \(y\), Z의 산화수를 \(z\)라 하자.

각 화합물은 중성이므로 산화수의 합 = 0:

\[\text{XYZ}_4: \quad -2 + y + 4z = 0 \quad \cdots (1)\]
\[\text{XYZ}_6: \quad 0 + y + 6z = 0 \quad \cdots (2)\]
\[\text{XY}_2\text{Z}_6: \quad +2 + 2y + 6z = 0 \quad \cdots (3)\]

식 (2)에서: \(y = -6z\)

식 (1)에 대입: \(-2 + (-6z) + 4z = 0\) → \(-2 - 2z = 0\) → \(z = -1\)

\(y = -6(-1) = +6\)

검증 (3): \(+2 + 2(+6) + 6(-1) = +2 + 12 - 6 = +8 \neq 0\)

다시 계산한다.

식 (3): \(+2 + 2y + 6z = 0\) → \(2y + 6z = -2\) → \(y + 3z = -1\) ... (3')

식 (2)에서: \(y + 6z = 0\) → \(y = -6z\) ... (2')

(3')에 대입: \(-6z + 3z = -1\) → \(-3z = -1\) → \(z = \dfrac{1}{3}\)

이는 정수가 아니므로, 화합물의 원자 수를 다시 생각한다.

\(z\)가 분수인 것은 비정상적이므로, Z가 다원자 그룹일 가능성을 고려하거나 산화수 규칙을 재적용한다.

보다 일반적으로 생각하면:

\(\text{X}\)의 산화수 변화를 관찰한다: \(-2 \rightarrow 0 \rightarrow +2\) (Y 수가 1→1→2, Z 수가 4→6→6)

\(\text{XYZ}_4\)에서 \(\text{XYZ}_6\)으로 갈 때: X 산화수 \(+2\) 증가, Z 2개 추가

\(\text{XYZ}_6\)에서 \(\text{XY}_2\text{Z}_6\)으로 갈 때: X 산화수 \(+2\) 증가, Y 1개 추가

Z 1개 추가 시 X 산화수 \(+1\) 증가 → Z는 전기음성도가 X보다 크다

Y 1개 추가 시 X 산화수 \(+2\) 증가 → Y는 전기음성도가 X보다 작다

전기음성도 순서: \(\text{Z} > \text{X} > \text{Y}\)

Y의 산화수: 식 (2)에서 \(y = -6z\)

\(z = \dfrac{1}{3}\)이면 \(y = -2\)

따라서: \(\text{Y}\)의 산화수 = \(-2\), \(\text{Z}\)의 산화수 = \(+\dfrac{1}{3}\)

이러한 분수 산화수가 나오는 것은 Z가 원자가 아닌 원자단일 수 있음을 시사한다. 예를 들어 \(\text{Z} = \text{F}\) (산화수 \(-1\))로 놓으면:

식 (2): \(0 + y + 6(-1) = 0\) → \(y = +6\)

식 (1): \(-2 + 6 + 4(-1) = 0\) ✓

식 (3): \(+2 + 2(6) + 6(-1) = +2 + 12 - 6 = 8 \neq 0\) ✗

\(\text{Z}\)의 산화수를 \(-1\)로 하면 식 (3)이 맞지 않으므로, 문제에서 주어진 조건 하에 \(\text{Y}\)의 산화수 = \(+1\), \(\text{Z}\)의 산화수 = \(-1\)로 재시도한다:

식 (1): \(-2 + 1 + 4(-1) = -5 \neq 0\) ✗

종합하면, 전기음성도 큰 순서: \(\boxed{\text{Z} > \text{X} > \text{Y}}\)

\(\text{CF}_4\), \(\text{CH}_2\text{F}_2\), \(\text{CF}_2\text{Cl}_2\)의 극성 비교:

\(\text{CF}_4\):

- 정사면체 구조, 4개의 동일한 \(\text{C-F}\) 결합

- 쌍극자 모멘트 벡터가 완전히 상쇄 → 무극성 (\(\mu = 0\))

\(\text{CH}_2\text{F}_2\):

- 사면체 구조, 2개의 \(\text{C-H}\)와 2개의 \(\text{C-F}\) 결합

- \(\text{C-F}\) 결합의 극성이 \(\text{C-H}\) 결합보다 훨씬 크므로 벡터 합이 상쇄되지 않음

- 상당한 쌍극자 모멘트를 가짐 → 극성 큼 (\(\mu = 1.97\) D)

\(\text{CF}_2\text{Cl}_2\) (프레온-12):

- 사면체 구조, 2개의 \(\text{C-F}\)와 2개의 \(\text{C-Cl}\) 결합

- \(\text{C-F}\)와 \(\text{C-Cl}\)의 극성이 다르므로 벡터 합이 상쇄되지 않음

- 쌍극자 모멘트를 가짐 → 극성 있음 (\(\mu = 0.51\) D)

극성 크기 순서:

\[\boxed{\text{CH}_2\text{F}_2 > \text{CF}_2\text{Cl}_2 > \text{CF}_4}\]

이유: \(\text{CF}_4\)는 대칭 구조로 무극성이고, \(\text{CH}_2\text{F}_2\)는 \(\text{C-H}\)와 \(\text{C-F}\)의 전기음성도 차이가 커서 극성이 가장 크며, \(\text{CF}_2\text{Cl}_2\)는 \(\text{C-F}\)와 \(\text{C-Cl}\)의 극성 차이가 \(\text{C-H}\)와 \(\text{C-F}\)의 차이보다 작아 극성이 중간이다.
