산화 환원 반응의 계수 결정과 pH 변화

[문제 1] 반응계수 \(a\)~\(f\) 결정 및 \(\text{MnO}_2\) 질량 계산

반응 (1): 산성 조건에서 $\text{Sn}^{2+}$과 $\text{MnO}_4^-$

\[a\text{Sn}^{2+} + 2\text{MnO}_4^- + 16\text{H}^+ \rightarrow b\text{Sn}^{4+} + 2\text{Mn}^{2+} + 8\text{H}_2\text{O}\]

산화수 변화:

- \(\text{Sn}^{2+} \rightarrow \text{Sn}^{4+}\): 산화수 \(+2\) 증가 (전자 2개 잃음)

- \(\text{Mn}\): \(\text{MnO}_4^-\)에서 Mn 산화수 = \(+7\), \(\text{Mn}^{2+}\)에서 산화수 = \(+2\), 산화수 \(5\) 감소 (전자 5개 얻음)

전자 균형:

- \(\text{Mn}\) 2개가 각각 5개의 전자를 얻으므로 총 10개 전자 획득

- \(\text{Sn}\) 1개가 2개의 전자를 잃으므로, \(a\)개의 \(\text{Sn}\)이 \(2a\)개의 전자를 잃음

- \(2a = 10\)이므로 \(a = 5\), \(b = 5\)

\[\boxed{a = 5, \quad b = 5}\]

검증: \(5\text{Sn}^{2+} + 2\text{MnO}_4^- + 16\text{H}^+ \rightarrow 5\text{Sn}^{4+} + 2\text{Mn}^{2+} + 8\text{H}_2\text{O}\)

- 전하: 좌변 = \(5(+2) + 2(-1) + 16(+1) = 10 - 2 + 16 = +24\)

- 우변 = \(5(+4) + 2(+2) = 20 + 4 = +24\) ✓

반응 (2): 중성 조건에서 $\text{Sn}^{2+}$과 $\text{MnO}_4^-$

\[c\text{Sn}^{2+} + 2\text{MnO}_4^- + 4\text{H}_2\text{O} \rightarrow d\text{Sn}^{4+} + 2\text{MnO}_2 + 8\text{OH}^-\]

산화수 변화:

- \(\text{Sn}^{2+} \rightarrow \text{Sn}^{4+}\): 전자 2개 잃음

- \(\text{Mn}\): \(\text{MnO}_4^-\)에서 \(+7\), \(\text{MnO}_2\)에서 \(+4\), 산화수 \(3\) 감소 (전자 3개 얻음)

전자 균형:

- \(\text{Mn}\) 2개: 총 6개 전자 획득

- \(c\)개의 \(\text{Sn}\): \(2c\)개 전자 손실

- \(2c = 6\)이므로 \(c = 3\), \(d = 3\)

\[\boxed{c = 3, \quad d = 3}\]

검증: \(3\text{Sn}^{2+} + 2\text{MnO}_4^- + 4\text{H}_2\text{O} \rightarrow 3\text{Sn}^{4+} + 2\text{MnO}_2 + 8\text{OH}^-\)

- 전하: 좌변 = \(3(+2) + 2(-1) + 0 = 6 - 2 = +4\)

- 우변 = \(3(+4) + 0 + 8(-1) = 12 - 8 = +4\) ✓

반응 (3): 산성 조건에서 $\text{Sn}^{2+}$과 $\text{Cr}_2\text{O}_7^{2-}$

\[e\text{Sn}^{2+} + \text{Cr}_2\text{O}_7^{2-} + 14\text{H}^+ \rightarrow f\text{Sn}^{4+} + 2\text{Cr}^{3+} + 7\text{H}_2\text{O}\]

산화수 변화:

- \(\text{Sn}^{2+} \rightarrow \text{Sn}^{4+}\): 전자 2개 잃음

- \(\text{Cr}\): \(\text{Cr}_2\text{O}_7^{2-}\)에서 각 Cr 산화수 = \(+6\), \(\text{Cr}^{3+}\)에서 \(+3\), 각 Cr이 전자 3개 얻음

전자 균형:

- \(\text{Cr}\) 2개: 총 6개 전자 획득

- \(e\)개의 \(\text{Sn}\): \(2e\)개 전자 손실

- \(2e = 6\)이므로 \(e = 3\), \(f = 3\)

\[\boxed{e = 3, \quad f = 3}\]

검증: \(3\text{Sn}^{2+} + \text{Cr}_2\text{O}_7^{2-} + 14\text{H}^+ \rightarrow 3\text{Sn}^{4+} + 2\text{Cr}^{3+} + 7\text{H}_2\text{O}\)

- 전하: 좌변 = \(3(+2) + (-2) + 14(+1) = 6 - 2 + 14 = +18\)

- 우변 = \(3(+4) + 2(+3) = 12 + 6 = +18\) ✓

$\text{MnO}_2$ 질량 계산:

\(\text{KMnO}_4\)의 화학식량 = \(39 + 55 + 4 \times 16 = 39 + 55 + 64 = 158\) g/mol

\(\text{KMnO}_4\) 0.79 g의 몰수:

\[n(\text{KMnO}_4) = \frac{0.79}{158} = 0.005 \text{ mol}\]

중성 조건 반응에서 \(\text{MnO}_4^-\)와 \(\text{MnO}_2\)의 몰비 = 2 : 2 = 1 : 1

따라서 \(n(\text{MnO}_2) = n(\text{MnO}_4^-) = 0.005\) mol

\(\text{MnO}_2\)의 화학식량 = \(55 + 2 \times 16 = 87\) g/mol

\[m(\text{MnO}_2) = 0.005 \times 87 = 0.435 \text{ g}\]
\[\boxed{m(\text{MnO}_2) = 0.435 \text{ g}}\]

[문제 2] 반응 진행에 따른 pH 변화

산성 조건의 반응:

\[5\text{Sn}^{2+} + 2\text{MnO}_4^- + 16\text{H}^+ \rightarrow 5\text{Sn}^{4+} + 2\text{Mn}^{2+} + 8\text{H}_2\text{O}\]

반응이 진행됨에 따라 \(\text{H}^+\)가 소모된다. \(\text{H}^+\) 농도가 감소하므로:

\[[\text{H}^+] \downarrow \quad \Rightarrow \quad \text{pH} = -\log[\text{H}^+] \uparrow\]

따라서 산성 조건에서 반응이 진행되면 pH가 증가한다 (용액이 덜 산성이 된다).

중성 조건의 반응:

\[3\text{Sn}^{2+} + 2\text{MnO}_4^- + 4\text{H}_2\text{O} \rightarrow 3\text{Sn}^{4+} + 2\text{MnO}_2 + 8\text{OH}^-\]

반응이 진행됨에 따라 \(\text{OH}^-\)가 생성된다.

\[K_w = [\text{H}^+][\text{OH}^-]\]

\([\text{OH}^-]\)가 증가하면, \(K_w\)가 일정(온도 일정)하므로 \([\text{H}^+]\)가 감소한다:

\[[\text{OH}^-] \uparrow \quad \Rightarrow \quad [\text{H}^+] \downarrow \quad \Rightarrow \quad \text{pH} \uparrow\]

따라서 중성 조건에서도 반응이 진행되면 pH가 증가한다 (용액이 염기성으로 변한다).

두 반응 모두 pH가 증가하지만, 그 메커니즘은 다르다:

- 산성 조건: \(\text{H}^+\)의 직접 소모로 pH 증가

- 중성 조건: \(\text{OH}^-\)의 생성으로 pH 증가
